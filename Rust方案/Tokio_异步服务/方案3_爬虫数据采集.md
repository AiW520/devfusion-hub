# Tokio 异步服务方案：方案3 - 爬虫数据采集

## 项目介绍

### 项目是什么
这是一个基于 Rust + Tokio 构建的高性能爬虫数据采集系统，支持并发爬取、代理轮换、请求限流、分布式采集、增量更新和反反爬策略。

### 解决什么问题
- 高效爬取网页数据
- 绕过反爬机制
- 管理大量并发请求
- 数据清洗和存储
- 增量更新和去重

### 核心技术亮点
| 特性 | 技术实现 |
|------|----------|
| 异步 HTTP | reqwest + async |
| HTML 解析 | scraper / selectors |
| 并发控制 | Semaphore + rate limit |
| 数据存储 | PostgreSQL + Redis |
| 代理轮换 | 代理池管理 |
| 反反爬 | User-Agent、Cookie、延迟 |
| 增量爬取 | Hash + 布隆过滤器 |

### 适合什么场景
- 竞品数据采集
- 价格监控
- 新闻聚合
- SEO 数据分析
- 学术数据收集

```rust
            method: "GET".to_string(),
            headers: HashMap::new(),
            body: None,
            priority: 0,
            created_at: Utc::now(),
            retry_count: 0,
            use_proxy: false,
        }
    }

        Self {
            id: uuid::Uuid::new_v4().to_string(),

            headers: HashMap::new(),
            body: Some(body.to_string()),
            priority: 0,
            created_at: Utc::now(),
            retry_count: 0,
            use_proxy: false,
        }
    }

        self.headers.insert(key.to_string(), value.to_string());
        self
    }
    
    /// 设置优先级
    pub fn with_priority(mut self, priority: i32) -> Self {
        self.priority = priority;
        self
    }
    
    /// 启用代理
    pub fn with_proxy(mut self) -> Self {
        self.use_proxy = true;
        self
    }
}

/// 爬取响应
#[derive(Debug, Clone)]
pub struct CrawlResponse {
    /// 请求 ID
    pub request_id: String,
    
    /// HTTP 状态码
    pub status_code: u16,
    
    /// 响应头
    pub headers: HashMap<String, String>,
    
    /// 响应体
    pub body: Vec<u8>,
    
    /// 最终 URL（跟随重定向后）
    pub final_url: String,
    
    /// 响应时间（毫秒）
    pub response_time_ms: u64,
    
    /// 使用的代理
    pub proxy: Option<String>,
    
    /// 错误信息
    pub error: Option<String>,
}

impl CrawlResponse {
    /// 是否成功
    pub fn is_success(&self) -> bool {
        self.error.is_none() && (200..300).contains(&self.status_code)
    }
    
    /// 是否是客户端错误
    pub fn is_client_error(&self) -> bool {
        (400..500).contains(&self.status_code)
    }
    
    /// 是否是服务器错误
    pub fn is_server_error(&self) -> bool {
        (500..600).contains(&self.status_code)
    }
    
    /// 获取响应体为字符串
    pub fn text(&self) -> Result<String, std::string::FromUtf8Error> {
        String::from_utf8(self.body.clone())
    }
    
    /// 获取响应体为 JSON
    pub fn json<T: serde::de::DeserializeOwned>(&self) -> Result<T, serde_json::Error> {
        serde_json::from_slice(&self.body)
    }
}
```

```rust
// proxy/pool.rs - 代理池管理
// 作用：管理和轮换代理服务器

use crate::config::ProxyConfig;
use reqwest::Proxy as ReqwestProxy;
use std::sync::Arc;
use tokio::sync::{RwLock, mpsc};
use std::time::Duration;
use tracing::{info, warn, debug};

// ==================== 代理信息 ====================

/// 代理信息
#[derive(Debug, Clone)]
pub struct Proxy {
    /// 代理地址 (ip:port)
    pub address: String,
    
    /// 协议类型 (http/https/socks5)
    pub protocol: String,
    
    /// 用户名（可选）
    pub username: Option<String>,
    
    /// 密码（可选）
    pub password: Option<String>,
    
    /// 是否可用
    pub available: bool,
    
    /// 最后使用时间
    pub last_used: std::time::Instant,
    
    /// 成功次数
    pub success_count: u32,
    
    /// 失败次数
    pub failure_count: u32,
}

impl Proxy {
    pub fn new(address: &str) -> Self {
        Self {
            address: address.to_string(),
            protocol: "http".to_string(),
            username: None,
            password: None,
            available: true,
            last_used: std::time::Instant::now(),
            success_count: 0,
            failure_count: 0,
        }
    }
    
    pub fn with_auth(mut self, username: &str, password: &str) -> Self {
        self.username = Some(username.to_string());
        self.password = Some(password.to_string());
        self
    }
    
    /// 记录成功
    pub fn record_success(&mut self) {
        self.success_count += 1;
        self.last_used = std::time::Instant::now();
    }
    
    /// 记录失败
    pub fn record_failure(&mut self) {
        self.failure_count += 1;
        self.last_used = std::time::Instant::now();
        
        // 失败超过阈值，标记为不可用
        if self.failure_count >= 5 {
            self.available = false;
            warn!("代理 {} 失败次数过多，标记为不可用", self.address);
        }
    }
    
    /// 获取成功率
    pub fn success_rate(&self) -> f64 {
        let total = self.success_count + self.failure_count;
        if total == 0 {
            return 1.0;
        }
        self.success_count as f64 / total as f64
    }
    
    /// 转换为 reqwest Proxy
    pub fn to_reqwest_proxy(&self) -> anyhow::Result<ReqwestProxy> {
        let url = if let (Some(user), Some(pass)) = (&self.username, &self.password) {
            format!("{}://{}:{}@{}", self.protocol, user, pass, self.address)
        } else {
            format!("{}://{}", self.protocol, self.address)
        };
        
        Ok(ReqwestProxy::http(&url)?)
    }
}

// ==================== 代理池 ====================

/// 代理池
pub struct ProxyPool {
    /// 代理列表
    proxies: RwLock<Vec<Proxy>>,
    
    /// 当前索引（轮询用）
    current_index: RwLock<usize>,
    
    /// 配置
    config: ProxyConfig,
}

impl ProxyPool {
    /// 创建新的代理池
    pub fn new(config: ProxyConfig) -> Self {
        Self {
            proxies: RwLock::new(Vec::new()),
            current_index: RwLock::new(0),
            config,
        }
    }
    
    /// 添加代理
    pub async fn add_proxy(&self, proxy: Proxy) {
        let mut proxies = self.proxies.write().await;
        if !proxies.iter().any(|p| p.address == proxy.address) {
            proxies.push(proxy);
            info!("添加代理: {}", proxy.address);
        }
    }
    
    /// 获取可用代理（轮询）
    pub async fn get_proxy(&self) -> Option<Proxy> {
        let proxies = self.proxies.read().await;
        let mut index = self.current_index.read().await;
        
        // 找到下一个可用的代理
        let start = *index;
        let len = proxies.len();
        
        if len == 0 {
            return None;
        }
        
        for i in 0..len {
            let idx = (start + i) % len;
            if proxies[idx].available {
                *index = (idx + 1) % len;
                return Some(proxies[idx].clone());
            }
        }
        
        // 所有代理都不可用，返回第一个
        if proxies.len() > 0 {
            let proxy = proxies[0].clone();
            return Some(proxy);
        }
        
        None
    }
    
    /// 记录代理使用结果
    pub async fn record_result(&self, address: &str, success: bool) {
        let mut proxies = self.proxies.write().await;
        
        if let Some(proxy) = proxies.iter_mut().find(|p| p.address == address) {
            if success {
                proxy.record_success();
            } else {
                proxy.record_failure();
            }
        }
    }
    
    /// 获取代理池大小
    pub async fn len(&self) -> usize {
        self.proxies.read().await.len()
    }
    
    /// 获取可用代理数量
    pub async fn available_count(&self) -> usize {
        self.proxies.read().await.iter().filter(|p| p.available).count()
    }
    
    /// 从 URL 加载代理
    pub async fn load_from_url(&self, name: &str, url: &str) -> anyhow::Result<()> {
        info!("从 {} 加载代理列表...", name);
        
        let response = reqwest::get(url).await?;
        let text = response.text().await?;
        
        // 解析代理列表（假设每行一个 ip:port）
        for line in text.lines() {
            let line = line.trim();
            if line.is_empty() || line.starts_with('#') {
                continue;
            }
            
            let proxy = Proxy::new(line);
            self.add_proxy(proxy).await;
        }
        
        info!("代理加载完成，当前池中有 {} 个代理", self.len().await);
        
        Ok(())
    }
}

/// 共享代理池
pub type SharedProxyPool = Arc<ProxyPool>;

/// 创建共享代理池
pub fn create_proxy_pool(config: ProxyConfig) -> SharedProxyPool {
    Arc::new(ProxyPool::new(config))
}
```

```rust
// crawler/fetcher.rs - HTTP 请求获取器
// 作用：执行 HTTP 请求，支持代理和重试

use crate::config::CrawlerConfig;
use crate::models::{CrawlRequest, CrawlResponse};
use crate::proxy::SharedProxyPool;
use reqwest::{Client, ClientBuilder};
use std::collections::HashMap;
use std::sync::Arc;
use std::time::{Duration, Instant};
use tokio::sync::Semaphore;
use tracing::{debug, error, warn};

// ==================== HTTP 获取器 ====================

/// HTTP 获取器
pub struct Fetcher {
    /// HTTP 客户端
    client: Client,
    
    /// 配置
    config: CrawlerConfig,
    
    /// 代理池
    proxy_pool: SharedProxyPool,
    
    /// 并发控制信号量
    semaphore: Arc<Semaphore>,
    
    /// User-Agent 列表
    user_agents: Vec<String>,
}

impl Fetcher {
    /// 创建新的获取器
    pub fn new(
        config: CrawlerConfig,
        proxy_pool: SharedProxyPool,
    ) -> anyhow::Result<Self> {
        // 构建 HTTP 客户端
        let client = ClientBuilder::new()
            .timeout(Duration::from_secs(config.timeout_secs))
            .cookie_store(true)  // 启用 Cookie
            .gzip(true)  // 启用 Gzip
            .brotli(true)
            .deflate(true)
            .build()?;
        
        // 创建信号量控制并发
        let semaphore = Arc::new(Semaphore::new(config.max_concurrent));
        
        Ok(Self {
            client,
            config,
            proxy_pool,
            semaphore,
            user_agents: config.user_agents.clone(),
        })
    }
    
    /// 获取随机 User-Agent
    fn random_user_agent(&self) -> &str {
        use std::collections::hash_map::DefaultHasher;
        use std::hash::{Hash, Hasher};
        
        let mut hasher = DefaultHasher::new();
        Instant::now().elapsed().hash(&mut hasher);
        let index = (hasher.finish() as usize) % self.user_agents.len();
        
        &self.user_agents[index]
    }
    
    /// 执行请求
    pub async fn fetch(&self, request: CrawlRequest) -> CrawlResponse {
        // 获取并发许可
        let _permit = self.semaphore.acquire().await.unwrap();
        
        let start = Instant::now();
        let mut last_error = None;
        
        // 获取代理（如果需要）
        let proxy = if request.use_proxy {
            self.proxy_pool.get_proxy().await
        } else {
            None
        };
        
        // 构建请求
        let mut req_builder = self.client
            .request(
                reqwest::Method::from_bytes(request.method.as_bytes()).unwrap(),
                &request.url,
            )
            .header("User-Agent", self.random_user_agent())
            .header("Accept", "text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8")
            .header("Accept-Language", "zh-CN,zh;q=0.9,en;q=0.8")
            .header("Accept-Encoding", "gzip, deflate, br")
            .header("Connection", "keep-alive");
        
        // 添加自定义请求头
        for (key, value) in &request.headers {
            req_builder = req_builder.header(key.as_str(), value.as_str());
        }
        
        // 添加代理
        if let Some(ref proxy_info) = proxy {
            match proxy_info.to_reqwest_proxy() {
                Ok(reqwest_proxy) => {
                    req_builder = req_builder.proxy(reqwest_proxy);
                }
                Err(e) => {
                    warn!("代理构建失败: {}", e);
                }
            }
        }
        
        // 添加请求体
        if let Some(body) = &request.body {
            req_builder = req_builder.body(body.clone());
        }
        
        // 发送请求（带重试）
        for attempt in 0..=self.config.retry_times {
            match req_builder.try_clone() {
                Some(req) => {
                    match req.send().await {
                        Ok(response) => {
                            let status_code = response.status().as_u16();
                            let final_url = response.url().to_string();
                            
                            // 提取响应头
                            let mut headers = HashMap::new();
                            for (key, value) in response.headers().iter() {
                                if let Ok(v) = value.to_str() {
                                    headers.insert(key.to_string(), v.to_string());
                                }
                            }
                            
                            // 读取响应体
                            match response.bytes().await {
                                Ok(bytes) => {
                                    // 记录成功
                                    if let Some(ref p) = proxy {
                                        self.proxy_pool.record_result(&p.address, true).await;
                                    }
                                    
                                    let response_time = start.elapsed().as_millis() as u64;
                                    
                                    debug!(
                                        "请求成功: {} -> {} ({}ms)",
                                        request.url, status_code, response_time
                                    );
                                    
                                    return CrawlResponse {
                                        request_id: request.id,
                                        status_code,
                                        headers,
                                        body: bytes.to_vec(),
                                        final_url,
                                        response_time_ms: response_time,
                                        proxy: proxy.map(|p| p.address),
                                        error: None,
                                    };
                                }
                                Err(e) => {
                                    last_error = Some(e.to_string());
                                    warn!("响应读取失败 (attempt {}): {}", attempt + 1, e);
                                }
                            }
                        }
                        Err(e) => {
                            last_error = Some(e.to_string());
                            
                            // 判断是否是可重试的错误
                            if e.is_timeout() || e.is_connect() {
                                warn!("请求失败 (attempt {}): {}", attempt + 1, e);
                                continue;
                            }
                            
                            // 其他错误不重试
                            break;
                        }
                    }
                }
                None => {
                    // 无法克隆请求（请求体是流），只尝试一次
                    break;
                }
            }
        }
        
        // 所有重试都失败
        let response_time = start.elapsed().as_millis() as u64;
        
        if let Some(ref p) = proxy {
            self.proxy_pool.record_result(&p.address, false).await;
        }
        
        error!(
            "请求最终失败: {} ({}ms) - {}",
            request.url, response_time, last_error.as_deref().unwrap_or("unknown")
        );
        
        CrawlResponse {
            request_id: request.id,
            status_code: 0,
            headers: HashMap::new(),
            body: Vec::new(),
            final_url: request.url,
            response_time_ms: response_time,
            proxy: proxy.map(|p| p.address),
            error: last_error,
        }
    }
    
    /// 批量获取
    pub async fn fetch_batch(&self, requests: Vec<CrawlRequest>) -> Vec<CrawlResponse> {
        let futures = requests.into_iter().map(|req| self.fetch(req));
        let results = futures_util::future::join_all(futures).await;
        results
    }
}
```

```rust
// 作用：解析 HTML 内容，提取数据

use crate::models::{ArticleItem, ProductItem, Item};
use scraper::{Html, Selector, ElementRef};

// ==================== HTML 解析器 ====================

/// HTML 解析器
pub struct HtmlParser;

impl HtmlParser {
    /// 解析文章
    pub fn parse_article(html: &str, base_url: &str) -> Option<ArticleItem> {
        let document = Html::parse_document(html);
        
        // 选择器定义
        let title_selector = Selector::parse("h1.title, h1.article-title, .article-title, article h1, .title")
            .ok()?;
        let content_selector = Selector::parse("article.content, .article-content, .content, article, .post-content, .entry-content")
            .ok()?;
        let author_selector = Selector::parse(".author, .byline, [rel=author], .article-author")
            .ok()?;
        let date_selector = Selector::parse("time, .date, .published, [datetime]")
            .ok()?;
        let summary_selector = Selector::parse(".summary, .excerpt, .description, meta[name=description]")
            .ok()?;
        let tags_selector = Selector::parse(".tags a, .tag, [rel=tag]")
            .ok()?;
        
        // 提取标题
        let title = document
            .select(&title_selector)
            .next()
            .map(|el| el.text().collect::<String>().trim().to_string())
            .unwrap_or_else(|| "Untitled".to_string());
        
        // 创建文章项
        let mut item = ArticleItem::new(base_url, &title);
        
        // 提取内容
        if let Some(content_el) = document.select(&content_selector).next() {
            // 提取文本并清理
            let content = Self::clean_html(&content_el.html());
            item.content = Some(content);
        }
        
        // 提取作者
        item.author = document
            .select(&author_selector)
            .next()
            .map(|el| el.text().collect::<String>().trim().to_string());
        
        // 提取发布日期
        if let Some(date_el) = document.select(&date_selector).next() {
            let datetime = date_el.value().attr("datetime")
                .or_else(|| date_el.value().attr("title"));
            
            if let Some(dt) = datetime {
                // 尝试解析日期
                if let Ok(parsed) = chrono::DateTime::parse_from_rfc3339(dt) {
                    item.published_at = Some(parsed.with_timezone(&chrono::Utc));
                }
            }
        }
        
        // 提取摘要
        if let Some(summary_el) = document.select(&summary_selector).next() {
            if summary_el.value().name() == "meta" {
                item.summary = summary_el.value().attr("content").map(|s| s.to_string());
            } else {
                item.summary = Some(summary_el.text().collect::<String>().trim().to_string());
            }
        }
        
        // 提取标签
        item.tags = document
            .select(&tags_selector)
            .filter_map(|el| {
                let text = el.text().collect::<String>().trim().to_string();
                if text.is_empty() { None } else { Some(text) }
            })
            .collect();
        
        Some(item)
    }
    
    /// 解析产品
    pub fn parse_product(html: &str, base_url: &str) -> Option<ProductItem> {
        let document = Html::parse_document(html);
        
        // 选择器定义
        let name_selector = Selector::parse("h1.product-title, .product-name, h1[itemprop=name]")
            .ok()?;
        let price_selector = Selector::parse(".price, [itemprop=price], .product-price")
            .ok()?;
        let desc_selector = Selector::parse(".description, [itemprop=description], .product-description")
            .ok()?;
        let image_selector = Selector::parse("img[itemprop=image], .product-image img, .product img")
            .ok()?;
        
        // 提取名称
        let name = document
            .select(&name_selector)
            .next()
            .map(|el| el.text().collect::<String>().trim().to_string())
            .unwrap_or_else(|| "Unknown Product".to_string());
        
        let mut item = ProductItem::new(base_url, &name);
        
        // 提取价格
        if let Some(price_el) = document.select(&price_selector).next() {
            let price_text = price_el.text().collect::<String>();
            // 提取数字
            let price: f64 = price_text
                .chars()
                .filter(|c| c.is_ascii_digit() || *c == '.')
                .collect::<String>()
                .parse()
                .unwrap_or(0.0);
            
            if price > 0.0 {
                item.price = Some(price);
            }
        }
        
        // 提取描述
        item.description = document
            .select(&desc_selector)
            .next()
            .map(|el| Self::clean_html(&el.html()));
        
        // 提取图片
        item.image_url = document
            .select(&image_selector)
            .next()
            .and_then(|el| el.value().attr("src").map(|s| s.to_string()));
        
        Some(item)
    }
    
    /// 解析链接
    pub fn parse_links(html: &str, base_url: &str) -> Vec<String> {
        let document = Html::parse_document(html);
        let selector = Selector::parse("a[href]").ok();
        
        let base = url::Url::parse(base_url).ok();
        
        selector
            .map(|selector| {
                document
                    .select(&selector)
                    .filter_map(|el| {
                        el.value().attr("href")
                            .and_then(|href| {
                                // 解析相对 URL
                                if let Ok(full_url) = base.as_ref().map(|b| b.join(href)) {
                                    if full_url.scheme() == "http" || full_url.scheme() == "https" {
                                        return Some(full_url.to_string());
                                    }
                                }
                                None
                            })
                    })
                    .collect()
            })
            .unwrap_or_default()
    }
    
    /// 清理 HTML（移除脚本、样式等）
    fn clean_html(html: &str) -> String {
        let document = Html::parse_fragment(html);
        
        // 移除 script 和 style
        let clean: String = document
            .root_element()
            .descendants()
            .filter(|node| {
                if let Some(elem) = node.value().as_element() {
                    let tag = elem.name();
                    tag != "script" && tag != "style" && tag != "noscript"
                } else {
                    true
                }
            })
            .text()
            .collect::<Vec<_>>()
            .join(" ")
            .split_whitespace()
            .collect::<Vec<_>>()
            .join(" ");
        
        clean
    }
    
    /// 提取 meta 标签
    pub fn extract_meta(html: &str) -> Map<String, Value> {
        let document = Html::parse_document(html);
        let selector = Selector::parse("meta[name], meta[property]").ok();
        
        let mut meta = Map::new();
        
        if let Some(selector) = selector {
            for el in document.select(&selector) {
                if let (Some(name), Some(content)) = (
                    el.value().attr("name").or_else(|| el.value().attr("property")),
                    el.value().attr("content"),
                ) {
                    meta.insert(name.to_string(), Value::String(content.to_string()));
                }
            }
        }
        
        meta
    }
}
```

```rust
// 作用：处理和存储爬取的数据

use crate::models::{ArticleItem, ProductItem, Item};
use crate::storage::SharedStorage;
use std::sync::Arc;
use tokio::sync::mpsc;
use tracing::{debug, error, info};

// ==================== 管道项 ====================

/// 管道消息
enum PipelineMessage {
    /// 文章数据
    Article(ArticleItem),
    /// 产品数据
    Product(ProductItem),
    /// 原始 JSON
    Json(String, serde_json::Value),
    /// 错误
    Error(String),
}

// ==================== 数据管道 ====================

/// 数据处理管道
pub struct Pipeline {
    /// 存储
    storage: SharedStorage,
    
    /// 发送通道
    sender: mpsc::Sender<PipelineMessage>,
}

impl Pipeline {
    /// 创建新的管道
    pub fn new(storage: SharedStorage) -> Self {
        let (sender, mut receiver) = mpsc::channel::<PipelineMessage>(1000);
        
        // 创建管道并启动处理循环
        let pipeline = Self {
            storage,
            sender,
        };
        
        // 启动处理任务
        let storage_clone = pipeline.storage.clone();
        tokio::spawn(async move {
            while let Some(msg) = receiver.recv().await {
                match msg {
                    PipelineMessage::Article(item) => {
                        if let Err(e) = storage_clone.save_item(&item).await {
                            error!("保存文章失败: {}", e);
                        } else {
                            debug!("保存文章: {}", item.title);
                        }
                    }
                    PipelineMessage::Product(item) => {
                        if let Err(e) = storage_clone.save_item(&item).await {
                            error!("保存产品失败: {}", e);
                        } else {
                            debug!("保存产品: {}", item.name);
                        }
                    }
                    PipelineMessage::Json(key, value) => {
                        if let Err(e) = storage_clone.save_json(&key, &value).await {
                            error!("保存 JSON 失败: {}", e);
                        }
                    }
                    PipelineMessage::Error(err) => {
                        error!("管道错误: {}", err);
                    }
                }
            }
        });
        
        pipeline
    }
    
    /// 发送文章
    pub fn send_article(&self, item: ArticleItem) {
        let _ = self.sender.try_send(PipelineMessage::Article(item));
    }
    
    /// 发送产品
    pub fn send_product(&self, item: ProductItem) {
        let _ = self.sender.try_send(PipelineMessage::Product(item));
    }
    
    /// 发送 JSON
    pub fn send_json(&self, key: &str, value: serde_json::Value) {
        let _ = self.sender.try_send(PipelineMessage::Json(key.to_string(), value));
    }
    
    /// 发送错误
    pub fn send_error(&self, error: &str) {
        let _ = self.sender.try_send(PipelineMessage::Error(error.to_string()));
    }
}

pub type SharedPipeline = Arc<Pipeline>;
```

```rust
// scheduler/task.rs - 任务调度器
// 作用：管理爬取任务队列

use crate::models::CrawlRequest;
use crate::crawler::{Fetcher, SharedFetcher};
use crate::storage::{SharedDatabaseStorage, SharedCache};
use crate::proxy::SharedProxyPool;
use crate::config::CrawlerConfig;
use dashmap::DashMap;
use std::sync::Arc;
use std::collections::BinaryHeap;
use std::cmp::Ordering;
use tokio::sync::{RwLock, mpsc};
use tracing::{debug, info, warn};

// ==================== 任务项 ====================

/// 爬取任务
struct TaskItem {
    request: CrawlRequest,
    schedule_time: std::time::Instant,
}

impl Ord for TaskItem {
    fn cmp(&self, other: &Self) -> Ordering {
        // 优先级高的先执行
        // 如果优先级相同，时间早的先执行
        other.request.priority.cmp(&self.request.priority)
            .then_with(|| other.schedule_time.cmp(&self.schedule_time))
    }
}

impl PartialOrd for TaskItem {
    fn partial_cmp(&self, other: &Self) -> Option<Ordering> {
        Some(self.cmp(other))
    }
}

// ==================== 任务调度器 ====================

/// 任务调度器
pub struct Scheduler {
    /// 任务队列（优先级队列）
    queue: RwLock<BinaryHeap<TaskItem>>,
    
    /// 已爬取的 URL（去重）
    crawled_urls: DashMap<String, std::time::Instant>,
    
    /// 活跃任务数
    active_tasks: Arc<RwLock<usize>>,
    
    /// 最大并发
    max_concurrent: usize,
    
    /// 配置
    config: CrawlerConfig,
}

impl Scheduler {
    /// 创建新的调度器
    pub fn new(config: CrawlerConfig) -> Self {
        Self {
            queue: RwLock::new(BinaryHeap::new()),
            crawled_urls: DashMap::new(),
            active_tasks: Arc::new(RwLock::new(0)),
            max_concurrent: config.max_concurrent,
            config,
        }
    }
    
    /// 添加任务
    pub async fn add_task(&self, request: CrawlRequest) -> bool {
        // 检查是否已爬取
        if self.is_crawled(&request.url) {
            debug!("URL 已爬取，跳过: {}", request.url);
            return false;
        }
        
        let task = TaskItem {
            request,
            schedule_time: std::time::Instant::now(),
        };
        
        self.queue.write().await.push(task);
        true
    }
    
    /// 添加 URL 列表
    pub async fn add_urls(&self, urls: Vec<String>, priority: i32) -> usize {
        let mut added = 0;
        
        for url in urls {
            let request = CrawlRequest::get(&url).with_priority(priority);
            if self.add_task(request).await {
                added += 1;
            }
        }
        
        info!("添加 {} 个 URL 到任务队列", added);
        added
    }
    
    /// 获取下一个任务
    pub async fn next_task(&self) -> Option<CrawlRequest> {
        // 检查是否达到并发上限
        let active = *self.active_tasks.read().await;
        if active >= self.max_concurrent {
            return None;
        }
        
        // 取出优先级最高的任务
        let task = self.queue.write().await.pop()?;
        
        // 标记为活跃
        *self.active_tasks.write().await += 1;
        
        Some(task.request)
    }
    
    /// 标记任务完成
    pub async fn task_complete(&self, url: &str) {
        // 标记为已爬取
        self.mark_crawled(url);
        
        // 减少活跃任务数
        *self.active_tasks.write().await = self.active_tasks.read().await.saturating_sub(1);
    }
    
    /// 检查 URL 是否已爬取
    pub fn is_crawled(&self, url: &str) -> bool {
        self.crawled_urls.contains_key(url)
    }
    
    /// 标记 URL 已爬取
    pub fn mark_crawled(&self, url: &str) {
        self.crawled_urls.insert(url.to_string(), std::time::Instant::now());
    }
    
    /// 获取队列大小
    pub async fn queue_size(&self) -> usize {
        self.queue.read().await.len()
    }
    
    /// 获取活跃任务数
    pub async fn active_count(&self) -> usize {
        *self.active_tasks.read().await
    }
    
    /// 清理过期记录
    pub fn cleanup_expired(&self, max_age_hours: u64) {
        let max_age = std::time::Duration::from_secs(max_age_hours * 3600);
        let now = std::time::Instant::now();
        
        self.crawled_urls.retain(|_, &time| now.duration_since(time) < max_age);
    }
}

/// 共享调度器
pub type SharedScheduler = Arc<Scheduler>;

/// 创建共享调度器
pub fn create_scheduler(config: CrawlerConfig) -> SharedScheduler {
    Arc::new(Scheduler::new(config))
}
```

## 面试要点

### 面试题 1：Tokio 中如何控制并发数量？

**标准答案**：
使用 `Semaphore` 控制并发许可数量。

```rust
let semaphore = Arc::new(Semaphore::new(max_concurrent));

// 每个任务获取许可
let permit = semaphore.acquire().await?;
do_work().await;
drop(permit); // 释放许可
```

### 面试题 2：async/await 和阻塞代码？

**标准答案**：
- async 代码不能直接调用阻塞代码
- 使用 `tokio::task::spawn_blocking` 处理 CPU 密集任务
- 或者使用专门的异步库

```rust
// 阻塞操作
let result = tokio::task::spawn_blocking(|| {
    // CPU 密集计算
    compute_heavy();
}).await?;
```

### 面试题 3：Rust 的 trait 对象池？

**标准答案**：
使用 `trait` 定义接口，`Arc<dyn Trait>` 实现对象池。

```rust
trait Item: Send {
    fn process(&self);
}

struct ItemPool {
    items: Vec<Arc<dyn Item>>,
}
```

### 面试题 4：爬虫的反反爬策略？

**标准答案**：
1. **User-Agent 轮换**：模拟不同浏览器
2. **代理池**：避免 IP 被封
3. **请求限流**：避免被识别为爬虫
4. **Cookie 处理**：保持会话
5. **延迟请求**：降低请求频率

### 面试题 5：Tokio 的任务调度原理？

**标准答案**：
1. **Work Stealing**：空闲线程从其他线程偷任务
2. **I/O 驱动**：使用 epoll/kqueue 等待 I/O
3. **Waker**：I/O 完成后唤醒任务
4. **Local Queue**：每个线程有自己的任务队列
