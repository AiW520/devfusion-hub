# Tokio 异步服务方案：方案2 - 实时消息推送

## 项目介绍

### 项目是什么
这是一个基于 Rust + Tokio 构建的实时消息推送服务，支持 WebSocket 长连接、房间管理、消息广播、私信、在线状态管理和心跳检测。

### 解决什么问题
- 实时聊天应用的后端服务
- 多人在线游戏的状态同步
- 实时协作应用的推送服务
- 在线教育平台的实时互动

### 核心技术亮点
| 特性 | 技术实现 |
|------|----------|
| WebSocket | tokio-tungstenite |
| 异步运行时 | Tokio |
| 消息路由 | 自定义路由器 |
| 会话管理 | Arc<RwLock<Map>> |
| 心跳检测 | Tokio Interval |
| 消息序列化 | serde + JSON |
| 广播 | 房间组播 |
| 限流 | 滑动窗口 |

### 适合什么场景
- 即时通讯应用
- 实时协作工具
- 在线游戏后端
- 通知推送服务

```rust
/// WebSocket 请求（客户端 → 服务器）
#[derive(Debug, Clone, Serialize, Deserialize)]
#[serde(tag = "action", content = "data")]
pub enum WsRequest {
    // 认证
    #[serde(rename = "auth")]
    Auth { token: String },
    
    // 加入房间
    #[serde(rename = "joinRoom")]
    JoinRoom { room_id: String },
    
    // 离开房间
    #[serde(rename = "leaveRoom")]
    LeaveRoom { room_id: String },
    
    // 发送消息
    #[serde(rename = "sendMessage")]
    SendMessage { content: String, target: Option<MessageTarget> },
    
    // 发送私聊
    #[serde(rename = "sendPrivate")]
    SendPrivate { to_user: String, content: String },
    
    // 发送房间消息
    #[serde(rename = "sendRoomMessage")]
    SendRoomMessage { room_id: String, content: String },
    
    // 心跳
    #[serde(rename = "heartbeat")]
    Heartbeat { timestamp: i64 },
    
    // 获取在线用户
    #[serde(rename = "getOnlineUsers")]
    GetOnlineUsers { room_id: Option<String> },
    
    // 获取房间列表
    #[serde(rename = "getRooms")]
    GetRooms,
}

/// WebSocket 响应（服务器 → 客户端）
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct WsResponse {
    /// 响应类型
    #[serde(rename = "type")]
    pub resp_type: ResponseType,
    
    /// 响应码
    pub code: u16,
    
    /// 消息
    pub message: String,
    
    /// 数据
    #[serde(skip_serializing_if = "Option::is_none")]
    pub data: Option<serde_json::Value>,
    
    /// 消息 ID（用于关联请求）
    #[serde(skip_serializing_if = "Option::is_none")]
    pub request_id: Option<String>,
}

impl WsResponse {
    /// 成功响应
    pub fn success(message: &str) -> Self {
        Self {
            resp_type: ResponseType::Success,
            code: 200,
            message: message.to_string(),
            data: None,
            request_id: None,
        }
    }
    
    /// 成功响应（带数据）
    pub fn success_data<T: Serialize>(message: &str, data: T) -> Self {
        Self {
            resp_type: ResponseType::Success,
            code: 200,
            message: message.to_string(),
            data: Some(serde_json::to_value(data).unwrap_or(serde_json::Value::Null)),
            request_id: None,
        }
    }
    
    /// 错误响应
    pub fn error(code: u16, message: &str) -> Self {
        Self {
            resp_type: ResponseType::Error,
            code,
            message: message.to_string(),
            data: None,
            request_id: None,
        }
    }
    
    /// 消息推送
    pub fn push(message: ChatMessage) -> Self {
        Self {
            resp_type: ResponseType::Push,
            code: 0,
            message: String::new(),
            data: Some(serde_json::to_value(message).unwrap_or(serde_json::Value::Null)),
            request_id: None,
        }
    }
    
    /// 设置请求 ID
    pub fn with_request_id(mut self, request_id: &str) -> Self {
        self.request_id = Some(request_id.to_string());
        self
    }
}

/// 响应类型
#[derive(Debug, Clone, Serialize, Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum ResponseType {
    Success,
    Error,
    Push,
}
```

```rust
        Self {
            id,
            user: None,
            sender,
            remote_addr,
            rooms: std::collections::HashSet::new(),
            created_at: Utc::now(),
            last_heartbeat: Utc::now(),
            authenticated: false,
        }
    }
    
    /// 设置用户
    pub fn set_user(&mut self, user: UserInfo) {
        self.user = Some(user);
        self.authenticated = true;
    }
    
    /// 更新心跳
    pub fn update_heartbeat(&mut self) {
        self.last_heartbeat = Utc::now();
    }
    
    /// 加入房间
    pub fn join_room(&mut self, room_id: &str) -> bool {
        self.rooms.insert(room_id.to_string())
    }
    
    /// 离开房间
    pub fn leave_room(&mut self, room_id: &str) -> bool {
        self.rooms.remove(room_id)
    }
    
    /// 是否在房间中
    pub fn is_in_room(&self, room_id: &str) -> bool {
        self.rooms.contains(room_id)
    }
    
    /// 发送消息
    pub fn send(&self, message: &str) -> bool {
        self.sender.send(message.to_string()).is_ok()
    }
    
    /// 是否在线（心跳超时检测）
    pub fn is_alive(&self, timeout_secs: i64) -> bool {
        let elapsed = Utc::now().signed_duration_since(self.last_heartbeat);
        elapsed.num_seconds() < timeout_secs
    }
}
```

```rust
        Self {
            room_id: room_id.to_string(),
            name: name.to_string(),
            description: None,
            room_type: RoomType::Public,
            members: HashSet::new(),
            max_members: None,
            creator: creator.to_string(),
            created_at: Utc::now(),
            admins: HashSet::from([creator.to_string()]),
            private: false,
        }
    }
    
    /// 创建私密房间
    pub fn new_private(room_id: &str, name: &str, creator: &str) -> Self {
        let mut room = Self::new(room_id, name, creator);
        room.private = true;
        room.room_type = RoomType::Private;
        room
    }
    
    /// 添加成员
    pub fn add_member(&mut self, user_id: &str) -> bool {
        if let Some(max) = self.max_members {
            if self.members.len() >= max {
                return false;
            }
        }
        self.members.insert(user_id.to_string())
    }
    
    /// 移除成员
    pub fn remove_member(&mut self, user_id: &str) -> bool {
        self.members.remove(user_id)
    }
    
    /// 是否是成员
    pub fn is_member(&self, user_id: &str) -> bool {
        self.members.contains(user_id)
    }
    
    /// 是否是管理员
    pub fn is_admin(&self, user_id: &str) -> bool {
        self.admins.contains(user_id)
    }
    
    /// 获取成员数
    pub fn member_count(&self) -> usize {
        self.members.len()
    }
    
    /// 是否已满
    pub fn is_full(&self) -> bool {
        self.max_members.map_or(false, |max| self.members.len() >= max)
    }
}
```

```rust
// services/session.rs - 会话管理服务
// 作用：管理所有 WebSocket 连接会话

use crate::models::{Session, SessionStats, UserInfo, WsResponse};
use crate::models::message::{ChatMessage, MessageType, MessageTarget};
use chrono::Utc;
use dashmap::DashMap;
use flume::Sender;
use std::net::SocketAddr;
use std::sync::Arc;
use tokio::sync::RwLock;
use tracing::{info, warn, debug};

// ==================== 会话管理器 ====================

/// 会话管理器
/// 使用 DashMap 实现高效的并发 HashMap
pub struct SessionManager {
    /// 会话存储：session_id → Session
    sessions: DashMap<String, Session>,
    
    /// 用户会话映射：user_id → session_id
    /// 一个用户只能有一个活跃会话
    user_sessions: DashMap<String, String>,
    
    /// 广播频道
    broadcast_tx: flume::Sender<BroadcastMessage>,
}

impl SessionManager {
    /// 创建新的会话管理器
    pub fn new(broadcast_tx: flume::Sender<BroadcastMessage>) -> Self {
        Self {
            sessions: DashMap::new(),
            user_sessions: DashMap::new(),
            broadcast_tx,
        }
    }
    
    /// 创建会话
    pub fn create_session(
        &self,
        session_id: String,
        sender: Sender<String>,
        remote_addr: SocketAddr,
    ) -> Arc<RwLock<Session>> {
        let session = Session::new(session_id.clone(), sender, remote_addr);
        let session = Arc::new(RwLock::new(session));
        
        self.sessions.insert(session_id.clone(), session.clone());
        
        debug!("创建新会话: {}", session_id);
        
        session
    }
    
    /// 获取会话
    pub fn get_session(&self, session_id: &str) -> Option<Arc<RwLock<Session>>> {
        self.sessions.get(session_id).map(|r| r.value().clone())
    }
    
    /// 通过用户 ID 获取会话
    pub fn get_session_by_user(&self, user_id: &str) -> Option<Arc<RwLock<Session>>> {
        self.user_sessions
            .get(user_id)
            .and_then(|session_id| self.sessions.get(session_id.as_str()))
            .map(|r| r.value().clone())
    }
    
    /// 移除会话
    pub fn remove_session(&self, session_id: &str) -> Option<Arc<RwLock<Session>>> {
        // 获取会话并移除用户映射
        if let Some(session) = self.sessions.remove(session_id) {
            let session = session.1;
            
            // 如果有用户，移除用户映射
            let user_id = session.read().await.user.as_ref().map(|u| u.user_id.clone());
            if let Some(user_id) = user_id {
                self.user_sessions.remove(&user_id);
            }
            
            info!("移除会话: {}", session_id);
            
            return Some(session);
        }
        
        None
    }
    
    /// 设置用户信息
    pub async fn set_user(&self, session_id: &str, user: UserInfo) -> bool {
        if let Some(session) = self.get_session(session_id) {
            let mut session = session.write().await;
            session.set_user(user.clone());
            
            // 建立用户到会话的映射
            self.user_sessions.insert(user.user_id.clone(), session_id.to_string());
            
            info!("用户 {} 认证成功，会话: {}", user.user_id, session_id);
            
            return true;
        }
        
        false
    }
    
    /// 获取在线用户数
    pub fn online_count(&self) -> usize {
        self.sessions.len()
    }
    
    /// 获取已认证用户数
    pub async fn authenticated_count(&self) -> usize {
        let mut count = 0;
        for session in self.sessions.iter() {
            if session.read().await.authenticated {
                count += 1;
            }
        }
        count
    }
    
    /// 获取所有在线用户
    pub async fn get_online_users(&self, room_id: Option<&str>) -> Vec<UserInfo> {
        let mut users = Vec::new();
        
        for session in self.sessions.iter() {
            let session = session.read().await;
            if session.authenticated {
                if let Some(ref user) = session.user {
                    // 如果指定了房间，只返回在房间中的用户
                    if let Some(room_id) = room_id {
                        if session.is_in_room(room_id) {
                            users.push(user.clone());
                        }
                    } else {
                        users.push(user.clone());
                    }
                }
            }
        }
        
        users
    }
    
    /// 获取会话统计
    pub async fn get_stats(&self) -> SessionStats {
        let mut stats = SessionStats::default();
        
        for session in self.sessions.iter() {
            let session = session.read().await;
            stats.total_connections += 1;
            
            if session.authenticated {
                stats.authenticated_users += 1;
            } else {
                stats.anonymous_connections += 1;
            }
        }
        
        stats
    }
    
    /// 广播消息给所有在线用户
    pub fn broadcast(&self, message: ChatMessage) {
        let broadcast = BroadcastMessage {
            target: MessageTarget::All,
            message,
            exclude_session: None,
        };
        
        let _ = self.broadcast_tx.send(broadcast);
    }
    
    /// 发送消息给指定用户
    pub async fn send_to_user(&self, user_id: &str, response: &WsResponse) -> bool {
        if let Some(session) = self.get_session_by_user(user_id) {
            let session = session.read().await;
            let json = serde_json::to_string(response).unwrap_or_default();
            session.send(&json)
        } else {
            false
        }
    }
    
    /// 更新心跳
    pub async fn update_heartbeat(&self, session_id: &str) -> bool {
        if let Some(session) = self.get_session(session_id) {
            let mut session = session.write().await;
            session.update_heartbeat();
            return true;
        }
        false
    }
    
    /// 清理超时会话
    pub fn cleanup_stale_sessions(&self, timeout_secs: i64) -> Vec<String> {
        let mut removed = Vec::new();
        let now = Utc::now();
        
        // 遍历并清理
        for session in self.sessions.iter() {
            let is_stale = {
                let session = session.read().await;
                let elapsed = now.signed_duration_since(session.last_heartbeat);
                elapsed.num_seconds() >= timeout_secs
            };
            
            if is_stale {
                removed.push(session.key().clone());
            }
        }
        
        // 移除超时会话
        for session_id in &removed {
            self.remove_session(session_id);
        }
        
        if !removed.is_empty() {
            warn!("清理了 {} 个超时会话", removed.len());
        }
        
        removed
    }
}
```

```rust
// services/heartbeat.rs - 心跳检测服务
// 作用：定期检测连接状态，清理无效连接

use crate::services::session::{SharedSessionManager, create_session_manager};
use crate::services::room::SharedRoomManager;
use flume::Receiver;
use std::sync::Arc;
use std::time::Duration;
use tokio::time;
use tracing::{info, warn};

/// 心跳服务
pub struct HeartbeatService {
    /// 心跳间隔
    interval: Duration,
    /// 心跳超时时间
    timeout: Duration,
    /// 清理间隔
    cleanup_interval: Duration,
}

impl HeartbeatService {
    /// 创建心跳服务
    pub fn new(interval_secs: u64, timeout_secs: i64) -> Self {
        Self {
            interval: Duration::from_secs(interval_secs),
            timeout: Duration::from_secs(timeout_secs as u64),
            cleanup_interval: Duration::from_secs(interval_secs * 2),
        }
    }
    
    /// 启动心跳服务
    pub async fn start(
        &self,
        session_manager: SharedSessionManager,
        room_manager: SharedRoomManager,
        broadcast_rx: Receiver<crate::services::session::BroadcastMessage>,
    ) {
        let mut interval = time::interval(self.interval);
        let mut cleanup_interval = time::interval(self.cleanup_interval);
        
        loop {
            tokio::select! {
                // 定期心跳检查
                _ = interval.tick() => {
                    self.check_heartbeat(&session_manager).await;
                }
                
                // 定期清理超时会话
                _ = cleanup_interval.tick() => {
                    self.cleanup_sessions(&session_manager, &room_manager).await;
                }
                
                // 处理广播消息
                msg = broadcast_rx.recv_async() => {
                    match msg {
                        Ok(broadcast) => {
                            self.handle_broadcast(&session_manager, broadcast).await;
                        }
                        Err(_) => {
                            // Channel 关闭
                            break;
                        }
                    }
                }
            }
        }
    }
    
    /// 检查心跳
    async fn check_heartbeat(&self, session_manager: &SharedSessionManager) {
        // 发送心跳请求
        let ping_message = serde_json::json!({
            "type": "success",
            "code": 0,
            "message": "pong",
            "data": {
                "timestamp": chrono::Utc::now().timestamp_millis()
            }
        });
        
        let ping_str = serde_json::to_string(&ping_message).unwrap_or_default();
        
        // 遍历所有会话
        for session in session_manager.sessions.iter() {
            let session = session.value();
            
            // 发送心跳
            if let Err(e) = session.read().await.sender.send(ping_str.clone()) {
                warn!("心跳发送失败: {}", e);
            }
        }
    }
    
    /// 清理超时会话
    async fn cleanup_sessions(
        &self,
        session_manager: &SharedSessionManager,
        room_manager: &SharedRoomManager,
    ) {
        let removed = session_manager.cleanup_stale_sessions(self.timeout.as_secs() as i64);
        
        // 从房间中移除
        for session_id in &removed {
            if let Some(session) = session_manager.get_session(session_id) {
                let session = session.read().await;
                
                // 离开所有房间
                for room_id in &session.rooms {
                    room_manager.leave_room(room_id, &session.user.as_ref().map(|u| u.user_id.clone()).unwrap_or_default());
                }
            }
        }
    }
    
    /// 处理广播消息
    async fn handle_broadcast(
        &self,
        session_manager: &SharedSessionManager,
        broadcast: crate::services::session::BroadcastMessage,
    ) {
        use crate::models::message::MessageTarget;
        
        let message = &broadcast.message;
        let message_json = serde_json::to_string(&WsResponse::push(message.clone())).unwrap_or_default();
        
        match &broadcast.target {
            MessageTarget::All => {
                // 广播给所有人
                for session in session_manager.sessions.iter() {
                    if Some(session.key().as_str()) != broadcast.exclude_session.as_deref() {
                        let _ = session.read().await.sender.send(message_json.clone());
                    }
                }
            }
            
            MessageTarget::User { user_id } => {
                // 发送给指定用户
                if let Some(session) = session_manager.get_session_by_user(user_id) {
                    let _ = session.read().await.sender.send(message_json.clone());
                }
            }
            
            MessageTarget::Room { room_id } => {
                // 发送给房间成员
                let members = room_manager.get_members(room_id);
                
                for user_id in members {
                    if let Some(session) = session_manager.get_session_by_user(&user_id) {
                        if Some(session.read().await.id.as_str()) != broadcast.exclude_session.as_deref() {
                            let _ = session.read().await.sender.send(message_json.clone());
                        }
                    }
                }
            }
            
            MessageTarget::Users { user_ids } => {
                // 发送给多个用户
                for user_id in user_ids {
                    if let Some(session) = session_manager.get_session_by_user(user_id) {
                        if Some(session.read().await.id.as_str()) != broadcast.exclude_session.as_deref() {
                            let _ = session.read().await.sender.send(message_json.clone());
                        }
                    }
                }
            }
        }
    }
}

/// 启动心跳服务
pub fn start_heartbeat_service(
    session_manager: SharedSessionManager,
    room_manager: SharedRoomManager,
    broadcast_rx: flume::Receiver<crate::services::session::BroadcastMessage>,
    heartbeat_interval_secs: u64,
    heartbeat_timeout_secs: i64,
) {
    let service = HeartbeatService::new(heartbeat_interval_secs, heartbeat_timeout_secs);
    
    tokio::spawn(async move {
        info!("启动心跳服务");
        service.start(session_manager, room_manager, broadcast_rx).await;
    });
}
```

```rust
// ws/handler.rs - WebSocket 连接处理器
// 作用：处理 WebSocket 连接和消息

use crate::models::message::{ChatMessage, WsRequest, WsResponse, ResponseType, MessageType, MessageTarget};
use crate::models::user::UserInfo;
use crate::services::session::{SharedSessionManager, create_session_manager};
use crate::services::room::SharedRoomManager;
use crate::services::heartbeat::start_heartbeat_service;
use flume::{Receiver, Sender};
use futures_util::{SinkExt, StreamExt};
use std::net::SocketAddr;
use std::sync::Arc;
use tokio::net::TcpStream;
use tokio_tungstenite::{accept_async, tungstenite::Message};
use tracing::{info, warn, error, debug};
use uuid::Uuid;

// ==================== WebSocket 处理器 ====================

/// WebSocket 处理器
pub struct WsHandler {
    /// 会话管理器
    session_manager: SharedSessionManager,
    
    /// 房间管理器
    room_manager: SharedRoomManager,
    
    /// 广播接收者
    broadcast_rx: Receiver<crate::services::session::BroadcastMessage>,
}

impl WsHandler {
    /// 创建新的 WebSocket 处理器
    pub fn new(
        session_manager: SharedSessionManager,
        room_manager: SharedRoomManager,
        broadcast_rx: Receiver<crate::services::session::BroadcastMessage>,
    ) -> Self {
        Self {
            session_manager,
            room_manager,
            broadcast_rx,
        }
    }
    
    /// 处理新的 WebSocket 连接
    pub async fn handle_connection(
        &self,
        stream: TcpStream,
        remote_addr: SocketAddr,
    ) {
        let session_id = Uuid::new_v4().to_string();
        
        info!("新的 WebSocket 连接: {} (session: {})", remote_addr, session_id);
        
        // 接受 WebSocket 连接
        let ws_stream = match accept_async(stream).await {
            Ok(ws) => ws,
            Err(e) => {
                error!("WebSocket 握手失败: {}", e);
                return;
            }
        };
        
        let (mut write, mut read) = ws_stream.split();
        
        // 创建消息 channel
        let (tx, mut rx) = flume::bounded::<String>(256);
        
        // 创建会话
        let session = self.session_manager.create_session(
            session_id.clone(),
            tx,
            remote_addr,
        );
        
        // 启动消息发送任务
        let session_clone = session.clone();
        tokio::spawn(async move {
            while let Some(msg) = rx.recv_async().await.ok() {
                if write.send(Message::Text(msg)).await.is_err() {
                    break;
                }
            }
            
            // 连接关闭，移除会话
            info!("WebSocket 连接断开: session {}", session_clone.read().await.id);
        });
        
        // 发送欢迎消息
        let welcome = WsResponse::success_data("欢迎连接", serde_json::json!({
            "sessionId": session_id,
            "serverTime": chrono::Utc::now().timestamp_millis(),
        }));
        let _ = session.read().await.sender.send(serde_json::to_string(&welcome).unwrap_or_default());
        
        // 处理消息循环
        while let Some(msg_result) = read.next().await {
            match msg_result {
                Ok(Message::Text(text)) => {
                    self.handle_message(&session_id, &text).await;
                }
                Ok(Message::Binary(data)) => {
                    if let Ok(text) = String::from_utf8(data) {
                        self.handle_message(&session_id, &text).await;
                    }
                }
                Ok(Message::Ping(data)) => {
                    // 自动响应 Pong
                    let _ = session.read().await.sender.send(
                        serde_json::to_string(&serde_json::json!({
                            "type": "pong",
                            "data": data
                        })).unwrap_or_default()
                    );
                }
                Ok(Message::Pong(_)) => {
                    // 更新心跳
                    let _ = self.session_manager.update_heartbeat(&session_id).await;
                }
                Ok(Message::Close(_)) => {
                    info!("客户端关闭连接: session {}", session_id);
                    break;
                }
                Err(e) => {
                    warn!("WebSocket 读取错误: {}", e);
                    break;
                }
            }
        }
        
        // 清理会话
        self.cleanup_session(&session_id).await;
    }
    
    /// 处理消息
    async fn handle_message(&self, session_id: &str, text: &str) {
        debug!("收到消息 (session {}): {}", session_id, text);
        
        // 解析请求
        let request: WsRequest = match serde_json::from_str(text) {
            Ok(req) => req,
            Err(e) => {
                let error = WsResponse::error(400, &format!("无效的消息格式: {}", e));
                self.send_to_session(session_id, &error).await;
                return;
            }
        };
        
        // 处理请求
        match request {
            WsRequest::Auth { token } => {
                self.handle_auth(session_id, &token).await;
            }
            
            WsRequest::JoinRoom { room_id } => {
                self.handle_join_room(session_id, &room_id).await;
            }
            
            WsRequest::LeaveRoom { room_id } => {
                self.handle_leave_room(session_id, &room_id).await;
            }
            
            WsRequest::SendMessage { content, target } => {
                self.handle_send_message(session_id, &content, target).await;
            }
            
            WsRequest::SendPrivate { to_user, content } => {
                self.handle_send_private(session_id, &to_user, &content).await;
            }
            
            WsRequest::SendRoomMessage { room_id, content } => {
                self.handle_send_room_message(session_id, &room_id, &content).await;
            }
            
            WsRequest::Heartbeat { timestamp } => {
                self.handle_heartbeat(session_id, timestamp).await;
            }
            
            WsRequest::GetOnlineUsers { room_id } => {
                self.handle_get_online_users(session_id, room_id.as_deref()).await;
            }
            
            WsRequest::GetRooms => {
                self.handle_get_rooms(session_id).await;
            }
        }
    }
    
    /// 处理认证
    async fn handle_auth(&self, session_id: &str, token: &str) {
        // 简化认证：使用 token 作为用户 ID
        // 实际应用中应该验证 JWT 或其他 token
        
        let user = UserInfo::new(token, &format!("User_{}", &token[..8.min(token.len())]));
        
        if self.session_manager.set_user(session_id, user.clone()).await {
            let response = WsResponse::success_data("认证成功", &user);
            self.send_to_session(session_id, &response).await;
        } else {
            let response = WsResponse::error(500, "认证失败");
            self.send_to_session(session_id, &response).await;
        }
    }
    
    /// 处理加入房间
    async fn handle_join_room(&self, session_id: &str, room_id: &str) {
        // 获取会话
        let user_id = match self.get_user_id(session_id).await {
            Some(id) => id,
            None => {
                let response = WsResponse::error(401, "请先认证");
                self.send_to_session(session_id, &response).await;
                return;
            }
        };
        
        // 加入房间
        match self.room_manager.join_room(room_id, &user_id) {
            Ok(room) => {
                // 更新会话
                if let Some(session) = self.session_manager.get_session(session_id) {
                    session.write().await.join_room(room_id);
                }
                
                let response = WsResponse::success_data("加入房间成功", serde_json::json!({
                    "room": {
                        "roomId": room.room_id,
                        "name": room.name,
                        "memberCount": room.member_count(),
                    }
                }));
                self.send_to_session(session_id, &response).await;
            }
            Err(e) => {
                let response = WsResponse::error(400, e);
                self.send_to_session(session_id, &response).await;
            }
        }
    }
    
    /// 处理离开房间
    async fn handle_leave_room(&self, session_id: &str, room_id: &str) {
        let user_id = match self.get_user_id(session_id).await {
            Some(id) => id,
            None => return,
        };
        
        if self.room_manager.leave_room(room_id, &user_id) {
            // 更新会话
            if let Some(session) = self.session_manager.get_session(session_id) {
                session.write().await.leave_room(room_id);
            }
            
            let response = WsResponse::success("已离开房间");
            self.send_to_session(session_id, &response).await;
        }
    }
    
    /// 处理发送消息
    async fn handle_send_message(&self, session_id: &str, content: &str, target: Option<MessageTarget>) {
        let (user_id, username) = match self.get_user_info(session_id).await {
            Some(info) => (info.user_id.clone(), info.username.clone()),
            None => {
                let response = WsResponse::error(401, "请先认证");
                self.send_to_session(session_id, &response).await;
                return;
            }
        };
        
        let message = ChatMessage::chat(content.to_string())
            .with_sender(&user_id, &username);
        
        // 根据目标发送
        match target {
            Some(MessageTarget::All) | None => {
                self.session_manager.broadcast(message);
            }
            Some(MessageTarget::User { user_id: target_id }) => {
                let response = WsResponse::push(message);
                let _ = self.session_manager.send_to_user(&target_id, &response).await;
            }
            Some(MessageTarget::Room { room_id }) => {
                self.room_manager.broadcast_to_room(&room_id, message);
            }
            Some(MessageTarget::Users { user_ids }) => {
                let response = WsResponse::push(message);
                for user_id in user_ids {
                    let _ = self.session_manager.send_to_user(&user_id, &response).await;
                }
            }
        }
        
        // 发送确认
        let ack = WsResponse::success("消息已发送");
        self.send_to_session(session_id, &ack).await;
    }
    
    /// 处理发送私信
    async fn handle_send_private(&self, session_id: &str, to_user: &str, content: &str) {
        let user_info = match self.get_user_info(session_id).await {
            Some(info) => info,
            None => {
                let response = WsResponse::error(401, "请先认证");
                self.send_to_session(session_id, &response).await;
                return;
            }
        };
        
        let message = ChatMessage::private(content.to_string(), to_user)
            .with_sender(&user_info.user_id, &user_info.username);
        
        let response = WsResponse::push(message);
        if !self.session_manager.send_to_user(to_user, &response).await {
            let response = WsResponse::error(404, "用户不在线");
            self.send_to_session(session_id, &response).await;
        }
    }
    
    /// 处理发送房间消息
    async fn handle_send_room_message(&self, session_id: &str, room_id: &str, content: &str) {
        let user_info = match self.get_user_info(session_id).await {
            Some(info) => info,
            None => {
                let response = WsResponse::error(401, "请先认证");
                self.send_to_session(session_id, &response).await;
                return;
            }
        };
        
        let session = match self.session_manager.get_session(session_id).await {
            Some(s) => s,
            None => return,
        };
        
        // 检查是否在房间中
        if !session.read().await.is_in_room(room_id) {
            let response = WsResponse::error(403, "你不在此房间中");
            self.send_to_session(session_id, &response).await;
            return;
        }
        
        let message = ChatMessage::room(room_id, content.to_string())
            .with_sender(&user_info.user_id, &user_info.username);
        
        self.room_manager.broadcast_to_room(room_id, message);
        
        let ack = WsResponse::success("消息已发送");
        self.send_to_session(session_id, &ack).await;
    }
    
    /// 处理心跳
    async fn handle_heartbeat(&self, session_id: &str, timestamp: i64) {
        let _ = self.session_manager.update_heartbeat(session_id).await;
        
        let response = WsResponse::success_data("心跳响应", serde_json::json!({
            "serverTime": chrono::Utc::now().timestamp_millis(),
            "clientTime": timestamp,
        }));
        self.send_to_session(session_id, &response).await;
    }
    
    /// 处理获取在线用户
    async fn handle_get_online_users(&self, session_id: &str, room_id: Option<&str>) {
        let users = self.session_manager.get_online_users(room_id).await;
        
        let response = WsResponse::success_data("在线用户列表", serde_json::json!({
            "users": users,
            "count": users.len(),
        }));
        self.send_to_session(session_id, &response).await;
    }
    
    /// 处理获取房间列表
    async fn handle_get_rooms(&self, session_id: &str) {
        let rooms = self.room_manager.get_all_rooms();
        
        let response = WsResponse::success_data("房间列表", serde_json::json!({
            "rooms": rooms,
        }));
        self.send_to_session(session_id, &response).await;
    }
    
    /// 发送消息到会话
    async fn send_to_session(&self, session_id: &str, response: &WsResponse) {
        if let Some(session) = self.session_manager.get_session(session_id) {
            let json = serde_json::to_string(response).unwrap_or_default();
            let _ = session.read().await.sender.send(json);
        }
    }
    
    /// 获取用户 ID
    async fn get_user_id(&self, session_id: &str) -> Option<String> {
        self.session_manager
            .get_session(session_id)
            .await?
            .read()
            .await
            .user
            .as_ref()
            .map(|u| u.user_id.clone())
    }
    
    /// 获取用户信息
    async fn get_user_info(&self, session_id: &str) -> Option<UserInfo> {
        self.session_manager
            .get_session(session_id)
            .await?
            .read()
            .await
            .user
            .clone()
    }
    
    /// 清理会话
    async fn cleanup_session(&self, session_id: &str) {
        // 移除会话
        if let Some(session) = self.session_manager.remove_session(session_id) {
            let session = session.read().await;
            
            // 离开所有房间
            for room_id in &session.rooms {
                self.room_manager.leave_room(room_id, &session.user.as_ref().map(|u| u.user_id.clone()).unwrap_or_default());
            }
            
            // 广播用户离线
            if let Some(user) = &session.user {
                let message = ChatMessage::system(&format!("用户 {} 已离线", user.username));
                self.session_manager.broadcast(message);
            }
        }
    }
}
```

## 面试要点

### 面试题 1：Tokio 的并发模型是什么？

**标准答案**：
1. **任务（Task）**：最小的调度单元
2. **工作线程池**：默认多线程运行时
3. **Waker 机制**：I/O 完成后唤醒任务
4. **协作式调度**：`await` 点释放执行权

```rust
// spawn 创建任务
let handle = tokio::spawn(async {
    do_something().await;
});

// join 并发等待多个任务
let (a, b) = tokio::join!(task1, task2);
```

### 面试题 2：async/await 和线程有什么区别？

**标准答案**：
| 对比项 | 异步 | 线程 |
|--------|------|------|
| 切换成本 | 低（微秒级） | 高（毫秒级） |
| 内存占用 | 低（KB 级） | 高（MB 级） |
| 阻塞 | 协作式 | 抢占式 |
| 适用场景 | I/O 密集 | CPU 密集 |

### 面试题 3：Channel 和共享内存的区别？

**标准答案**：
- **Channel**：消息传递，适合并发解耦
- **共享内存**：直接访问，适合高并发读写

```rust
// Channel
let (tx, rx) = flume::bounded(100);
tx.send_async(msg).await?;

// 共享内存
let data = Arc::new(RwLock::new(SharedData::new()));
let data_clone = data.clone();
tokio::spawn(async move {
    let mut guard = data_clone.write().await;
    guard.modify();
});
```

### 面试题 4：DashMap 和标准 HashMap 的区别？

**标准答案**：
- DashMap 是 sharded HashMap，减少锁竞争
- 适合读多写多的场景
- 跨 shard 操作仍然需要锁

```rust
use dashmap::DashMap;

let map = DashMap::new();
map.insert("key", "value");
let value = map.get("key").map(|r| r.value().clone());
```

### 面试题 5：WebSocket 的心跳机制？

**标准答案**：
1. 客户端定期发送心跳
2. 服务器响应并检测超时
3. 清理断开的连接

```rust
// 客户端发送
ws.send(Message::Ping(data)).await?;

// 服务器响应
if let Ok(Message::Ping(data)) = msg {
    ws.send(Message::Pong(data)).await?;
}
```
