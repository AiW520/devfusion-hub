# Actix + React 方案：方案1 - 博客平台

## 项目介绍

### 项目是什么
这是一个基于 Rust + Actix Web + React + PostgreSQL 构建的全栈博客平台，支持文章发布、评论、分类、标签、搜索和用户管理。

### 解决什么问题
- 独立的博客系统，无需第三方平台
- 高性能后端（Rust）
- 现代化前端（React）
- 支持 Markdown 编辑
- SEO 友好

### 核心技术亮点
| 特性 | 技术实现 |
|------|----------|
| 后端框架 | Actix-web 4 |
| ORM | SQLx（异步） |
| 数据库 | PostgreSQL |
| 前端框架 | React 18 + TypeScript |
| 状态管理 | Zustand |
| 样式 | Tailwind CSS |
| Markdown | react-markdown |
| 认证 | JWT |
| API 风格 | RESTful |

### 适合什么场景
- 个人技术博客
- 企业内网博客
- 内容管理系统
- 技术文档站点

### 2. 后端主程序：backend/src/main.rs
```rust
// main.rs - Actix Web 博客后端主程序

mod config;
mod db;
mod errors;
mod models;
mod handlers;
mod middleware;

use actix_cors::Cors;
use actix_web::{web, App, HttpServer, middleware::Logger};
use tracing::info;
use tracing_subscriber::{layer::SubscriberExt, util::SubscriberInitExt};

#[actix_web::main]
async fn main() -> std::io::Result<()> {
    // 初始化日志
    tracing_subscriber::registry()
        .with(tracing_subscriber::EnvFilter::try_from_default_env()
            .unwrap_or_else(|_| "blog_backend=debug,actix_web=info".into()))
        .with(tracing_subscriber::fmt::layer())
        .init();
    
    info!("🚀 启动博客后端服务...");
    
    // 加载配置
    let config = config::Config::load().expect("配置加载失败");
    
    // 初始化数据库
    let db_pool = db::create_pool(&config.database)
        .await
        .expect("数据库连接失败");
    
    // 运行迁移
    db::run_migrations(&db_pool)
        .await
        .expect("数据库迁移失败");
    
    info!("✅ 数据库初始化完成");
    
    let bind_address = format!("{}:{}", config.server.host, config.server.port);
    
    info!("🌐 服务器启动在 http://{}", bind_address);
    
    // 启动 HTTP 服务器
    HttpServer::new(move || {
        let cors = Cors::default()
            .allow_any_origin()
            .allow_any_method()
            .allow_any_header()
            .max_age(3600);
        
        App::new()
            // 添加数据库池
            .app_data(web::Data::new(db_pool.clone()))
            // 添加配置
            .app_data(web::Data::new(config.clone()))
            // 中间件
            .wrap(Logger::default())
            .wrap(cors)
            // 健康检查
            .route("/health", web::get().to(handlers::health_check))
            // 认证路由
            .service(
                web::scope("/api/auth")
                    .route("/register", web::post().to(handlers::auth::register))
                    .route("/login", web::post().to(handlers::auth::login))
                    .route("/me", web::get().to(handlers::auth::me))
            )
            // 文章路由
            .service(
                web::scope("/api/posts")
                    .route("", web::get().to(handlers::posts::list_posts))
                    .route("", web::post().to(handlers::posts::create_post))
                    .route("/{id}", web::get().to(handlers::posts::get_post))
                    .route("/{id}", web::put().to(handlers::posts::update_post))
                    .route("/{id}", web::delete().to(handlers::posts::delete_post))
                    // 评论
                    .route("/{id}/comments", web::get().to(handlers::comments::list_comments))
                    .route("/{id}/comments", web::post().to(handlers::comments::create_comment))
            )
            // 分类和标签
            .service(
                web::scope("/api")
                    .route("/categories", web::get().to(handlers::posts::list_categories))
                    .route("/tags", web::get().to(handlers::posts::list_tags))
            )
    })
    .bind(&bind_address)?
    .run()
    .await
}
```

### 7. 错误处理：backend/src/errors.rs
```rust
// errors.rs - 错误处理

use actix_web::{HttpResponse, ResponseError};
use std::fmt;

#[derive(Debug)]
pub enum AppError {
    NotFound(String),
    BadRequest(String),
    Unauthorized(String),
    Forbidden(String),
    InternalError(String),
    DatabaseError(String),
    ValidationError(String),
}

impl fmt::Display for AppError {
    fn fmt(&self, f: &mut fmt::Formatter<'_>) -> fmt::Result {
        match self {
            AppError::NotFound(msg) => write!(f, "未找到: {}", msg),
            AppError::BadRequest(msg) => write!(f, "请求错误: {}", msg),
            AppError::Unauthorized(msg) => write!(f, "未授权: {}", msg),
            AppError::Forbidden(msg) => write!(f, "禁止访问: {}", msg),
            AppError::InternalError(msg) => write!(f, "内部错误: {}", msg),
            AppError::DatabaseError(msg) => write!(f, "数据库错误: {}", msg),
            AppError::ValidationError(msg) => write!(f, "验证错误: {}", msg),
        }
    }
}

impl ResponseError for AppError {
    fn error_response(&self) -> HttpResponse {
        let (status, error) = match self {
            AppError::NotFound(msg) => (actix_web::http::StatusCode::NOT_FOUND, msg.clone()),
            AppError::BadRequest(msg) => (actix_web::http::StatusCode::BAD_REQUEST, msg.clone()),
            AppError::Unauthorized(msg) => (actix_web::http::StatusCode::UNAUTHORIZED, msg.clone()),
            AppError::Forbidden(msg) => (actix_web::http::StatusCode::FORBIDDEN, msg.clone()),
            AppError::InternalError(msg) => (actix_web::http::StatusCode::INTERNAL_SERVER_ERROR, msg.clone()),
            AppError::DatabaseError(msg) => (actix_web::http::StatusCode::INTERNAL_SERVER_ERROR, msg.clone()),
            AppError::ValidationError(msg) => (actix_web::http::StatusCode::BAD_REQUEST, msg.clone()),
        };
        
        HttpResponse::build(status).json(serde_json::json!({
            "error": error,
        }))
    }
}

impl From<sqlx::Error> for AppError {
    fn from(err: sqlx::Error) -> Self {
        match err {
            sqlx::Error::RowNotFound => AppError::NotFound("记录不存在".to_string()),
            _ => AppError::DatabaseError(err.to_string()),
        }
    }
}

impl From<jsonwebtoken::errors::Error> for AppError {
    fn from(err: jsonwebtoken::errors::Error) -> Self {
        AppError::Unauthorized(format!("Token 无效: {}", err))
    }
}
```

### 8. 数据库操作：backend/src/db.rs
```rust
// db.rs - 数据库操作

use sqlx::{postgres::PgPoolOptions, PgPool, migrations::MigrationSource};
use crate::config::DatabaseConfig;

pub async fn create_pool(config: &DatabaseConfig) -> anyhow::Result<PgPool> {
    let pool = PgPoolOptions::new()
        .max_connections(config.max_connections)
        .connect(&config.url)
        .await?;
    
    Ok(pool)
}

pub async fn run_migrations(pool: &PgPool) -> anyhow::Result<()> {
    // 创建迁移表
    sqlx::query(
        r#"
        CREATE TABLE IF NOT EXISTS schema_migrations (
            version VARCHAR(255) PRIMARY KEY,
            applied_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
        )
        "#,
    )
    .execute(pool)
    .await?;
    
    // 创建用户表
    sqlx::query(
        r#"
        CREATE TABLE IF NOT EXISTS users (
            id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
            username VARCHAR(50) UNIQUE NOT NULL,
            email VARCHAR(255) UNIQUE NOT NULL,
            password_hash VARCHAR(255) NOT NULL,
            nickname VARCHAR(100),
            avatar TEXT,
            bio TEXT,
            role VARCHAR(20) NOT NULL DEFAULT 'user',
            created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
            updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
        )
        "#,
    )
    .execute(pool)
    .await?;
    
    // 创建分类表
    sqlx::query(
        r#"
        CREATE TABLE IF NOT EXISTS categories (
            id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
            name VARCHAR(100) NOT NULL,
            slug VARCHAR(100) UNIQUE NOT NULL,
            description TEXT,
            created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
        )
        "#,
    )
    .execute(pool)
    .await?;
    
    // 创建文章表
    sqlx::query(
        r#"
        CREATE TABLE IF NOT EXISTS posts (
            id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
            title VARCHAR(200) NOT NULL,
            slug VARCHAR(200) UNIQUE NOT NULL,
            content TEXT NOT NULL,
            summary TEXT,
            cover_image TEXT,
            author_id UUID NOT NULL REFERENCES users(id),
            category_id UUID REFERENCES categories(id),
            status VARCHAR(20) NOT NULL DEFAULT 'draft',
            view_count BIGINT NOT NULL DEFAULT 0,
            created_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
            updated_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
            published_at TIMESTAMPTZ
        )
        "#,
    )
    .execute(pool)
    .await?;
    
    // 创建标签表
    sqlx::query(
        r#"
        CREATE TABLE IF NOT EXISTS tags (
            id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
            name VARCHAR(50) UNIQUE NOT NULL,
            slug VARCHAR(50) UNIQUE NOT NULL
        )
        "#,
    )
    .execute(pool)
    .await?;
    
    // 创建文章标签关联表
    sqlx::query(
        r#"
        CREATE TABLE IF NOT EXISTS post_tags (
            post_id UUID REFERENCES posts(id) ON DELETE CASCADE,
            tag_id UUID REFERENCES tags(id) ON DELETE CASCADE,
            PRIMARY KEY (post_id, tag_id)
        )
        "#,
    )
    .execute(pool)
    .await?;
    
    // 创建评论表
    sqlx::query(
        r#"
        CREATE TABLE IF NOT EXISTS comments (
            id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
            post_id UUID NOT NULL REFERENCES posts(id) ON DELETE CASCADE,
            author_id UUID REFERENCES users(id),
            author_name VARCHAR(100),
            content TEXT NOT NULL,
            parent_id UUID REFERENCES comments(id),
            status VARCHAR(20) NOT NULL DEFAULT 'pending',
            created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
        )
        "#,
    )
    .execute(pool)
    .await?;
    
    // 创建索引
    sqlx::query(
        r#"
        CREATE INDEX IF NOT EXISTS idx_posts_author ON posts(author_id);
        CREATE INDEX IF NOT EXISTS idx_posts_category ON posts(category_id);
        CREATE INDEX IF NOT EXISTS idx_posts_status ON posts(status);
        CREATE INDEX IF NOT EXISTS idx_posts_created ON posts(created_at DESC);
        CREATE INDEX IF NOT EXISTS idx_comments_post ON comments(post_id);
        "#
    )
    .execute(pool)
    .await?;
    
    tracing::info!("✅ 数据库迁移完成");
    
    Ok(())
}
```

```rust
// handlers/auth.rs - 认证处理器

use actix_web::{web, HttpResponse};
use argon2::{Argon2, PasswordHash, PasswordVerifier};
use chrono::{Duration, Utc};
use jsonwebtoken::{encode, EncodingKey, Header};
use sqlx::PgPool;
use uuid::Uuid;
use validator::Validate;

use crate::config::Config;
use crate::errors::AppError;
use crate::models::{CreateUserRequest, LoginRequest, LoginResponse, UserResponse, Claims};

/// 注册
pub async fn register(
    pool: web::Data<PgPool>,
    config: web::Data<Config>,
    body: web::Json<CreateUserRequest>,
) -> Result<HttpResponse, AppError> {
    // 验证输入
    body.validate().map_err(|e| AppError::ValidationError(e.to_string()))?;
    
    // 检查用户名是否存在
    let exists: bool = sqlx::query_scalar(
        "SELECT EXISTS(SELECT 1 FROM users WHERE username = $1)"
    )
    .bind(&body.username)
    .fetch_one(pool.get_ref())
    .await?;
    
    if exists {
        return Err(AppError::BadRequest("用户名已存在".to_string()));
    }
    
    // 检查邮箱是否存在
    let exists: bool = sqlx::query_scalar(
        "SELECT EXISTS(SELECT 1 FROM users WHERE email = $1)"
    )
    .bind(&body.email)
    .fetch_one(pool.get_ref())
    .await?;
    
    if exists {
        return Err(AppError::BadRequest("邮箱已被注册".to_string()));
    }
    
    // 哈希密码
    let password_hash = argon2::hash_encoded(
        body.password.as_bytes(),
        b"randomsalt",
        &Argon2::default(),
    )
    .map_err(|_| AppError::InternalError("密码哈希失败".to_string()))?;
    
    // 创建用户
    let user_id = Uuid::new_v4();
    sqlx::query(
        r#"

/// 生成 JWT Token
fn generate_token(
    config: &crate::config::JwtConfig,
    user_id: &str,
    username: &str,
    role: &str,
) -> Result<String, AppError> {
    let now = Utc::now();
    let exp = (now + Duration::hours(config.expiry_hours as i64)).timestamp() as usize;
    
    let claims = Claims {
        sub: user_id.to_string(),
        username: username.to_string(),
        role: role.to_string(),
        exp,
        iat: now.timestamp() as usize,
    };
    
    let token = encode(
        &Header::default(),
        &claims,
        &EncodingKey::from_secret(config.secret.as_bytes()),
    )?;
    
    Ok(token)
}
```

```rust
/// 生成 URL slug
fn generate_slug(title: &str) -> String {
    title
        .to_lowercase()
        .chars()
        .map(|c| {
            if c.is_alphanumeric() || c == '-' {
                c
            } else if c.is_whitespace() {
                '-'
            } else {
                ''
            }
        })
        .collect::<String>()
        .split('-')
        .filter(|s| !s.is_empty())
        .take(10)
        .collect::<Vec<_>>()
        .join("-")
}
```

```typescript
const API_BASE = import.meta.env.VITE_API_URL || 'http://localhost:8080/api';

interface ApiResponse<T> {
  data?: T;
  error?: string;
}

class ApiClient {
  private token: string | null = null;

  setToken(token: string | null) {
    this.token = token;
    localStorage.setItem('token', token || '');
  }

  getToken(): string | null {
    if (!this.token) {
      this.token = localStorage.getItem('token');
    }
    return this.token;
  }

  private async request<T>(
    method: string,
    path: string,
    body?: unknown
  ): Promise<T> {
    const headers: Record<string, string> = {
      'Content-Type': 'application/json',
    };

    if (this.getToken()) {
      headers['Authorization'] = `Bearer ${this.getToken()}`;
    }

    const response = await fetch(`${API_BASE}${path}`, {
      method,
      headers,
      body: body ? JSON.stringify(body) : undefined,
    });

    if (!response.ok) {
      const error = await response.json().catch(() => ({ error: '请求失败' }));
      throw new Error(error.error || '请求失败');
    }

    return response.json();
  }

  // ==================== 认证 API ====================
  
  async register(username: string, email: string, password: string) {
    return this.request<{ token: string; user: User }>('POST', '/auth/register', {
      username,
      email,
      password,
    });
  }

    return this.request<{ token: string; user: User }>('POST', '/auth/login', {
      username,
      password,
    });
  }

    return this.request<User>('GET', '/auth/me');
  }

    pageSize?: number;
    category?: string;
    tag?: string;

    if (params?.pageSize) searchParams.set('page_size', params.pageSize.toString());
    if (params?.category) searchParams.set('category', params.category);
    if (params?.tag) searchParams.set('tag', params.tag);

    return this.request<PaginatedResponse<Post>>(
      'GET',
      `/posts?${searchParams.toString()}`
    );
  }

  async getPost(id: string) {
    return this.request<Post>('GET', `/posts/${id}`);
  }

  async createPost(data: CreatePostData) {
    return this.request<{ id: string; slug: string }>(

      '/posts',
```

```tsx
            onClick={() => setPage((p) => Math.max(1, p - 1))}
            disabled={page === 1}
            className="px-4 py-2 border rounded-lg disabled:opacity-50 disabled:cursor-not-allowed hover:bg-gray-50"
          >
            上一页
          </button>
          
          <span className="px-4 py-2">
            第 {page} / {totalPages} 页
          </span>
          
          <button
            onClick={() => setPage((p) => Math.min(totalPages, p + 1))}
            disabled={page === totalPages}
            className="px-4 py-2 border rounded-lg disabled:opacity-50 disabled:cursor-not-allowed hover:bg-gray-50"
          >
            下一页
          </button>
        </div>
      )}
    </div>
  );
};
```

## 面试要点

### 面试题 1：Actix-web 和 Axum 的区别？

**标准答案**：
| 对比项 | Actix-web | Axum |
|--------|-----------|------|
| 社区 | 更大更成熟 | 较新但活跃 |
| 性能 | 极高 | 高 |
| 生态 | 更丰富 | 与 Tokio 集成更好 |
| 迁移 | - | 与 Tower 兼容 |

### 面试题 2：SQLx 相比 Diesel 的优势？

**标准答案**：
1. **异步优先**：SQLx 完全异步
2. **编译时检查**：编译期验证 SQL
3. **无代码生成**：运行时反射
4. **连接池**：内置支持

### 面试题 3：JWT Token 安全考虑？

**标准答案**：
1. **签名算法**：使用 RS256 而非 HS256
2. **过期时间**：短期 Access Token
3. **刷新 Token**：分离机制
4. **黑名单**：Redis 存储

### 面试题 4：Rust 的错误处理？

**标准答案**：
```rust
// Result<T, E>
fn read_file() -> Result<String, io::Error> {
```

**标准答案**：
