# Actix + React 方案：方案3 - 文件分享服务

## 项目介绍

### 项目是什么
这是一个基于 Rust + Actix Web + React + PostgreSQL 构建的文件分享服务，支持文件上传、下载、分享链接、二维码分享、过期时间和密码保护。

### 解决什么问题
- 临时文件分享（类似 WeTransfer）
- 大文件传输
- 安全的文件共享
- 链接管理和追踪

### 核心技术亮点
| 特性 | 技术实现 |
|------|----------|
| 后端框架 | Actix-web 4 |
| 文件存储 | 本地 / S3 |
| 数据库 | PostgreSQL |
| 前端框架 | React 18 + TypeScript |
| 上传组件 | react-dropzone |
| 二维码 | qrcode.react |
| 拖拽上传 | 前端组件 |

### 适合什么场景
- 临时文件分享
- 大文件传输
- 企业内部文件共享
- 个人文件分享

### 3. 存储服务：backend/src/storage/mod.rs
```rust
// storage/mod.rs - 文件存储

use anyhow::Result;
use std::path::{Path, PathBuf};
use tokio::fs;
use tokio::io::AsyncWriteExt;
use sha2::{Sha256, Digest};

/// 本地文件存储
pub struct LocalStorage {
    /// 存储根目录
    base_path: PathBuf,
}

impl LocalStorage {
    pub fn new(base_path: &str) -> Self {
        Self {
            base_path: PathBuf::from(base_path),
        }
    }
    
    /// 初始化存储目录
    pub async fn init(&self) -> Result<()> {
        fs::create_dir_all(&self.base_path).await?;
        Ok(())
    }
    
    /// 保存文件
    /// 返回：存储名称、文件路径、校验和
    pub async fn save(
        &self,
        data: Vec<u8>,
        original_name: &str,
    ) -> Result<(String, String, String)> {
        // 生成唯一存储名称
        let storage_name = format!(
            "{}_{}",
            uuid::Uuid::new_v4(),
            sanitize_filename(original_name)
        );
        
        // 创建日期目录
        let now = chrono::Utc::now();
        let date_path = format!("{}/{}/{}", now.year(), now.month(), now.day());
        let dir_path = self.base_path.join(&date_path);
        fs::create_dir_all(&dir_path).await?;
        
        // 文件完整路径
        let file_path = dir_path.join(&storage_name);
        
        // 计算校验和
        let mut hasher = Sha256::new();
        hasher.update(&data);
        let checksum = hex::encode(hasher.finalize());
        
        // 写入文件
        let mut file = fs::File::create(&file_path).await?;
        file.write_all(&data).await?;
        file.flush().await?;
        
        // 返回相对路径、完整路径、校验和
        let relative_path = format!("{}/{}", date_path, storage_name);
        Ok((
            storage_name,
            relative_path,
            checksum,
        ))
    }
    
    /// 读取文件
    pub async fn read(&self, path: &str) -> Result<Vec<u8>> {
        let file_path = self.base_path.join(path);
        Ok(fs::read(&file_path).await?)
    }
    
    /// 删除文件
    pub async fn delete(&self, path: &str) -> Result<()> {
        let file_path = self.base_path.join(path);
        if file_path.exists() {
            fs::remove_file(&file_path).await?;
        }
        Ok(())
    }
    
    /// 获取文件大小
    pub async fn size(&self, path: &str) -> Result<i64> {
        let file_path = self.base_path.join(path);
        let metadata = fs::metadata(&file_path).await?;
        Ok(metadata.len() as i64)
    }
    
    /// 检查文件是否存在
    pub async fn exists(&self, path: &str) -> bool {
        let file_path = self.base_path.join(path);
        file_path.exists()
    }
}

/// 清理文件名
fn sanitize_filename(name: &str) -> String {
    name.chars()
        .filter(|c| c.is_alphanumeric() || *c == '.' || *c == '-' || *c == '_')
        .take(100)  // 限制长度
        .collect()
}

/// 共享存储实例
pub static STORAGE: once_cell::sync::Lazy<LocalStorage> = 
    once_cell::sync::Lazy::new(|| {
        LocalStorage::new("./uploads")
    });
```

// handlers/files.rs - 文件处理处理器

use actix_multipart::Multipart;
use actix_web::{web, HttpRequest, HttpResponse, Result};
use chrono::{Duration, Utc};
use sqlx::PgPool;
use uuid::Uuid;

use crate::config::Config;
use crate::errors::AppError;
use crate::models::{
    CreateFileRequest, CreateFileResponse, FileInfoResponse, 
    ShareLinkInfo, FileRecord,
};
use crate::storage::STORAGE;

// ==================== 上传文件 ====================

/// 上传文件
pub async fn upload_file(
    req: HttpRequest,
    pool: web::Data<PgPool>,
    config: web::Data<Config>,
    mut payload: Multipart,
) -> Result<HttpResponse, AppError> {
    // 获取客户端 IP
    let creator_ip = req
        .connection_info()
        .realip_remote_addr()
        .map(|s| s.to_string());
    
    let mut file_data: Option<Vec<u8>> = None;
    let mut file_name: Option<String> = None;
    let mut mime_type: Option<String> = None;
    
    // 解析 multipart 数据
    while let Some(item) = payload.next().await {
        let mut field = item.map_err(|_| AppError::BadRequest("上传失败".to_string()))?;
        
        let content_disposition = field.content_disposition();
        let name = content_disposition
            .get_name()
            .unwrap_or("file");
        
        if name == "file" {
            file_name = content_disposition
                .get_filename()
                .map(|s| s.to_string());
            
            // 读取文件内容
            let mut data = Vec::new();
            while let Some(chunk) = field.next().await {
                let chunk = chunk.map_err(|_| AppError::BadRequest("读取文件失败".to_string()))?;
                data.extend_from_slice(&chunk);
            }
            
            file_data = Some(data);
            mime_type = file_name
                .as_ref()
                .and_then(|n| mime_guess::from_path(n).first())
                .map(|m| m.to_string());
        }
    }
    
    // 验证文件
    let file_data = file_data.ok_or_else(|| AppError::BadRequest("未提供文件".to_string()))?;
    let file_name = file_name.ok_or_else(|| AppError::BadRequest("未提供文件名".to_string()))?;
    let mime_type = mime_type.unwrap_or_else(|| "application/octet-stream".to_string());
    
    // 检查文件大小
    let max_size = config.upload.max_file_size;
    if file_data.len() as i64 > max_size {
        return Err(AppError::BadRequest(format!(
            "文件大小超过限制 (最大 {}MB)",
            max_size / 1024 / 1024
        )));
    }
    
    // 保存文件
    let (storage_name, file_path, checksum) = STORAGE
        .save(file_data, &file_name)
        .await
        .map_err(|e| AppError::InternalError(e.to_string()))?;
    
    // 生成分享码
    let share_code = generate_share_code();
    let file_id = Uuid::new_v4();
    
    // 计算过期时间
    let expires_hours = 24; // 默认 24 小时
    let expires_at = Utc::now() + Duration::hours(expires_hours);
    
    // 保存到数据库
    sqlx::query(
        r#"
        INSERT INTO files (
            id, share_code, original_name, storage_name, file_path,
            file_size, mime_type, checksum, download_count,
            expires_at, created_at, creator_ip
        )
        VALUES ($1, $2, $3, $4, $5, $6, $7, $8, 0, $9, NOW(), $10)
        "#,
    )
    .bind(file_id)
    .bind(&share_code)
    .bind(&file_name)
    .bind(&storage_name)
    .bind(&file_path)
    .bind(file_data.len() as i64)
    .bind(&mime_type)
    .bind(&checksum)
    .bind(expires_at)
    .bind(&creator_ip)
    .execute(pool.get_ref())
    .await?;
    
    // 构建分享 URL
    let share_url = format!("{}/share/{}", config.server.base_url, share_code);
    
    Ok(HttpResponse::Ok().json(CreateFileResponse {
        id: file_id,
        share_code: share_code.clone(),
        share_url,
        expires_at: Some(expires_at),
        max_downloads: None,
        file_name,
        file_size: file_data.len() as i64,
    }))
}

// ==================== 获取文件信息 ====================

/// 获取文件信息（无需密码）
pub async fn get_file_info(
    pool: web::Data<PgPool>,
    path: web::Path<String>,
) -> Result<HttpResponse, AppError> {
    let share_code = path.into_inner();
    
    let file: Option<FileRecord> = sqlx::query_as(
        r#"
        SELECT id, original_name, file_size, mime_type, 
               download_count, max_downloads, expires_at, 
               created_at, (password_hash IS NOT NULL) as requires_password
        FROM files
        WHERE share_code = $1
        "#
    )
    .bind(&share_code)
    .fetch_optional(pool.get_ref())
    .await?;
    
    match file {
        Some(f) => {
            // 检查是否过期
            if let Some(expires_at) = f.expires_at {
                if expires_at < Utc::now() {
                    return Err(AppError::NotFound("文件已过期".to_string()));
                }
            }
            
            // 检查下载次数
            if let Some(max_downloads) = f.max_downloads {
                if f.download_count >= max_downloads {
                    return Err(AppError::NotFound("下载次数已用完".to_string()));
                }
            }
            
            Ok(HttpResponse::Ok().json(FileInfoResponse {
                id: f.id,
                file_name: f.original_name,
                file_size: f.file_size,
                mime_type: f.mime_type,
                download_count: f.download_count,
                remaining_downloads: f.max_downloads.map(|m| m - f.download_count),
                expires_at: f.expires_at,
                created_at: f.created_at,
                requires_password: f.password_hash.is_some(),
            }))
        }
        None => Err(AppError::NotFound("文件不存在".to_string())),
    }
}

// ==================== 下载文件 ====================

/// 下载文件
pub async fn download_file(
    pool: web::Data<PgPool>,
    path: web::Path<String>,
    query: web::Query<DownloadRequest>,
) -> Result<HttpResponse, AppError> {
    let share_code = path.into_inner();
    
    // 获取文件信息
    let file: Option<FileRecord> = sqlx::query_as(
        "SELECT * FROM files WHERE share_code = $1"
    )
    .bind(&share_code)
    .fetch_optional(pool.get_ref())
    .await?;
    
    let file = file.ok_or_else(|| AppError::NotFound("文件不存在".to_string()))?;
    
    // 检查过期
    if let Some(expires_at) = file.expires_at {
        if expires_at < Utc::now() {
            return Err(AppError::NotFound("文件已过期".to_string()));
        }
    }
    
    // 检查下载次数
    if let Some(max_downloads) = file.max_downloads {
        if file.download_count >= max_downloads {
            return Err(AppError::NotFound("下载次数已用完".to_string()));
        }
    }
    
    // 检查密码
    if let Some(ref password_hash) = file.password_hash {
        if let Some(ref password) = query.password {
            // 验证密码
            if !verify_password(password, password_hash) {
                return Err(AppError::Unauthorized("密码错误".to_string()));
            }
        } else {
            return Err(AppError::Unauthorized("需要密码".to_string()));
        }
    }
    
    // 读取文件
    let file_content = STORAGE
        .read(&file.file_path)
        .await
        .map_err(|e| AppError::InternalError(e.to_string()))?;
    
    // 更新下载次数
    sqlx::query(
        "UPDATE files SET download_count = download_count + 1 WHERE id = $1"
    )
    .bind(file.id)
    .execute(pool.get_ref())
    .await?;
    
    // 构建响应
    let disposition = format!(
        "attachment; filename=\"{}\"",
        file.original_name
    );
    
    Ok(HttpResponse::Ok()
        .content_type(&file.mime_type)
        .insert_header(("Content-Disposition", disposition))
        .body(file_content))
}

// ==================== 辅助函数 ====================

/// 生成分享码
fn generate_share_code() -> String {
    use rand::Rng;
    
    const CHARS: &[u8] = b"ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789";
    let mut rng = rand::thread_rng();
    
    (0..8)
        .map(|_| {
            let idx = rng.gen_range(0..CHARS.len());
            CHARS[idx] as char
        })
        .collect()
}

/// 验证密码
fn verify_password(password: &str, hash: &str) -> bool {
    // 简化实现：实际应使用 argon2 或 bcrypt
    // 这里使用简单的 SHA256 比较
    use sha2::{Sha256, Digest};
    
    let mut hasher = Sha256::new();
    hasher.update(password.as_bytes());
    let computed = hex::encode(hasher.finalize());
    
    computed == hash
}

// UploadZone.tsx - 文件上传组件

import React, { useCallback, useState } from 'react';
import { useDropzone } from 'react-dropzone';
import { api } from '../api';

interface UploadZoneProps {
  onUploadComplete?: (result: UploadResult) => void;
}

interface UploadResult {
  id: string;
  share_code: string;
  share_url: string;
  file_name: string;
  file_size: number;
  expires_at: string;
}

export const UploadZone: React.FC<UploadZoneProps> = ({ onUploadComplete }) => {
  const [uploading, setUploading] = useState(false);
  const [progress, setProgress] = useState(0);
  const [dragActive, setDragActive] = useState(false);
  const [error, setError] = useState<string | null>(null);
  const [result, setResult] = useState<UploadResult | null>(null);
  
  const onDrop = useCallback(async (acceptedFiles: File[]) => {
    if (acceptedFiles.length === 0) {
      setError('请选择文件');
      return;
    }
    
    const file = acceptedFiles[0];
    
    // 检查文件大小（100MB）
    if (file.size > 100 * 1024 * 1024) {
      setError('文件大小不能超过 100MB');
      return;
    }
    
    setUploading(true);
    setError(null);
    setProgress(0);
    
    try {
      // 创建 FormData
      const formData = new FormData();
      formData.append('file', file);
      
      // 上传
      const response = await fetch('/api/upload', {
        method: 'POST',
        headers: {
          'Authorization': `Bearer ${api.getToken()}`,
        },
        body: formData,
      });
      
      if (!response.ok) {
        const err = await response.json();
        throw new Error(err.error || '上传失败');
      }
      
      const uploadResult: UploadResult = await response.json();
      setResult(uploadResult);
      onUploadComplete?.(uploadResult);
      
      setProgress(100);
    } catch (err) {
      setError(err instanceof Error ? err.message : '上传失败');
    } finally {
      setUploading(false);
    }
  }, [onUploadComplete]);
  
  const { getRootProps, getInputProps } = useDropzone({
    onDrop,
    accept: {
      'image/*': ['.png', '.jpg', '.jpeg', '.gif'],
      'application/pdf': ['.pdf'],
      'application/zip': ['.zip'],
      'application/x-rar-compressed': ['.rar'],
    },
    maxFiles: 1,
    maxSize: 100 * 1024 * 1024, // 100MB
    disabled: uploading,
  });
  
  // 格式化文件大小
  const formatSize = (bytes: number) => {
    if (bytes < 1024) return `${bytes} B`;
    if (bytes < 1024 * 1024) return `${(bytes / 1024).toFixed(1)} KB`;
    if (bytes < 1024 * 1024 * 1024) return `${(bytes / 1024 / 1024).toFixed(1)} MB`;
    return `${(bytes / 1024 / 1024 / 1024).toFixed(1)} GB`;
  };
  
  return (
    <div className="max-w-2xl mx-auto p-6">
      {/* 上传区域 */}
      <div
        {...getRootProps()}
        className={`
          border-2 border-dashed rounded-xl p-8 text-center cursor-pointer
          transition-colors duration-200
          ${dragActive ? 'border-indigo-500 bg-indigo-50' : 'border-gray-300 hover:border-indigo-400'}
          ${uploading ? 'pointer-events-none opacity-75' : ''}
        `}
      >
        <input {...getInputProps()} />
        
        {uploading ? (
          <div className="space-y-4">
            <div className="animate-spin w-12 h-12 mx-auto border-4 border-indigo-500 border-t-transparent rounded-full" />
            <p className="text-gray-600">上传中...</p>
            <div className="w-full bg-gray-200 rounded-full h-2">
              <div
                className="bg-indigo-500 h-2 rounded-full transition-all duration-300"
                style={{ width: `${progress}%` }}
              />
            </div>
          </div>
        ) : (
          <div className="space-y-4">
            <div className="w-16 h-16 mx-auto bg-indigo-100 rounded-full flex items-center justify-center">
              <svg className="w-8 h-8 text-indigo-600" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M7 16a4 4 0 01-.88-7.903A5 5 0 1115.9 6L16 6a5 5 0 011 9.9M15 13l-3-3m0 0l-3 3m3-3v12" />
              </svg>
            </div>
            <div>
              <p className="text-lg font-medium text-gray-700">
                拖拽文件到此处，或点击选择

// Download.tsx - 文件下载页面

import React, { useEffect, useState } from 'react';
import { useParams } from 'react-router-dom';

interface FileInfo {
  id: string;
  file_name: string;
  file_size: number;
  mime_type: string;
  download_count: number;
  remaining_downloads: number | null;
  expires_at: string | null;
  created_at: string;
  requires_password: boolean;
}

export const DownloadPage: React.FC = () => {
  const { shareCode } = useParams<{ shareCode: string }>();
  const [fileInfo, setFileInfo] = useState<FileInfo | null>(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);
  const [password, setPassword] = useState('');
  const [downloading, setDownloading] = useState(false);
  
  // 获取文件信息
  useEffect(() => {
    const fetchFileInfo = async () => {
      try {
        const response = await fetch(`/api/files/${shareCode}`);
        if (!response.ok) {
          const err = await response.json();
          throw new Error(err.error || '获取文件信息失败');
        }
        const data = await response.json();
        setFileInfo(data);
      } catch (err) {
        setError(err instanceof Error ? err.message : '获取文件信息失败');
      } finally {
        setLoading(false);
      }
    };
    
    fetchFileInfo();
  }, [shareCode]);
  
  // 下载文件
  const handleDownload = async () => {
    if (!fileInfo) return;
    
    setDownloading(true);
    
    try {
      const url = `/api/download/${shareCode}${fileInfo.requires_password ? `?password=${encodeURIComponent(password)}` : ''}`;
      const response = await fetch(url);
      
      if (!response.ok) {
        const err = await response.json();
        throw new Error(err.error || '下载失败');
      }
      
      // 获取文件名
      const contentDisposition = response.headers.get('Content-Disposition');
      let filename = fileInfo.file_name;
      if (contentDisposition) {
        const match = contentDisposition.match(/filename="?(.+)"?/);
        if (match) {
          filename = match[1];
        }
      }
      
      // 创建下载链接
      const blob = await response.blob();
      const downloadUrl = URL.createObjectURL(blob);
      const a = document.createElement('a');
      a.href = downloadUrl;
      a.download = filename;
      document.body.appendChild(a);
      a.click();
      document.body.removeChild(a);
      URL.revokeObjectURL(downloadUrl);
      
    } catch (err) {
      setError(err instanceof Error ? err.message : '下载失败');
    } finally {
      setDownloading(false);
    }
  };
  
  // 格式化文件大小
  const formatSize = (bytes: number) => {
    if (bytes < 1024) return `${bytes} B`;
    if (bytes < 1024 * 1024) return `${(bytes / 1024).toFixed(1)} KB`;
    if (bytes < 1024 * 1024 * 1024) return `${(bytes / 1024 / 1024).toFixed(1)} MB`;
    return `${(bytes / 1024 / 1024 / 1024).toFixed(1)} GB`;
  };
  
  // 格式化过期时间
  const formatExpiry = (dateStr: string | null) => {
    if (!dateStr) return '永不过期';
    const date = new Date(dateStr);
    return date.toLocaleString('zh-CN');
  };
  
  if (loading) {
    return (
      <div className="min-h-screen flex items-center justify-center">
        <div className="animate-spin w-12 h-12 border-4 border-indigo-500 border-t-transparent rounded-full" />
      </div>
    );
  }
  
  if (error) {
    return (
      <div className="min-h-screen flex items-center justify-center">
        <div className="max-w-md p-8 bg-red-50 border border-red-200 rounded-xl text-center">
          <div className="w-16 h-16 mx-auto mb-4 bg-red-100 rounded-full flex items-center justify-center">
            <svg className="w-8 h-8 text-red-600" fill="none" stroke="currentColor" viewBox="0 0 24 24">
              <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M12 9v2m0 4h.01m-6.938 4h13.856c1.54 0 2.502-1.667 1.732-3L13.732 4c-.77-1.333-2.694-1.333-3.464 0L3.34 16c-.77 1.333.192 3 1.732 3z" />
            </svg>
          </div>
          <h2 className="text-xl font-bold text-red-800 mb-2">文件不可用</h2>
          <p className="text-red-600">{error}</p>
        </div>
      </div>
    );
  }
  
  if (!fileInfo) return null;
  
  return (
    <div className="min-h-screen bg-gray-100 py-12">
      <div className="max-w-lg mx-auto px-4">
        <div className="bg-white rounded-2xl shadow-lg overflow-hidden">
          {/* 文件图标 */}
          <div className="bg-gradient-to-r from-indigo-500 to-purple-600 p-8 text-center">
            <div className="w-20 h-20 mx-auto bg-white/20 rounded-2xl flex items-center justify-center">
              <svg className="w-10 h-10 text-white" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M7 21h10a2 2 0 002-2V9.414a1 1 0 00-.293-.707l-5.414-5.414A1 1 0 0012.586 3H7a2 2 0 00-2 2v14a2 2 0 002 2z" />
              </svg>
            </div>
          </div>
          
          {/* 文件信息 */}
          <div className="p-8 space-y-6">
            <div className="text-center">
              <h1 className="text-2xl font-bold text-gray-900 mb-2">
                {fileInfo.file_name}
              </h1>
              <div className="flex justify-center gap-4 text-sm text-gray-500">
                <span>{formatSize(fileInfo.file_size)}</span>
                <span>·</span>
                <span>{formatExpiry(fileInfo.expires_at)}</span>
              </div>
            </div>
            
            {/* 密码输入 */}
            {fileInfo.requires_password && (
              <div>
                <label className="block text-sm font-medium text-gray-700 mb-2">
                  请输入密码
                </label>
                <input
                  type="password"
                  value={password}
                  onChange={(e) => setPassword(e.target.value)}
                  placeholder="输入访问密码"
                  className="w-full px-4 py-3 border rounded-lg focus:ring-2 focus:ring-indigo-500 focus:border-indigo-500"
                />
              </div>
            )}
            
            {/* 下载按钮 */}
            <button
              onClick={handleDownload}
              disabled={downloading || (fileInfo.requires_password && !password)}
              className="w-full py-4 bg-indigo-600 text-white rounded-xl font-medium text-lg
                         hover:bg-indigo-700 disabled:opacity-50 disabled:cursor-not-allowed
                         transition-colors flex items-center justify-center gap-2"
            >
              {downloading ? (
                <>
                  <div className="animate-spin w-5 h-5 border-2 border-white border-t-transparent rounded-full" />
                  下载中...
                </>
              ) : (
                <>
                  <svg className="w-5 h-5" fill="none" stroke="currentColor" viewBox="0 0 24 24">
                    <path strokeLinecap="round" strokeLinejoin="round" strokeWidth={2} d="M4 16v1a3 3 0 003 3h10a3 3 0 003-3v-1m-4-4l-4 4m0 0l-4-4m4 4V4" />
                  </svg>
                  下载文件
                </>
              )}
            </button>
            
            {/* 统计信息 */}
            <div className="flex justify-center gap-8 text-sm text-gray-500 pt-4 border-t">
              <div className="text-center">
                <div className="text-2xl font-bold text-gray-900">{fileInfo.download_count}</div>
                <div>已下载次数</div>

                  <div>剩余次数</div>
                </div>
              )}
            </div>
          </div>
        </div>
      </div>
    </div>
  );
};

## 面试要点

### 面试题 1：文件上传的安全考虑？

**标准答案**：
1. **文件类型验证**：MIME 类型和扩展名
2. **文件大小限制**：防止 DoS
3. **文件名清理**：防止路径遍历
4. **内容扫描**：病毒检测
5. **存储隔离**：不与代码同目录

### 面试题 2：如何处理大文件上传？

**标准答案**：
1. **分片上传**：将文件分成小块
2. **断点续传**：记录已上传的分片
3. **流式处理**：边读边写，不占用内存
4. **进度显示**：实时反馈

### 面试题 3：SHA256 和 MD5 的区别？

**标准答案**：
| 对比项 | SHA256 | MD5 |
|--------|--------|-----|
| 安全性 | 高 | 低（已破解） |
| 输出长度 | 256 位 | 128 位 |
| 速度 | 较慢 | 较快 |
| 用途 | 安全校验 | 快速校验 |

### 面试题 4：Rust 的 HashMap vs std::collections::HashMap？

**标准答案**：
- `std::collections::HashMap`：同步，线程安全
- `std::collections::HashMap`：`Sync`
- `parking_lot::RwLock<HashMap>`：更细粒度锁
- `dashmap::DashMap`：无锁，分片实现

### 面试题 5：React 文件上传最佳实践？

**标准答案**：
1. **FormData**：正确编码
2. **进度追踪**：XMLHttpRequest
3. **错误处理**：网络错误重试
4. **文件预览**：图片预览
5. **拖拽支持**：用户体验
