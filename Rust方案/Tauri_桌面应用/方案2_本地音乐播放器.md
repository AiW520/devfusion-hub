# Tauri 桌面应用方案：方案2 - 本地音乐播放器

## 项目介绍

### 项目是什么
这是一个基于 Tauri + React + TypeScript 构建的本地音乐播放器，支持播放本地音频文件、管理播放列表、显示歌词（同步/未同步）、可视化音频波形、倍速播放等功能。

### 解决什么问题
- 无需联网的本地音乐播放
- 轻量级替代网易云音乐桌面版
- 支持多种音频格式（MP3、FLAC、WAV、AAC、OGG）
- 歌词同步显示
- 音频可视化效果

### 核心技术亮点
| 特性 | 技术实现 |
|------|----------|
| 音频处理 | Web Audio API + Howler.js |
| 歌词解析 | 解析 .lrc 文件格式 |
| 音频可视化 | Canvas + Web Audio API |
| 状态管理 | Zustand |
| 前端框架 | React 18 + TypeScript 5 |
| 后端核心 | Rust + Tauri 2.0 |
| 音频格式 | Rust 的 symphonia 库 |

### 适合什么场景
- 程序员写代码时的背景音乐播放
- 学习时的轻量音乐播放器
- 对隐私敏感的用户（无需联网）
- 喜欢简洁界面的用户

```typescript
// playerStore.ts - Zustand 状态管理
// 作用：管理播放器的所有状态，包括播放列表、播放进度、音频设置等

import { create } from 'zustand';

// ==================== 类型定义 ====================

/**
 * 歌曲类型
 * 包含歌曲的元数据和文件信息
 */
export interface Track {
  id: string;           // 唯一标识符（基于文件路径生成）
  title: string;        // 歌曲标题
  artist: string;        // 艺术家
  album: string;         // 专辑名
  duration: number;      // 时长（秒）
  path: string;          // 文件路径
  coverArt?: string;     // 封面图片路径（Base64 或本地路径）
  lyrics?: LyricLine[];  // 歌词数组
}

/**
 * 歌词行类型
 * 包含时间戳和对应的歌词文本
 */
export interface LyricLine {
  time: number;    // 时间戳（秒），如 12.34
  text: string;    // 歌词文本
  translation?: string;  // 翻译（可选）
}

/**
 * 播放模式枚举
 */
export type PlayMode = 'repeat-one' | 'repeat-all' | 'shuffle';

/**
 * 均衡器预设
 */
export interface EqualizerPreset {
  name: string;              // 预设名称
  bands: number[];           // 10 个频段的增益值（dB）
}

// ==================== Store 类型定义 ====================

interface PlayerState {
  // ==================== 播放列表 ====================
  playlist: Track[];           // 播放列表
  currentTrackIndex: number;   // 当前播放的曲目索引
  currentTime: number;         // 当前播放时间（秒）
  duration: number;           // 当前曲目总时长（秒）
  volume: number;             // 音量 0-1
  isMuted: boolean;           // 是否静音
  
  // ==================== 播放状态 ====================
  isPlaying: boolean;          // 是否正在播放
  isLoading: boolean;         // 是否正在加载
  
  // ==================== 播放设置 ====================
  playMode: PlayMode;          // 播放模式
  playbackRate: number;        // 播放速度（0.5 - 2.0）
  equalizerEnabled: boolean;   // 均衡器开关
  
  // ==================== UI 状态 ====================
  showPlaylist: boolean;        // 是否显示播放列表
  showLyrics: boolean;         // 是否显示歌词
  lyricsMode: 'synced' | 'original';  // 歌词模式
  
  // ==================== 动作方法 ====================
  // 列表操作
  setPlaylist: (tracks: Track[]) => void;
  addTrack: (track: Track) => void;
  removeTrack: (id: string) => void;
  clearPlaylist: () => void;
  
  // 播放控制
  play: () => void;
  pause: () => void;
  togglePlay: () => void;
  playTrack: (index: number) => void;
  playNext: () => void;
  playPrevious: () => void;
  seek: (time: number) => void;
  setVolume: (volume: number) => void;
  toggleMute: () => void;
  
  // 设置操作
  setPlayMode: (mode: PlayMode) => void;
  setPlaybackRate: (rate: number) => void;
  toggleEqualizer: () => void;
  
  // 时间更新
  setCurrentTime: (time: number) => void;
  setDuration: (duration: number) => void;
  setLoading: (loading: boolean) => void;
  
  // UI 操作
  togglePlaylist: () => void;
  toggleLyrics: () => void;
  setLyricsMode: (mode: 'synced' | 'original') => void;
}

// ==================== 创建 Store ====================

export const usePlayerStore = create<PlayerState>((set, get) => ({
  // ==================== 初始状态 ====================
  
  // 播放列表初始为空
  playlist: [],
  currentTrackIndex: -1,
  currentTime: 0,
  duration: 0,
  volume: 0.8,
  isMuted: false,
  
  // 播放状态初始为停止
  isPlaying: false,
  isLoading: false,
  
  // 默认设置
  playMode: 'repeat-all',
  playbackRate: 1.0,
  equalizerEnabled: false,
  
  // 默认显示播放列表
  showPlaylist: true,
  showLyrics: false,
  lyricsMode: 'synced',
  
  // ==================== 列表操作 ====================
  
  /** 设置播放列表 */
  setPlaylist: (tracks) => set({ playlist: tracks }),
  
  /** 添加曲目到列表 */
  addTrack: (track) => set((state) => ({
    playlist: [...state.playlist, track]
  })),
  
  /** 从列表移除曲目 */
  removeTrack: (id) => set((state) => {
    const index = state.playlist.findIndex((t) => t.id === id);
    const newPlaylist = state.playlist.filter((t) => t.id !== id);
    
    // 计算新的当前曲目索引
    let newIndex = state.currentTrackIndex;
    if (index < state.currentTrackIndex) {
      newIndex -= 1;
    } else if (index === state.currentTrackIndex) {
      // 移除的是当前曲目，播放列表可能变为空
      newIndex = Math.min(newIndex, newPlaylist.length - 1);
    }
    
    return {
      playlist: newPlaylist,
      currentTrackIndex: newIndex,
    };
  }),
  
  /** 清空播放列表 */
  clearPlaylist: () => set({
    playlist: [],
    currentTrackIndex: -1,
    isPlaying: false,
    currentTime: 0,
    duration: 0,
  }),
  
  // ==================== 播放控制 ====================
  
  /** 开始播放 */
  play: () => set({ isPlaying: true }),
  
  /** 暂停播放 */
  pause: () => set({ isPlaying: false }),
  
  /** 切换播放/暂停 */
  togglePlay: () => set((state) => ({ isPlaying: !state.isPlaying })),
  
  /** 播放指定曲目 */
  playTrack: (index) => {
    const state = get();
    if (index < 0 || index >= state.playlist.length) return;
    
    set({
      currentTrackIndex: index,
      currentTime: 0,
      isPlaying: true,
    });
  },
  
  /** 播放下一曲 */
  playNext: () => {
    const state = get();
    if (state.playlist.length === 0) return;
    
    let nextIndex: number;
    
    switch (state.playMode) {
      case 'repeat-one':
        // 单曲循环：保持当前曲目索引
        nextIndex = state.currentTrackIndex;
        break;
        
      case 'shuffle':
        // 随机播放：随机选择一个不同的曲目
        do {
          nextIndex = Math.floor(Math.random() * state.playlist.length);
        } while (state.playlist.length > 1 && nextIndex === state.currentTrackIndex);
        break;
        
      case 'repeat-all':
      default:
        // 列表循环：下一个曲目，到末尾则回到开头
        nextIndex = (state.currentTrackIndex + 1) % state.playlist.length;
        break;
    }
    
    // 如果是单曲循环模式，重播当前曲目
    if (state.playMode === 'repeat-one') {
      set({ currentTime: 0 });
    } else {
      set({
        currentTrackIndex: nextIndex,
        currentTime: 0,
        isPlaying: true,
      });
    }
  },
  
  /** 播放上一曲 */
  playPrevious: () => {
    const state = get();
    if (state.playlist.length === 0) return;
    
    // 如果当前播放时间超过 3 秒，回到曲目开头
    if (state.currentTime > 3) {
      set({ currentTime: 0 });
      return;
    }
    
    // 否则播放上一曲
    const prevIndex = state.currentTrackIndex <= 0 
      ? state.playlist.length - 1 
      : state.currentTrackIndex - 1;
    
    set({
      currentTrackIndex: prevIndex,
      currentTime: 0,
      isPlaying: true,
    });
  },
  
  /** 跳转到指定时间 */
  seek: (time) => set({ currentTime: Math.max(0, Math.min(time, get().duration)) }),
  
  /** 设置音量 */
  setVolume: (volume) => set({
    volume: Math.max(0, Math.min(1, volume)),
    isMuted: volume === 0,
  }),
  
  /** 切换静音 */
  toggleMute: () => set((state) => ({ isMuted: !state.isMuted })),
  
  // ==================== 设置操作 ====================
  
  /** 设置播放模式 */
  setPlayMode: (mode) => set({ playMode: mode }),
  
  /** 设置播放速度 */
  setPlaybackRate: (rate) => set({
    playbackRate: Math.max(0.5, Math.min(2.0, rate))
  }),
  
  /** 切换均衡器 */
  toggleEqualizer: () => set((state) => ({ 
    equalizerEnabled: !state.equalizerEnabled 
  })),
  
  // ==================== 时间更新 ====================
  
  /** 更新当前播放时间 */
  setCurrentTime: (time) => set({ currentTime: time }),
  
  /** 设置曲目总时长 */
  setDuration: (duration) => set({ duration }),
  
  /** 设置加载状态 */
  setLoading: (loading) => set({ isLoading: loading }),
  
  // ==================== UI 操作 ====================
  
  /** 切换播放列表显示 */
  togglePlaylist: () => set((state) => ({ 
    showPlaylist: !state.showPlaylist 
  })),
  
  /** 切换歌词显示 */
  toggleLyrics: () => set((state) => ({ 
    showLyrics: !state.showLyrics 
  })),
  
  /** 设置歌词模式 */
  setLyricsMode: (mode) => set({ lyricsMode: mode }),
}));

// ==================== 辅助函数 ====================

/**
 * 格式化时间为 mm:ss 格式
 * @param seconds - 秒数
 * @returns 格式化后的时间字符串
 * 
 * @example
 * formatTime(65)  // returns "1:05"
 * formatTime(3600) // returns "60:00"
 */
export const formatTime = (seconds: number): string => {
  // 处理无效输入
  if (!seconds || isNaN(seconds) || seconds < 0) {
    return '0:00';
  }
  
  // 向下取整
  const totalSeconds = Math.floor(seconds);
  
  // 计算分钟和秒
  const minutes = Math.floor(totalSeconds / 60);
  const secs = totalSeconds % 60;
  
  // 格式化为 mm:ss
  // padStart 确保秒数是两位数（如 05 而非 5）
  return `${minutes}:${secs.toString().padStart(2, '0')}`;
};
```

```typescript
// lyrics.ts - 歌词解析工具
// 作用：解析 .lrc 格式的歌词文件

import { LyricLine } from '../stores/playerStore';

/**
 * 解析 LRC 格式歌词
 * 
 * LRC 格式示例：
 * [ti:歌曲标题]
 * [ar:艺术家]
 * [00:12.34]这是第一句歌词
 * [00:34.56]这是第二句歌词
 * [00:34.56]<00:35.00>这是带翻译的歌词
 */
export function parseLRC(lrcContent: string): {
  metadata: Record<string, string>;
  lyrics: LyricLine[];
} {
  const lines = lrcContent.split('\n');
  const metadata: Record<string, string> = {};
  const lyrics: LyricLine[] = [];
  
  // LRC 时间标签正则表达式
  // 匹配格式：[mm:ss.xx] 或 [mm:ss:xx]
  const timeTagRegex = /\[(\d{2}):(\d{2})[\.:](\d{2,3})\]/g;
  
  // 元数据正则表达式
  // 匹配格式：[key:value]
  const metaRegex = /\[([a-z]{2}):(.+)\]/;
  
  for (const line of lines) {
    const trimmedLine = line.trim();
    if (!trimmedLine) continue;
    
    // 尝试匹配元数据标签
    const metaMatch = trimmedLine.match(metaRegex);
    if (metaMatch && !timeTagRegex.test(trimmedLine)) {
      // 这是一个元数据行（没有时间标签）
      const [, key, value] = metaMatch;
      metadata[key] = value;
      continue;
    }
    
    // 提取所有时间标签
    const timeTags: number[] = [];
    let match;
    
    // 重置正则表达式状态
    timeTagRegex.lastIndex = 0;
    
    while ((match = timeTagRegex.exec(trimmedLine)) !== null) {
      const minutes = parseInt(match[1], 10);
      const seconds = parseInt(match[2], 10);
      const hundredths = parseInt(match[3], 10);
      
      // 处理毫秒精度（可能是两位或三位数）
      const milliseconds = match[3].length === 3 ? hundredths : hundredths * 10;
      
      // 计算总秒数
      const time = minutes * 60 + seconds + milliseconds / 1000;
      timeTags.push(time);
    }
    
    // 如果有时间标签，提取歌词文本
    if (timeTags.length > 0) {
      // 移除所有时间标签，获取纯文本
      const text = trimmedLine
        .replace(timeTagRegex, '')
        .replace(/<[^>]+>/g, '')  // 移除额外的时间标签
        .trim();
      
      // 跳过空行
      if (!text) continue;
      
      // 为每个时间标签创建一行歌词
      for (const time of timeTags) {
        lyrics.push({ time, text });
      }
    }
  }
  
  // 按时间排序
  lyrics.sort((a, b) => a.time - b.time);
  
  return { metadata, lyrics };
}

/**
 * 获取当前时间对应的歌词索引
 * 
 * @param lyrics - 歌词数组
 * @param currentTime - 当前播放时间（秒）
 * @returns 当前歌词索引，如果找不到返回 -1
 */
export function getCurrentLyricIndex(
  lyrics: LyricLine[],
  currentTime: number
): number {
  if (!lyrics || lyrics.length === 0) return -1;
  
  // 找到最后一个小于等于当前时间的歌词
  let index = -1;
  for (let i = 0; i < lyrics.length; i++) {
    if (lyrics[i].time <= currentTime) {
      index = i;
    } else {
      break;
    }
  }
  
  return index;
}

/**
 * 生成简单的时间标签
 * 用于创建 LRC 文件
 * 
 * @param seconds - 秒数
 * @returns LRC 格式的时间标签，如 "[00:12.34]"
 */
export function formatTimeTag(seconds: number): string {
  const mins = Math.floor(seconds / 60);
  const secs = Math.floor(seconds % 60);
  const hundredths = Math.floor((seconds % 1) * 100);
  
  return `[${mins.toString().padStart(2, '0')}:${secs.toString().padStart(2, '0')}.${hundredths.toString().padStart(2, '0')}]`;
}

/**
 * 将歌词数组转换为 LRC 格式文本
 */
export function lyricsToLRC(lyrics: LyricLine[], metadata?: Record<string, string>): string {
  const lines: string[] = [];
  
  // 添加元数据
  if (metadata) {
    for (const [key, value] of Object.entries(metadata)) {
      lines.push(`[${key}:${value}]`);
    }
    lines.push('');  // 空行分隔
  }
  
  // 添加歌词行
  for (const line of lyrics) {
    const timeTag = formatTimeTag(line.time);
    lines.push(`${timeTag}${line.text}`);
  }
  
  return lines.join('\n');
}

/**
 * 创建空歌词
 * 用于没有歌词的情况
 */
export function createEmptyLyrics(duration: number): LyricLine[] {
  return [
    { time: 0, text: '♪ 纯音乐 ♪' },
  ];
}
```

```typescript
// audio.ts - 音频工具函数
// 作用：封装音频相关的工具函数

import { Track } from '../stores/playerStore';

/**
 * 从文件路径生成歌曲 ID
 * 使用简单的哈希算法确保唯一性
 */
export function generateTrackId(path: string): string {
  let hash = 0;
  for (let i = 0; i < path.length; i++) {
    const char = path.charCodeAt(i);
    hash = ((hash << 5) - hash) + char;
    hash = hash & hash;  // Convert to 32bit integer
  }
  return Math.abs(hash).toString(36);
}

/**
 * 从文件名解析歌曲信息
 * 尝试从文件名中提取标题和艺术家
 * 
 * @example
 * parseFileName("周杰伦 - 晴天.mp3")
 * // returns { title: "晴天", artist: "周杰伦" }
 */
export function parseFileName(fileName: string): {
  title: string;
  artist: string;
  album: string;
} {
  // 移除文件扩展名
  const nameWithoutExt = fileName.replace(/\.[^/.]+$/, '');
  
  // 常见的分隔符
  const separators = [' - ', ' – ', ' — ', '_', '。'];
  
  // 尝试各种分隔符
  for (const sep of separators) {
    if (nameWithoutExt.includes(sep)) {
      const parts = nameWithoutExt.split(sep);
      if (parts.length >= 2) {
        return {
          artist: parts[0].trim(),
          title: parts.slice(1).join(sep).trim(),
          album: '',
        };
      }
    }
  }
  
  // 如果没有分隔符，整个文件名作为标题
  return {
    title: nameWithoutExt,
    artist: '未知艺术家',
    album: '',
  };
}

/**
 * 格式化文件大小
 */
export function formatFileSize(bytes: number): string {
  if (bytes === 0) return '0 B';
  
  const units = ['B', 'KB', 'MB', 'GB'];
  const k = 1024;
  const i = Math.floor(Math.log(bytes) / Math.log(k));
  
  return `${(bytes / Math.pow(k, i)).toFixed(1)} ${units[i]}`;
}

export function getFileExtension(path: string): string {

}

/**
 * 检查是否是支持的音频格式
 */
export function isSupportedAudioFormat(path: string): boolean {
  const ext = getFileExtension(path);
  const supportedExts = ['mp3', 'flac', 'wav', 'aac', 'ogg', 'webm', 'm4a'];
  return supportedExts.includes(ext);
}

/**
 * 创建音频 Track 对象
 */
export function createTrackFromPath(

  const { title, artist, album } = parseFileName(fileName);
```

```typescript
 * - 跳转
 * - 音量控制
 * - 播放速度控制
 * - 自动播放下一曲
 */
export function useAudio() {
  // 获取音频元素引用
  const audioRef = useRef<HTMLAudioElement | null>(null);
  
  // 获取 store 状态
  const {
    playlist,
    currentTrackIndex,
    isPlaying,
    volume,
    isMuted,
    playbackRate,
    playMode,
    setCurrentTime,
    setDuration,
    playNext,
    setLoading,
  } = usePlayerStore();
  
  // 获取当前曲目
  const currentTrack = currentTrackIndex >= 0 
    ? playlist[currentTrackIndex] 
    : null;
  
  // ==================== 音频元素初始化 ====================
  
  useEffect(() => {
    // 创建音频元素
    if (!audioRef.current) {
      audioRef.current = new Audio();
      audioRef.current.preload = 'metadata';  // 只预加载元数据
    }
    
    const audio = audioRef.current;
    
    // ==================== 事件监听器 ====================
    
    /**
     * 音频加载完成事件
     * 获取音频时长
     */
    const handleLoadedMetadata = () => {
      setDuration(audio.duration);
      setLoading(false);
    };
    
    /**
     * 音频时间更新事件
     * 更新当前播放时间
     */
    const handleTimeUpdate = () => {
      setCurrentTime(audio.currentTime);
    };
    
    /**
     * 音频播放结束事件
     * 根据播放模式决定下一步操作
     */
    const handleEnded = () => {
      switch (playMode) {
        case 'repeat-one':
          // 单曲循环：重新播放当前曲目
          audio.currentTime = 0;
          audio.play();
          break;
        case 'repeat-all':
        case 'shuffle':
        default:
          // 列表循环：播放下一曲
          playNext();
          break;
      }
    };
    
    /**
     * 音频开始加载事件
     */
    const handleLoadStart = () => {
      setLoading(true);
    };
    
    /**
     * 音频错误事件
     */
    const handleError = () => {
      console.error('音频加载错误:', audio.error);
      setLoading(false);
      // 可以在这里显示错误提示
    };
    
    /**
     * 音频可以播放事件
     */
    const handleCanPlay = () => {
      setLoading(false);
    };
    
    // 添加事件监听器
    audio.addEventListener('loadedmetadata', handleLoadedMetadata);
    audio.addEventListener('timeupdate', handleTimeUpdate);
    audio.addEventListener('ended', handleEnded);
    audio.addEventListener('loadstart', handleLoadStart);
    audio.addEventListener('error', handleError);
    audio.addEventListener('canplay', handleCanPlay);
    
    // 清理函数
    return () => {
      audio.removeEventListener('loadedmetadata', handleLoadedMetadata);
      audio.removeEventListener('timeupdate', handleTimeUpdate);
      audio.removeEventListener('ended', handleEnded);
      audio.removeEventListener('loadstart', handleLoadStart);
      audio.removeEventListener('error', handleError);
      audio.removeEventListener('canplay', handleCanPlay);
    };
  }, [playMode, playNext, setCurrentTime, setDuration, setLoading]);
  
  // ==================== 曲目切换处理 ====================
  
  useEffect(() => {
    const audio = audioRef.current;
    if (!audio || !currentTrack) return;
    
    // 设置音频源
    // 使用 file:// 协议加载本地文件
    // 在 Tauri 中可以直接使用文件路径
    audio.src = currentTrack.path;
    
    // 如果是播放状态则开始播放
    if (isPlaying) {
      audio.play().catch((error) => {
        console.error('播放失败:', error);
      });
    }
  }, [currentTrack]);
  
  // ==================== 播放状态处理 ====================
  
  useEffect(() => {
    const audio = audioRef.current;
    if (!audio) return;
    
    if (isPlaying) {
      // 播放
      audio.play().catch((error) => {
        console.error('播放失败:', error);
        usePlayerStore.getState().pause();
      });
    } else {
      // 暂停
      audio.pause();
    }
  }, [isPlaying]);
  
  // ==================== 音量控制 ====================
  
  useEffect(() => {
    const audio = audioRef.current;
    if (!audio) return;
    
    // 设置音量
    audio.volume = isMuted ? 0 : volume;
  }, [volume, isMuted]);
  
  // ==================== 播放速度控制 ====================
  
  useEffect(() => {
    const audio = audioRef.current;
    if (!audio) return;
    
    // 设置播放速度
    audio.playbackRate = playbackRate;
  }, [playbackRate]);
  
  // ==================== 暴露的控制方法 ====================
  
  /**
   * 跳转到指定时间
   */
  const seek = useCallback((time: number) => {
    const audio = audioRef.current;
    if (!audio) return;
    
    audio.currentTime = Math.max(0, Math.min(time, audio.duration || 0));
  }, []);
  
  /**
   * 快进/快退
   * @param delta - 时间变化量（秒），正数为快进，负数为快退
   */
  const skip = useCallback((delta: number) => {
    const audio = audioRef.current;
    if (!audio) return;
    
    audio.currentTime = Math.max(0, Math.min(audio.currentTime + delta, audio.duration || 0));
  }, []);
  
  return {
    audio: audioRef.current,  // 音频元素引用
    seek,                       // 跳转方法
    skip,                       // 快进/快退方法
  };
}
```

```tsx
// Visualizer.tsx - 音频可视化组件
// 作用：使用 Canvas 绘制音频波形和频谱

import React, { useEffect, useRef, useCallback } from 'react';
import { usePlayerStore } from '../stores/playerStore';

/**
 * 音频可视化类型
 */
type VisualizerType = 'bars' | 'wave' | 'circles';

/**
 * 频谱可视化组件
 * 使用 Web Audio API 获取频率数据
 * 使用 Canvas 绘制可视化效果
 */
export const Visualizer: React.FC = () => {
  // Canvas 引用
  const canvasRef = useRef<HTMLCanvasElement>(null);
  
  // 音频上下文和节点引用
  const audioContextRef = useRef<AudioContext | null>(null);
  const analyserRef = useRef<AnalyserNode | null>(null);
  const animationFrameRef = useRef<number | null>(null);
  
  // 可视化类型
  const [visualizerType, setVisualizerType] = React.useState<VisualizerType>('bars');
  
  // 获取播放状态
  const { isPlaying } = usePlayerStore();
  
  // 初始化音频分析器
  const initAudioAnalyser = useCallback(() => {
    // 如果已经有分析器，不再重复创建
    if (analyserRef.current) return;
    
    // 创建音频上下文
    // 注意：在某些浏览器中需要用户交互后才能创建
    const audioContext = new (window.AudioContext || (window as any).webkitAudioContext)();
    audioContextRef.current = audioContext;
    
    // 创建分析器节点
    const analyser = audioContext.createAnalyser();
    analyser.fftSize = 256;  // FFT 大小，决定频率分辨率
    analyser.smoothingTimeConstant = 0.8;  // 平滑系数
    analyserRef.current = analyser;
    
    // 连接音频源到分析器
    // 注意：实际项目中需要从 audio 元素获取 MediaElementSource
    // const source = audioContext.createMediaElementSource(audioElement);
    // source.connect(analyser);
    // analyser.connect(audioContext.destination);
  }, []);
  
  // 绘制可视化
  const draw = useCallback(() => {
    const canvas = canvasRef.current;
    const analyser = analyserRef.current;
    if (!canvas || !analyser) return;
    
    const ctx = canvas.getContext('2d');
    if (!ctx) return;
    
    // 获取频率数据
    const bufferLength = analyser.frequencyBinCount;
    const dataArray = new Uint8Array(bufferLength);
    analyser.getByteFrequencyData(dataArray);
    
    // 清除画布
    const { width, height } = canvas;
    ctx.clearRect(0, 0, width, height);
    
    // 根据类型绘制
    switch (visualizerType) {
      case 'bars':
        drawBars(ctx, dataArray, width, height);
        break;
      case 'wave':
        drawWave(ctx, dataArray, width, height);
        break;
      case 'circles':
        drawCircles(ctx, dataArray, width, height);
        break;
    }
    
    // 继续动画循环
    animationFrameRef.current = requestAnimationFrame(draw);
  }, [visualizerType]);
  
  // 条形可视化
  const drawBars = (
    ctx: CanvasRenderingContext2D,
    dataArray: Uint8Array,
    width: number,
    height: number
  ) => {
    const barCount = 64;  // 柱状条数量
    const barWidth = width / barCount;
    const gap = 2;
    
    // 渐变色
    const gradient = ctx.createLinearGradient(0, height, 0, 0);
    gradient.addColorStop(0, '#4f46e5');   // 底部：靛蓝色
    gradient.addColorStop(0.5, '#ec4899');  // 中间：粉色
    gradient.addColorStop(1, '#f59e0b');    // 顶部：橙色
    
    for (let i = 0; i < barCount; i++) {
      // 从数据数组中取样
      const dataIndex = Math.floor((i / barCount) * dataArray.length);
      const value = dataArray[dataIndex];
      const barHeight = (value / 255) * height;
      
      // 绘制柱状条
      const x = i * barWidth + gap / 2;
      ctx.fillStyle = gradient;
      ctx.fillRect(x, height - barHeight, barWidth - gap, barHeight);
    }
  };
  
  // 波浪可视化
  const drawWave = (
    ctx: CanvasRenderingContext2D,
    dataArray: Uint8Array,
    width: number,
    height: number
  ) => {
    ctx.beginPath();
    ctx.strokeStyle = '#6366f1';
    ctx.lineWidth = 2;
    
    const sliceWidth = width / dataArray.length;
    let x = 0;
    
    for (let i = 0; i < dataArray.length; i++) {
      const v = dataArray[i] / 255;
      const y = (v * height) / 2 + height / 2;
      
      if (i === 0) {
        ctx.moveTo(x, y);
      } else {
        ctx.lineTo(x, y);
      }
      x += sliceWidth;
    }
    
    ctx.stroke();
  };
  
  // 圆形可视化
  const drawCircles = (
    ctx: CanvasRenderingContext2D,
    dataArray: Uint8Array,
    width: number,
    height: number
  ) => {
    const centerX = width / 2;
    const centerY = height / 2;
    const baseRadius = Math.min(width, height) / 4;
    
    for (let i = 0; i < dataArray.length / 4; i++) {
      const value = dataArray[i] / 255;
      const radius = baseRadius + value * 100;
      
      ctx.beginPath();
      ctx.arc(centerX, centerY, radius, 0, Math.PI * 2);
      ctx.strokeStyle = `hsla(${200 + i * 2}, 70%, ${50 + value * 30}%, ${0.3 + value * 0.5})`;
      ctx.lineWidth = 2;
      ctx.stroke();
    }
  };
  
  // 启动/停止动画
  useEffect(() => {
    if (isPlaying) {
      // 初始化分析器
      initAudioAnalyser();
      // 开始绘制
      draw();
    } else {
      // 停止动画
      if (animationFrameRef.current) {
        cancelAnimationFrame(animationFrameRef.current);
      }
    }
    
    return () => {
      if (animationFrameRef.current) {
        cancelAnimationFrame(animationFrameRef.current);
      }
    };
  }, [isPlaying, initAudioAnalyser, draw]);
  
  // 处理画布大小变化
  useEffect(() => {
    const handleResize = () => {
      const canvas = canvasRef.current;
      if (!canvas) return;
      
      canvas.width = canvas.clientWidth;
      canvas.height = canvas.clientHeight;
    };
    
    handleResize();
    window.addEventListener('resize', handleResize);
    
    return () => window.removeEventListener('resize', handleResize);
  }, []);
  
  return (
    <div className="relative h-32 bg-gray-800 rounded-lg overflow-hidden">
      {/* Canvas 画布 */}
      <canvas 
        ref={canvasRef}
        className="w-full h-full"
      />
      
      {/* 可视化类型切换 */}
      <div className="absolute bottom-2 right-2 flex gap-1">
        {(['bars', 'wave', 'circles'] as VisualizerType[]).map((type) => (
          <button
            key={type}
            onClick={() => setVisualizerType(type)}
            className={`
              px-2 py-1 text-xs rounded transition-colors
              ${visualizerType === type 
                ? 'bg-indigo-600 text-white' 
                : 'bg-gray-700 text-gray-300 hover:bg-gray-600'}
            `}
          >
            {type === 'bars' ? '▮' : type === 'wave' ? '∿' : '○'}
          </button>
        ))}
      </div>
    </div>
  );
};
```

```tsx
// 作用：提供播放控制界面，包括播放/暂停、进度条、音量控制等
```

```rust
#[tauri::command]
pub async fn list_audio_files(path: String) -> Result<Vec<AudioFile>, String> {
    log::info!("📁 扫描音频目录: {}", path);
    
    let entries = fs::read_dir(&path).map_err(|e| format!("读取目录失败: {}", e))?;
    
    let mut audio_files = Vec::new();
    
    for entry in entries.flatten() {
        let file_path = entry.path();
        
        // 检查是否是文件
        if !file_path.is_file() {
            continue;
        }
        
        // 检查扩展名
        if let Some(ext) = file_path.extension() {
            let ext_lower = ext.to_string_lossy().to_lowercase();
            if SUPPORTED_FORMATS.contains(&ext_lower.as_str()) {
                let name = file_path
                    .file_name()
                    .and_then(|n| n.to_str())
                    .unwrap_or("未知")
                    .to_string();
                
                let size = file_path
                    .metadata()
                    .map(|m| m.len())
                    .unwrap_or(0);
                
                audio_files.push(AudioFile {
                    path: file_path.to_string_lossy().to_string(),
                    name,
                    size,
                });
            }
        }
    }
    
    // 按文件名排序
    audio_files.sort_by(|a, b| a.name.to_lowercase().cmp(&b.name.to_lowercase()));
    
    log::info!("📋 找到 {} 个音频文件", audio_files.len());
    Ok(audio_files)
}

    // 简化实现：直接搜索 key
    let search = format!("\"{}\"", key);
    if let Some(pos) = json.find(&search) {
        // 找到值的位置
        let rest = &json[pos..];
        if let Some(colon_pos) = rest.find(':') {

                let clean = value.trim().trim_matches('"').trim();
                if !clean.is_empty() {
                    return Some(clean.to_string());
                }
            }
        }
    }
    None
```

### 面试题 1：HTML5 Audio 和 Web Audio API 的区别？

**标准答案**：
| 对比项 | HTML5 Audio | Web Audio API |
|--------|-------------|---------------|
| 用途 | 简单音频播放 | 专业音频处理 |
| API 层级 | 高层 API | 低层 API |
| 音效处理 | 有限 | 强大（均衡器、混响等） |
| 可视化 | 不支持 | 支持（AnalyserNode） |

**应用场景**：
- 简单播放：HTML5 Audio
- 音频可视化：Web Audio API
- 可以结合使用：HTML5 Audio → MediaElementSource → Web Audio API

### 面试题 2：Rust 的 Trait 是什么？和 Interface 有什么区别？

**标准答案**：
```rust
// Trait 定义
trait Drawable {
    fn draw(&self);  // 方法签名
    fn area(&self) -> f64 {  // 带默认实现
        0.0
    }
}

// 实现 Trait
struct Circle { radius: f64 }
impl Drawable for Circle {
    fn draw(&self) {
        println!("画圆形，半径={}", self.radius);
    }
}

// 使用泛型约束
fn render<T: Drawable>(shape: T) {
    shape.draw();
}
```

**与 Interface 的区别**：
1. **默认实现**：Trait 可以有默认实现，Interface 通常没有
2. **静态分发**：泛型 + Trait Bound 在编译时确定
3. **动态分发**：使用 `dyn Trait` 实现运行时多态
4. **孤儿规则**：只能为你的类型或在你的类型上实现 Trait

```rust
    if x.len() > y.len() { x } else { y }
}

// 结构体中的生命周期
struct Excerpt<'a> {
    part: &'a str,  // 必须标注
}

// 'static 生命周期
// 表示程序整个运行期间都有效
let s: &'static str = "hello";
```

### 面试题 4：React 中的 useRef 和 useState 的区别？

**标准答案**：
| 对比项 | useState | useRef |
|--------|----------|--------|
| 用途 | 触发重新渲染 | 不触发重新渲染 |
| 变化 | 产生新值 | 保持引用 |
| 典型场景 | UI 状态 | DOM 引用、定时器 |

```typescript
// useState：需要更新 UI
const [count, setCount] = useState(0);
<button onClick={() => setCount(count + 1)}>{count}</button>

// useRef：不需要更新 UI
const audioRef = useRef<HTMLAudioElement>(null);
const timerRef = useRef<NodeJS.Timeout>();
```

### 面试题 5：Canvas 绑定音频可视化原理？

**标准答案**：
```typescript
// 1. 创建音频上下文和分析器
const audioContext = new AudioContext();
const analyser = audioContext.createAnalyser();
analyser.fftSize = 256;

// 2. 连接音频源
const source = audioContext.createMediaElementSource(audioElement);
source.connect(analyser);
analyser.connect(audioContext.destination);

// 3. 获取频率数据
const dataArray = new Uint8Array(analyser.frequencyBinCount);
analyser.getByteFrequencyData(dataArray);

// 4. 在 Canvas 上绘制
function draw() {
    requestAnimationFrame(draw);
    analyser.getByteFrequencyData(dataArray);
    
    // 清空画布
    ctx.clearRect(0, 0, width, height);
    
    // 绘制频谱条
    const barWidth = width / dataArray.length;
    dataArray.forEach((value, i) => {
        const barHeight = (value / 255) * height;
        ctx.fillRect(i * barWidth, height - barHeight, barWidth, barHeight);
    });
}
```
