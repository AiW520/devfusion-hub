# Ktor 微服务方案二：文件上传服务

## 项目介绍

### 项目是什么
这是一个轻量级的文件上传/下载微服务，使用 Kotlin + Ktor 构建，支持文件上传、下载、管理等功能。

### 解决什么问题
- 为前端应用提供文件存储服务
- 支持多种文件类型（图片、文档、视频等）
- 提供文件访问权限控制
- 支持文件元数据管理

### 核心技术亮点
| 技术 | 说明 |
|------|------|
| Ktor | Kotlin 异步 Web 框架 |
| MinIO/S3 | 对象存储（可替换本地存储） |
| Coroutines Flow | 响应式文件处理 |
| MD5/SHA256 | 文件完整性校验 |

### 适合场景
- 微服务架构中的文件管理服务
- 学习 Ktor 文件处理
- 构建需要文件上传功能的应用

### 4. Storage.kt（存储配置）

```kotlin
package com.example.file.plugin

import io.minio.*
import io.minio.http.*
import io.ktor.server.application.*
import org.slf4j.LoggerFactory

/**
 * 存储配置插件
 * 
 * 支持两种存储方式：
 * 1. MinIO（分布式对象存储，S3 兼容）
 * 2. 本地文件系统（简单场景）
 * 
 * 这里使用 MinIO 作为示例
 */
private val logger = LoggerFactory.getLogger("Storage")

// 存储客户端
lateinit var minioClient: MinioClient

// 存储桶名称
const val BUCKET_NAME = "files"

fun Application.configureStorage() {
    // ===== 初始化 MinIO 客户端 =====
    // MinIO 是一个开源的对象存储服务器，兼容 Amazon S3 API
    
    minioClient = MinioClient.builder()
        .endpoint("http://localhost:9000")           // MinIO API 地址
        .credentials(
            "minioadmin",                             // Access Key
            "minioadmin123"                          // Secret Key
        )
        .build()
    
    logger.info("✅ MinIO client initialized")
    
    // ===== 确保存储桶存在 =====
    ensureBucketExists()
}

/**
 * 确保存储桶存在，不存在则创建
 */
private fun ensureBucketExists() {
    try {
        // bucketExists: 检查存储桶是否存在
        val exists = minioClient.bucketExists(
            BucketExistsArgs.builder()
                .bucket(BUCKET_NAME)
                .build()
        )
        
        if (!exists) {
            // makeBucket: 创建存储桶
            minioClient.makeBucket(
                MakeBucketArgs.builder()
                    .bucket(BUCKET_NAME)
                    .build()
            )
            logger.info("📁 Created bucket: $BUCKET_NAME")
            
            // 设置存储桶策略为公开下载
            setBucketPolicyPublic()
        } else {
            logger.info("📁 Bucket already exists: $BUCKET_NAME")
        }
    } catch (e: Exception) {
        logger.error("❌ Failed to ensure bucket exists: ${e.message}")
        // 存储桶创建失败不影响服务启动
        // 文件操作时会再次尝试
    }
}

/**
 * 设置存储桶为公开访问
 * 允许任何人下载文件
 */
private fun setBucketPolicyPublic() {
    try {
        // JSON 格式的存储桶策略
        val policy = """
        {
            "Version": "2012-10-17",
            "Statement": [
                {
                    "Effect": "Allow",
                    "Principal": {"AWS": ["*"]},
                    "Action": ["s3:GetObject"],
                    "Resource": ["arn:aws:s3:::$BUCKET_NAME/*"]
                }
            ]
        }
        """.trimIndent()
        
        // setBucketPolicy: 设置存储桶策略
        minioClient.setBucketPolicy(
            SetBucketPolicyArgs.builder()
                .bucket(BUCKET_NAME)
                .config(policy)
                .build()
        )
        
        logger.info("🔓 Bucket policy set to public")
    } catch (e: Exception) {
        logger.warn("⚠️ Failed to set bucket policy: ${e.message}")
    }
}

/**
 * 获取 MinIO 客户端
 */
fun getMinioClient(): MinioClient = minioClient
```

```kotlin
package com.example.file.model

import java.time.LocalDateTime

/**
 * 文件元数据实体
 * 
 * 用于存储文件的描述信息
 * 实际文件内容存储在 MinIO 中
 */
data class FileMetadata(
    val id: String,              // UUID 唯一标识
    val originalName: String,    // 原始文件名
    val storedName: String,      // 存储文件名（UUID）
    val extension: String,        // 文件扩展名
    val mimeType: String,         // MIME 类型
    val size: Long,               // 文件大小（字节）
    val md5: String,             // MD5 校验码
    val bucket: String,           // 存储桶名称
    val path: String,            // 存储路径
    val uploadedBy: Int?,        // 上传者用户 ID
    val isPublic: Boolean,        // 是否公开访问
    val createdAt: LocalDateTime,
    val updatedAt: LocalDateTime
) {
    /**
     * 获取完整文件路径
     */
    fun getFullPath(): String = "$path/$storedName"
    
    /**
     * 获取文件大小的人类可读格式
     */
    fun getReadableSize(): String {
        return when {
            size < 1024 -> "$size B"
            size < 1024 * 1024 -> String.format("%.2f KB", size / 1024.0)
            size < 1024 * 1024 * 1024 -> String.format("%.2f MB", size / (1024.0 * 1024))
            else -> String.format("%.2f GB", size / (1024.0 * 1024 * 1024))
        }
    }
}

/**
 * 文件类型分类
 */
enum class FileCategory(val extensions: List<String>, val mimePrefix: String) {
    IMAGE(listOf("jpg", "jpeg", "png", "gif", "bmp", "webp", "svg"), "image/"),
    VIDEO(listOf("mp4", "avi", "mov", "wmv", "flv", "mkv", "webm"), "video/"),
    AUDIO(listOf("mp3", "wav", "flac", "aac", "ogg", "m4a"), "audio/"),
    DOCUMENT(listOf("pdf", "doc", "docx", "xls", "xlsx", "ppt", "pptx", "txt", "md"), "application/"),
    ARCHIVE(listOf("zip", "rar", "7z", "tar", "gz"), "application/"),
    OTHER(emptyList(), "");
    
    companion object {
        /**
         * 根据 MIME 类型判断分类
         */
        fun fromMimeType(mimeType: String): FileCategory {
            return entries.find { 
                mimeType.startsWith(it.mimePrefix) && it.mimePrefix.isNotEmpty() 
            } ?: OTHER
        }
        
        /**
         * 根据扩展名判断分类
         */
        fun fromExtension(extension: String): FileCategory {
            val ext = extension.lowercase()
            return entries.find { ext in it.extensions } ?: OTHER
        }
    }
}
```

### 7. FileService.kt

```kotlin
package com.example.file.service

import com.example.file.plugin.*
import com.example.file.model.*
import com.example.file.dto.*
import io.minio.*
import io.minio.http.*
import io.ktor.server.application.*
import io.ktor.http.*
import io.ktor.utils.io.*
import kotlinx.coroutines.Dispatchers
import kotlinx.coroutines.withContext
import org.apache.tika.Tika
import org.apache.tika.mime.MimeType
import org.apache.tika.mime.MimeTypes
import java.io.*
import java.security.*
import java.time.*
import java.util.*

/**
 * 文件服务
 * 
 * 职责：
 * - 文件上传到 MinIO
 * - 文件下载
 * - 文件元数据管理
 * - 文件类型检测
 * - 完整性校验
 */
class FileService {
    
    // Tika 用于检测文件 MIME 类型
    private val tika = Tika()
    
    // MIME 类型注册表
    private val mimeTypes = MimeTypes.getDefaultMimeTypes()
    
    // 文件元数据存储（实际项目应使用数据库）
    private val fileMetadataStore = mutableMapOf<String, FileMetadata>()
    
    /**
     * 上传文件
     * 
     * 流程：
     * 1. 生成唯一文件 ID
     * 2. 检测文件类型
     * 3. 计算 MD5
     * 4. 上传到 MinIO
     * 5. 保存元数据
     * 
     * @param fileName 原始文件名
     * @param contentType MIME 类型
     * @param fileBytes 文件内容
     * @param userId 上传者 ID
     * @param isPublic 是否公开
     * @return 文件元数据
     */
    suspend fun uploadFile(
        fileName: String,
        contentType: String?,
        fileBytes: ByteArray,
        userId: Int?,
        isPublic: Boolean = false
    ): FileMetadata = withContext(Dispatchers.IO) {
        // ===== 1. 提取文件扩展名 =====
        val extension = extractExtension(fileName)
        
        // ===== 2. 检测真实 MIME 类型 =====
        // 使用 Apache Tika 检测文件内容，而不是依赖客户端声明
        val detectedMimeType = detectMimeType(fileBytes, extension)
        val finalMimeType = contentType ?: detectedMimeType
        
        // ===== 3. 生成唯一 ID 和存储名 =====
        val fileId = UUID.randomUUID().toString()
        val storedName = "$fileId.$extension"
        
        // ===== 4. 计算 MD5 校验码 =====
        val md5 = calculateMD5(fileBytes)
        
        // ===== 5. 上传到 MinIO =====
        uploadToMinio(storedName, fileBytes, finalMimeType, isPublic)
        
        // ===== 6. 创建元数据 =====
        val now = LocalDateTime.now()
        val metadata = FileMetadata(
            id = fileId,
            originalName = fileName,
            storedName = storedName,
            extension = extension,
            mimeType = finalMimeType,
            size = fileBytes.size.toLong(),
            md5 = md5,
            bucket = BUCKET_NAME,
            path = "",
            uploadedBy = userId,
            isPublic = isPublic,
            createdAt = now,
            updatedAt = now
        )
        
        // ===== 7. 保存元数据 =====
        fileMetadataStore[fileId] = metadata
        
        return@withContext metadata
    }
    
    /**
     * 获取文件
     * 
     * @param fileId 文件 ID
     * @return 文件对象（包含内容和元数据）
     */
    suspend fun getFile(fileId: String): FileResult? = withContext(Dispatchers.IO) {
        // 获取元数据
        val metadata = fileMetadataStore[fileId] ?: return@withContext null
        
        // 从 MinIO 下载
        val fileBytes = downloadFromMinio(metadata.storedName)
            ?: return@withContext null
        
        return@withContext FileResult(
            metadata = metadata,
            content = fileBytes
        )
    }
    
    /**
     * 获取文件元数据
     */
    suspend fun getMetadata(fileId: String): FileMetadata? {
        return fileMetadataStore[fileId]
    }
    
    /**
     * 获取文件列表
     */
    suspend fun listFiles(
        userId: Int? = null,
        category: String? = null,
        page: Int = 1,
        pageSize: Int = 20
    ): FileListResponse {
        // 过滤文件
        var files = fileMetadataStore.values.toList()
        
        if (userId != null) {
            files = files.filter { it.uploadedBy == userId }
        }
        
        if (category != null) {
            files = files.filter { 
                FileCategory.fromMimeType(it.mimeType).name == category 
            }
        }
        
        // 按时间倒序
        files = files.sortedByDescending { it.createdAt }
        
        // 分页
        val total = files.size
        val start = (page - 1) * pageSize
        val end = minOf(start + pageSize, files.size)
        val pagedFiles = if (start < files.size) files.subList(start, end) else emptyList()
        
        return FileListResponse(
            files = pagedFiles.map { it.toResponse("http://localhost:8080") },
            total = total,
            page = page,
            pageSize = pageSize
        )
    }
    
    /**
     * 删除文件
     */
    suspend fun deleteFile(fileId: String, userId: Int?): Boolean = withContext(Dispatchers.IO) {
        val metadata = fileMetadataStore[fileId] ?: return@withContext false
        
        // 检查权限：只有上传者可以删除
        if (userId != null && metadata.uploadedBy != userId) {
            return@withContext false
        }
        
        // 从 MinIO 删除
        deleteFromMinio(metadata.storedName)
        
        // 删除元数据
        fileMetadataStore.remove(fileId)
        
        return@withContext true
    }
    
    /**
     * 获取文件预览 URL（带过期时间）
     */
    suspend fun getPresignedUrl(fileId: String, expirySeconds: Int = 3600): String? {
        val metadata = fileMetadataStore[fileId] ?: return null
        
        return try {
            // presignedGetObject: 生成带签名的临时访问 URL
            // 签名 URL 在过期时间后自动失效
            val url = minioClient.presignedGetObject(
                PresignedGetObjectArgs.builder()
                    .bucket(BUCKET_NAME)
                    .`object`(metadata.storedName)
                    .expiry(expirySeconds)  // 过期时间（秒）
                    .build()
            )
            url
        } catch (e: Exception) {
            null
        }
    }
    
    // ==================== 私有方法 ====================
    
    /**
     * 提取文件扩展名
     */
    private fun extractExtension(fileName: String): String {
        val lastDot = fileName.lastIndexOf('.')
        return if (lastDot > 0 && lastDot < fileName.length - 1) {
            fileName.substring(lastDot + 1).lowercase()
        } else {
            "bin"  // 默认扩展名
        }
    }
    
    /**
     * 检测文件 MIME 类型
     * 
     * 使用 Apache Tika 根据文件内容检测类型
     * 这是最准确的方式，不依赖文件扩展名或客户端声明
     */
    private fun detectMimeType(fileBytes: ByteArray, extension: String): String {
        return try {
            // tika.parse: 检测文件内容
            val detected = tika.detect(ByteArrayInputStream(fileBytes))
            if (detected != null && detected != "application/octet-stream") {
                detected
            } else {
                // 回退：根据扩展名判断
                guessMimeType(extension)
            }
        } catch (e: Exception) {
            guessMimeType(extension)
        }
    }
    
    /**
     * 根据扩展名猜测 MIME 类型
     */
    private fun guessMimeType(extension: String): String {
        return try {
            val mimeType = mimeTypes.forName(".$extension")
            mimeType.toString()
        } catch (e: Exception) {
            "application/octet-stream"  // 默认二进制类型
        }
    }
    
    /**
     * 计算 MD5 校验码
     */
    private fun calculateMD5(data: ByteArray): String {
        val md = MessageDigest.getInstance("MD5")
        val digest = md.digest(data)
        return digest.joinToString("") { "%02x".format(it) }
    }
    
    /**
     * 上传到 MinIO
     */
    private fun uploadToMinio(
        objectName: String,
        data: ByteArray,
        contentType: String,
        isPublic: Boolean
    ) {
        val inputStream = ByteArrayInputStream(data)
        
        // PutObjectArgs: 上传参数
        val args = PutObjectArgs.builder()
            .bucket(BUCKET_NAME)
            .`object`(objectName)           // 对象名称
            .stream(inputStream, data.size.toLong(), -1)  // 输入流和大小
            .contentType(contentType)       // MIME 类型
            .build()
        
        minioClient.putObject(args)
    }
    
    /**
     * 从 MinIO 下载
     */
    private fun downloadFromMinio(objectName: String): ByteArray? {
        return try {
            val stream = minioClient.getObject(
                GetObjectArgs.builder()
                    .bucket(BUCKET_NAME)
                    .`object`(objectName)
                    .build()
            )
            
            stream.readAllBytes()
        } catch (e: Exception) {
            null
        }
    }
    
    /**
     * 从 MinIO 删除
     */
    private fun deleteFromMinio(objectName: String) {
        try {
            minioClient.removeObject(
                RemoveObjectArgs.builder()
                    .bucket(BUCKET_NAME)
                    .`object`(objectName)
                    .build()
            )
        } catch (e: Exception) {
            // 忽略删除失败
        }
    }
}

/**
 * 文件下载结果
 */
data class FileResult(
    val metadata: FileMetadata,
    val content: ByteArray
) {
    override fun equals(other: Any?): Boolean {
        if (this === other) return true
        if (javaClass != other?.javaClass) return false
        other as FileResult
        return metadata == other.metadata && content.contentEquals(other.content)
    }
    
    override fun hashCode(): Int {
        var result = metadata.hashCode()
        result = 31 * result + content.contentHashCode()
        return result
    }
}
```

### 8. FileRoute.kt（文件路由）

```kotlin
package com.example.file.route

import com.example.file.service.*
import com.example.file.dto.*
import io.ktor.server.application.*
import io.ktor.server.request.*
import io.ktor.server.response.*
import io.ktor.server.routing.*
import io.ktor.http.*
import io.ktor.utils.io.*
import kotlinx.coroutines.Dispatchers
import kotlinx.coroutines.withContext

/**
 * 文件路由
 * 
 * 支持：
 * - 单文件上传
 * - 多文件上传
 * - 文件下载
 * - 文件预览（图片）
 * - 文件元数据查询
 * - 文件删除
 */
fun Routing.configureFileRoute(fileService: FileService) {
    
    route("/api/files") {
        
        /**
         * POST /api/files/upload - 上传单个文件
         * 
         * 支持 multipart/form-data 格式
         */
        post("/upload") {
            // ===== 1. 解析 multipart 数据 =====
            val multipart = call.receiveMultipart()
            
            // ===== 2. 获取请求头参数 =====
            val isPublic = call.request.headers["X-File-Public"]?.toBoolean() ?: false
            val userId = call.request.headers["X-User-Id"]?.toIntOrNull()
            
            // ===== 3. 处理文件数据 =====
            var fileMetadata: FileMetadata? = null
            var fileBytes: ByteArray? = null
            var fileName = "unknown"
            var contentType: String? = null
            
            // consumePart: 遍历 multipart 的每个部分
            multipart.forEachPart { part ->
                when (part) {
                    is PartData.FileItem -> {
                        // 读取文件名
                        fileName = part.originalFileName ?: "unknown"
                        
                        // 读取内容类型
                        contentType = part.contentType?.toString()
                        
                        // 读取文件内容
                        // readRemaining: 读取剩余所有字节
                        fileBytes = part.streamProvider().buffered().readBytes()
                    }
                    is PartData.FormItem -> {
                        // 处理普通表单字段（如果有）
                        println("Form field: ${part.name} = ${part.value}")
                    }
                }
            }
            
            // ===== 4. 验证文件 =====
            if (fileBytes == null || fileBytes!!.isEmpty()) {
                call.respond(HttpStatusCode.BadRequest, mapOf("error" to "文件不能为空"))
                return@post
            }
            
            // 大小限制：100MB
            if (fileBytes!!.size > 100 * 1024 * 1024) {
                call.respond(HttpStatusCode.BadRequest, mapOf("error" to "文件大小不能超过 100MB"))
                return@post
            }
            
            // ===== 5. 上传文件 =====
            try {
                fileMetadata = fileService.uploadFile(
                    fileName = fileName,
                    contentType = contentType,
                    fileBytes = fileBytes!!,
                    userId = userId,
                    isPublic = isPublic
                )
                
                // ===== 6. 返回结果 =====
                call.respond(HttpStatusCode.Created, 
                    fileMetadata!!.toUploadResponse("http://localhost:8080")
                )
                
            } catch (e: Exception) {
                call.respond(
                    HttpStatusCode.InternalServerError,
                    mapOf("error" to "上传失败: ${e.message}")
                )
            }
        }
        
        /**
         * POST /api/files/upload/multiple - 上传多个文件
         */
        post("/upload/multiple") {
            val multipart = call.receiveMultipart()
            val isPublic = call.request.headers["X-File-Public"]?.toBoolean() ?: false
            val userId = call.request.headers["X-User-Id"]?.toIntOrNull()
            
            val uploadedFiles = mutableListOf<FileUploadResponse>()
            val errors = mutableListOf<String>()
            
            multipart.forEachPart { part ->
                when (part) {
                    is PartData.FileItem -> {
                        val fileName = part.originalFileName ?: "unknown"
                        val contentType = part.contentType?.toString()
                        val fileBytes = part.streamProvider().buffered().readBytes()
                        
                        // 检查大小
                        if (fileBytes.size <= 100 * 1024 * 1024) {
                            try {
                                val metadata = fileService.uploadFile(
                                    fileName, contentType, fileBytes, userId, isPublic
                                )
                                uploadedFiles.add(metadata.toUploadResponse("http://localhost:8080"))
                            } catch (e: Exception) {
                                errors.add("$fileName: ${e.message}")
                            }
                        } else {
                            errors.add("$fileName: 文件过大")
                        }
                    }
                    else -> {}
                }
            }
            
            call.respond(HttpStatusCode.Created, mapOf(
                "uploaded" to uploadedFiles,
                "errors" to errors
            ))
        }
        
        /**
         * GET /api/files - 获取文件列表
         */
        get {
            val userId = call.parameters["userId"]?.toIntOrNull()
            val category = call.parameters["category"]
            val page = call.parameters["page"]?.toIntOrNull() ?: 1
            val pageSize = call.parameters["pageSize"]?.toIntOrNull() ?: 20
            
            val response = fileService.listFiles(userId, category, page, pageSize)
            call.respond(HttpStatusCode.OK, response)
        }
        
        /**
         * GET /api/files/{id} - 获取文件元数据
         */
        get("/{id}") {
            val fileId = call.parameters["id"] ?: run {
                call.respond(HttpStatusCode.BadRequest, mapOf("error" to "文件 ID 不能为空"))
                return@get
            }
            
            val metadata = fileService.getMetadata(fileId)
            
            if (metadata == null) {
                call.respond(HttpStatusCode.NotFound, mapOf("error" to "文件不存在"))
                return@get
            }
            
            call.respond(HttpStatusCode.OK, metadata.toResponse("http://localhost:8080"))
        }
        
        /**
         * GET /api/files/{id}/download - 下载文件
         */
        get("/{id}/download") {
            val fileId = call.parameters["id"] ?: run {
                call.respond(HttpStatusCode.BadRequest, mapOf("error" to "文件 ID 不能为空"))
                return@get
            }
            
            val fileResult = fileService.getFile(fileId)
            
            if (fileResult == null) {
                call.respond(HttpStatusCode.NotFound, mapOf("error" to "文件不存在"))
                return@get
            }
            
            // 设置响应头
            call.response.headers.append(
                HttpHeaders.ContentDisposition,
                ContentDisposition.Attachment
                    .withParameter(ContentDisposition.Parameters.FileName, fileResult.metadata.originalName)
                    .toString()
            )
            
            call.respondBytes(
                contentType = ContentType.parse(fileResult.metadata.mimeType),
                bytes = fileResult.content
            )
        }
        
        /**
         * GET /api/files/{id}/preview - 预览文件（图片）
         * 直接返回图片数据，用于 <img> 标签
         */
        get("/{id}/preview") {
            val fileId = call.parameters["id"] ?: run {
                call.respond(HttpStatusCode.BadRequest, mapOf("error" to "文件 ID 不能为空"))
                return@get
            }
            
            val fileResult = fileService.getFile(fileId)
            
            if (fileResult == null) {
                call.respond(HttpStatusCode.NotFound, mapOf("error" to "文件不存在"))
                return@get
            }
            
            // 检查是否是图片
            if (!fileResult.metadata.mimeType.startsWith("image/")) {
                call.respond(HttpStatusCode.BadRequest, mapOf("error" to "不是图片文件"))
                return@get
            }
            
            call.respondBytes(
                contentType = ContentType.parse(fileResult.metadata.mimeType),
                bytes = fileResult.content
            )
        }
        
        /**
         * GET /api/files/{id}/url - 获取临时访问 URL
         */
        get("/{id}/url") {
            val fileId = call.parameters["id"] ?: run {
                call.respond(HttpStatusCode.BadRequest, mapOf("error" to "文件 ID 不能为空"))
                return@get
            }
            
            val expiry = call.parameters["expiry"]?.toIntOrNull() ?: 3600  // 默认 1 小时
            
            val url = fileService.getPresignedUrl(fileId, expiry)
            
            if (url == null) {
                call.respond(HttpStatusCode.NotFound, mapOf("error" to "文件不存在"))
                return@get
            }
            
            call.respond(HttpStatusCode.OK, mapOf("url" to url, "expiresIn" to expiry))
        }
        
        /**
         * DELETE /api/files/{id} - 删除文件
         */
        delete("/{id}") {
            val fileId = call.parameters["id"] ?: run {
                call.respond(HttpStatusCode.BadRequest, mapOf("error" to "文件 ID 不能为空"))
                return@delete
            }
            
            val userId = call.request.headers["X-User-Id"]?.toIntOrNull()
            
            val success = fileService.deleteFile(fileId, userId)
            
            if (success) {
                call.respond(HttpStatusCode.NoContent)
            } else {
                call.respond(HttpStatusCode.NotFound, mapOf("error" to "文件不存在或无权删除"))
            }
        }
    }
    
    // ===== 公开访问路由 =====
    // 这些路由不需要认证，用于公开文件的访问
    
    /**
     * GET /files/{id} - 公开文件访问
     * 可以直接在浏览器中打开
     */
    get("/files/{id}") {
        val fileId = call.parameters["id"] ?: run {
            call.respond(HttpStatusCode.BadRequest)
            return@get
        }
        
        val fileResult = fileService.getFile(fileId)
        
        if (fileResult == null) {
            call.respond(HttpStatusCode.NotFound)
            return@get
        }
        
        // 检查是否公开
        if (!fileResult.metadata.isPublic) {
            call.respond(HttpStatusCode.Forbidden, mapOf("error" to "文件不公开"))
            return@get
        }
        
        call.respondBytes(
            contentType = ContentType.parse(fileResult.metadata.mimeType),
            bytes = fileResult.content
        )
    }
}
```

## 代码关键点说明

### 1. 文件上传处理

**关键点**：使用 Ktor 的 `receiveMultipart()` 处理文件上传。

```kotlin
// ❌ 错误：直接读取请求体
val bytes = call.receive<ByteArray>()

// ✅ 正确：使用 multipart 处理
val multipart = call.receiveMultipart()
multipart.forEachPart { part ->
    when (part) {
        is PartData.FileItem -> {
            val fileName = part.originalFileName
            val contentType = part.contentType
            val bytes = part.streamProvider().readBytes()
        }
    }
}
```

**可能踩的坑**：
```kotlin
// 1. 没有关闭流
val stream = part.streamProvider()
// 使用后要关闭，否则资源泄漏
stream.close()

// 2. 没有检查文件大小
if (bytes.size > MAX_SIZE) {
    throw IllegalStateException("File too large")
}

// 3. 没有验证文件类型
val mimeType = tika.detect(bytes)  // 使用内容检测，不要信任客户端声明
```

### 2. MinIO 对象存储

**关键点**：MinIO 是 S3 兼容的对象存储。

```kotlin
// 上传文件
minioClient.putObject(
    PutObjectArgs.builder()
        .bucket(bucket)
        .`object`(objectName)
        .stream(inputStream, size, -1)
        .contentType(contentType)
        .build()
)

// 下载文件
val stream = minioClient.getObject(
    GetObjectArgs.builder()
        .bucket(bucket)
        .`object`(objectName)
        .build()
)
val bytes = stream.readAllBytes()

// 生成签名 URL
val url = minioClient.presignedGetObject(
    PresignedGetObjectArgs.builder()
        .bucket(bucket)
        .`object`(objectName)
        .expiry(3600)  // 1 小时过期
        .build()
)
```

### 3. 文件类型检测

**关键点**：使用 Apache Tika 检测文件真实类型。

```kotlin
// Tika 可以根据文件内容（魔术字节）判断类型
val tika = Tika()
val mimeType = tika.detect(ByteArrayInputStream(fileBytes))
// 比如：PNG 图片的魔术字节是 89 50 4E 47
```

### 题目1：Kotlin 的 Flow 是什么？和 RxJava 有什么区别？

**标准答案**：

Kotlin Flow 是 Kotlin 协程库中的**响应式流处理库**，用于处理异步数据序列。

**核心对比**：

| 特性 | Kotlin Flow | RxJava |
|------|-------------|--------|
| 线程模型 | 基于协程 | 自己的线程调度 |
| 背压 | suspend 支持 | 内置背压 |
| 学习曲线 | 简单 | 陡峭 |
| 体积 | 小（协程内置） | 大 |
| 操作符 | 较少但够用 | 非常丰富 |

**代码示例**：
```kotlin
// Flow 创建
fun getFiles(): Flow<File> = flow {
    files.forEach { emit(it) }
}

// Flow 收集
filesFlow.collect { file ->
    println(file.name)
}

// 背压处理
filesFlow.buffer().collect { process(it) }
```

### 题目3：Kotlin Sealed Class 和 Enum 有什么区别？

**标准答案**：

| 特性 | Sealed Class | Enum |
|------|--------------|------|
| 子类数量 | 有限但可扩展 | 固定 |
| 子类类型 | 可以是不同类 | 只能是单例 |
| 构造参数 | 可以有不同参数 | 必须有相同参数 |
| 方法实现 | 可以有不同的实现 | 所有枚举共用 |
| 状态 | 可以有状态 | 通常没有状态 |

**代码示例**：
```kotlin
// Sealed Class - 子类可以有不同结构
sealed class Result<out T> {
    data class Success<T>(val data: T) : Result<T>()
    data class Error(val message: String) : Result<Nothing>()
    object Loading : Result<Nothing>()
}

// Enum - 所有值有相同结构
enum class Priority {
    LOW, MEDIUM, HIGH
}
```

**在项目中的应用**：
```kotlin
// API 响应使用 Sealed Class
sealed class ApiResponse<out T> {
    data class Ok<T>(val data: T) : ApiResponse<T>()
    data class Error(val code: Int, val message: String) : ApiResponse<Nothing>()
    object Loading : ApiResponse<Nothing>()
}
```
