# 方案一：Express + React 任务看板系统

## 项目介绍

### 项目是什么
这是一个团队协作任务看板应用，类似 Trello/Jira 的看板功能。支持多项目管理、任务卡片拖拽、成员分配、评论协作等功能。

### 解决什么问题
- 团队任务管理混乱的问题
- 展示 React 拖拽交互能力
- 学习实时协作的 WebSocket 基础

### 核心技术亮点
- **React DnD**: 拖拽交互实现
- **WebSocket**: 实时任务更新
- **TypeScript**: 完整的类型安全保障
- **Context + Reducer**: 复杂状态管理

### 适合什么场景
- 团队协作工具
- 项目管理后台
- 个人任务规划

import mongoose, { Schema, Document } from 'mongoose';
import { ICard, CardPriority, IComment, IAttachment, ICardLabel } from '../types';

export interface CardDocument extends ICard, Document {}

      maxlength: [2000, '评论最多2000个字符'],
    },
    author: {
      type: Schema.Types.ObjectId,
      ref: 'User',
      required: true,
    },
    mentions: [{
      type: Schema.Types.ObjectId,
      ref: 'User',
    }],
  },
  { timestamps: true }
);

// 附件子 Schema
const attachmentSchema = new Schema<IAttachment>(
  {
    name: { type: String, required: true },
    url: { type: String, required: true },
    type: { type: String, required: true },
    size: { type: Number, required: true },
    uploadedBy: {
      type: Schema.Types.ObjectId,
      ref: 'User',
      required: true,
    },
  },
  { timestamps: true }
);

// 标签子 Schema
const labelSchema = new Schema<ICardLabel>(
  {
    id: { type: String, required: true },
    name: { type: String, required: true },
    color: { type: String, required: true },
  },
  { _id: false }
);

// 卡片主 Schema
const cardSchema = new Schema<CardDocument>(
  {
    title: {
      type: String,
      required: [true, '卡片标题不能为空'],
      trim: true,
      maxlength: [200, '标题最多200个字符'],
    },
    description: {
      type: String,
      maxlength: [10000, '描述最多10000个字符'],
    },
    board: {
      type: Schema.Types.ObjectId,
      ref: 'Board',
      required: true,
      index: true,
    },
    columnId: {
      type: String,  // 使用字符串 ID，因为列是内嵌的
      required: true,
    },
    assignee: {
      type: Schema.Types.ObjectId,
      ref: 'User',
      default: null,
    },
    labels: [labelSchema],
    priority: {
      type: String,
      enum: Object.values(CardPriority),
      default: CardPriority.Medium,
    },
    dueDate: {
      type: Date,
      default: null,
    },
    comments: [commentSchema],
    attachments: [attachmentSchema],
    order: {
      type: Number,
      default: 0,
    },
  },
  {
    timestamps: true,
    toJSON: {
      virtuals: true,
      transform: (_doc, ret) => {
        ret.id = ret._id;
        delete ret._id;
        delete ret.__v;
        return ret;
      },
    },
  }
);

// 索引
cardSchema.index({ board: 1, columnId: 1, order: 1 });
cardSchema.index({ assignee: 1 });
cardSchema.index({ dueDate: 1 });

// 虚拟字段：是否逾期
cardSchema.virtual('isOverdue').get(function (this: CardDocument) {
  if (!this.dueDate) return false;
  return new Date() > this.dueDate;
});

// 静态方法：按列获取卡片
cardSchema.statics.findByColumn = async function (
  boardId: string,
  columnId: string
): Promise<CardDocument[]> {
  return this.find({ board: boardId, columnId })
    .populate('assignee', 'username avatar')
    .populate('comments.author', 'username avatar')
    .sort({ order: 1 });
};

// 静态方法：批量获取看板所有卡片
cardSchema.statics.findByBoard = async function (
  boardId: string
): Promise<CardDocument[]> {
  return this.find({ board: boardId })
    .populate('assignee', 'username avatar')
    .populate('comments.author', 'username avatar')
    .sort({ columnId: 1, order: 1 });
};

export const Card = mongoose.model<CardDocument>('Card', cardSchema, 'cards');

export default Card;

**backend/src/websocket/index.ts**

/**
 * WebSocket 服务
 * 处理实时通信，支持任务协作
 */

import { Server as HttpServer } from 'http';
import { Server, Socket } from 'socket.io';
import jwt from 'jsonwebtoken';
import { config } from '../config';
import { WebSocketEvent, WSMessage, JWTPayload } from '../types';

/**
 * WebSocket 命名空间
 */
interface AuthenticatedSocket extends Socket {
  userId?: string;
  userName?: string;
}

class WebSocketService {
  private io: Server;
  private userSockets: Map<string, Set<string>>;  // userId -> Set<socketId>

  constructor() {
    this.io = new Server({
      cors: {
        origin: config.server.corsOrigins,
        credentials: true,
      },
    });
    this.userSockets = new Map();
    this.setupMiddleware();
    this.setupEventHandlers();
  }

  /**
   * 初始化 WebSocket 服务
   */
  initialize(server: HttpServer): Server {
    this.io.attach(server);
    console.log('🔌 WebSocket 服务已初始化');
    return this.io;
  }

  /**
   * 中间件：认证
   */
  private setupMiddleware(): void {
    this.io.use((socket: AuthenticatedSocket, next) => {
      try {
        const token = socket.handshake.auth.token || 
                      socket.handshake.headers.authorization?.split(' ')[1];
        
        if (!token) {
          return next(new Error('未提供认证 Token'));
        }

        // 验证 Token
        const decoded = jwt.verify(token, config.jwt.secret) as JWTPayload;
        socket.userId = decoded.userId;
        socket.userName = decoded.username;
        
        next();
      } catch (error) {
        next(new Error('无效的 Token'));
      }
    });
  }

  /**
   * 事件处理
   */
  private setupEventHandlers(): void {
    this.io.on('connection', (socket: AuthenticatedSocket) => {
      console.log(`🔗 用户 ${socket.userName} (${socket.id}) 已连接`);

      // 记录用户 socket
      this.addUserSocket(socket.userId!, socket.id);

      // 加入项目房间
      socket.on('join:project', (projectId: string) => {
        socket.join(`project:${projectId}`);
        console.log(`用户 ${socket.userName} 加入项目 ${projectId}`);
      });

      // 离开项目房间
      socket.on('leave:project', (projectId: string) => {
        socket.leave(`project:${projectId}`);
        console.log(`用户 ${socket.userName} 离开项目 ${projectId}`);
      });

      // 加入看板房间
      socket.on('join:board', (boardId: string) => {
        socket.join(`board:${boardId}`);
        console.log(`用户 ${socket.userName} 加入看板 ${boardId}`);
      });

      // 离开看板房间
      socket.on('leave:board', (boardId: string) => {
        socket.leave(`board:${boardId}`);
        console.log(`用户 ${socket.userName} 离开看板 ${boardId}`);
      });

      // 断开连接
      socket.on('disconnect', () => {
        console.log(`🔌 用户 ${socket.userName} (${socket.id}) 已断开`);
        this.removeUserSocket(socket.userId!, socket.id);
      });
    });
  }

  /**
   * 添加用户 socket
   */
  private addUserSocket(userId: string, socketId: string): void {
    if (!this.userSockets.has(userId)) {
      this.userSockets.set(userId, new Set());
    }
    this.userSockets.get(userId)!.add(socketId);
  }

  /**
   * 移除用户 socket
   */
  private removeUserSocket(userId: string, socketId: string): void {
    const sockets = this.userSockets.get(userId);
    if (sockets) {
      sockets.delete(socketId);
      if (sockets.size === 0) {
        this.userSockets.delete(userId);
      }
    }
  }

  /**
   * 广播到项目房间
   */
  broadcastToProject<T>(projectId: string, event: WebSocketEvent, payload: T, excludeUserId?: string): void {
    const room = `project:${projectId}`;
    
    this.io.to(room).except(`user:${excludeUserId}`).emit(event, {
      event,
      payload,
      timestamp: Date.now(),
    } as WSMessage<T>);
  }

  /**
   * 广播到看板房间
   */
  broadcastToBoard<T>(boardId: string, event: WebSocketEvent, payload: T, excludeUserId?: string): void {
    const room = `board:${boardId}`;
    
    this.io.to(room).emit(event, {
      event,
      payload,
      timestamp: Date.now(),
      userId: excludeUserId,
    } as WSMessage<T>);
  }

  /**
   * 发送私信给用户
   */
  sendToUser<T>(userId: string, event: WebSocketEvent, payload: T): void {
    const sockets = this.userSockets.get(userId);
    if (sockets) {
      sockets.forEach(socketId => {
        this.io.to(socketId).emit(event, {
          event,
          payload,
          timestamp: Date.now(),
        } as WSMessage<T>);
      });
    }
  }

  /**
   * 获取 IO 实例
   */
  getIO(): Server {
    return this.io;
  }
}

// 单例导出
export const wsService = new WebSocketService();
export default wsService;

/**
 * 卡片控制器
 * 处理卡片的 CRUD 和移动操作

import { Request, Response, NextFunction } from 'express';
import { Card } from '../models/Card';
import { Board } from '../models/Board';
import { wsService } from '../websocket';
import { WebSocketEvent } from '../types';
import { ApiError, asyncHandler } from '../middleware/errorHandler';

const asyncErr = asyncHandler;

/**
 * 获取看板的所有卡片
 * GET /api/v1/boards/:boardId/cards
 */
export const getCards = asyncErr(async (
  req: Request,
  res: Response
) => {
  const { boardId } = req.params;
  
  const cards = await Card.findByBoard(boardId);
  
  return res.json({
    success: true,
    message: '获取成功',
    data: cards,
  });
});

/**
 * 创建卡片
 * POST /api/v1/boards/:boardId/cards
 */
export const createCard = asyncErr(async (
  req: Request,
  res: Response
) => {
  const { boardId } = req.params;
  const userId = (req as any).user?.userId;
  const { title, description, columnId, assignee, labels, priority, dueDate } = req.body;
  
  // 验证看板存在
  const board = await Board.findById(boardId);
  if (!board) {
    throw new ApiError('看板不存在', 404);
  }
  
  // 验证列存在
  const column = board.columns.find(c => c._id?.toString() === columnId || c._id === columnId);
  if (!column) {
    throw new ApiError('列不存在', 400);
  }
  
  // 获取该列最后一张卡片的 order
  const lastCard = await Card.findOne({ board: boardId, columnId })
    .sort({ order: -1 });
  const newOrder = lastCard ? lastCard.order + 1 : 0;
  
  // 创建卡片
  const card = new Card({
    title,
    description,
    board: boardId,
    columnId,
    assignee,
    labels: labels || [],
    priority: priority || 'medium',
    dueDate: dueDate ? new Date(dueDate) : undefined,
    order: newOrder,
  });
  
  await card.save();
  await card.populate('assignee', 'username avatar');
  
  // 广播给房间内的其他用户
  wsService.broadcastToBoard(boardId, WebSocketEvent.CardCreated, card, userId);
  
  // 更新 Board 的列中的 cardIds
  await Board.findByIdAndUpdate(boardId, {
    $push: { 'columns.$[col].cardIds': card._id }
  }, {
    arrayFilters: [{ 'col._id': columnId }]
  });
  
  return res.status(201).json({
    success: true,
    message: '卡片创建成功',
    data: card,
  });
});

/**
 * 更新卡片
 * PUT /api/v1/cards/:cardId
 */
export const updateCard = asyncErr(async (
  req: Request,
  res: Response
) => {
  const { cardId } = req.params;
  const userId = (req as any).user?.userId;
  const updates = req.body;
  
  const card = await Card.findById(cardId);
  if (!card) {
    throw new ApiError('卡片不存在', 404);
  }
  
  // 更新字段（只更新提供的字段）
  const allowedFields = ['title', 'description', 'assignee', 'labels', 'priority', 'dueDate'];
  allowedFields.forEach(field => {
    if (updates[field] !== undefined) {
      if (field === 'dueDate' && updates[field]) {
        (card as any)[field] = new Date(updates[field]);
      } else {
        (card as any)[field] = updates[field];
      }
    }
  });
  
  await card.save();
  await card.populate('assignee', 'username avatar');
  
  // 广播更新
  wsService.broadcastToBoard(card.board.toString(), WebSocketEvent.CardUpdated, card, userId);
  
  return res.json({
    success: true,
    message: '卡片更新成功',
    data: card,
  });
});

/**
 * 移动卡片（拖拽）
 * PUT /api/v1/cards/:cardId/move
 */
export const moveCard = asyncErr(async (
  req: Request,
  res: Response
) => {
  const { cardId } = req.params;
  const userId = (req as any).user?.userId;
  const { sourceColumnId, targetColumnId, newOrder } = req.body;
  
  const card = await Card.findById(cardId);
  if (!card) {
    throw new ApiError('卡片不存在', 404);
  }
  
  const boardId = card.board.toString();
  
  // 如果是同列内移动
  if (sourceColumnId === targetColumnId) {
    // 更新同列卡片的 order
    await Card.updateMany(
      { 
        board: boardId, 
        columnId: sourceColumnId,
        _id: { $ne: cardId }
      },
      [
        // 使用聚合管道更新
        { $set: { order: { $cond: [{ $gt: ['$order', card.order] }, { $subtract: ['$order', 1] }, '$order'] }}}
      ]
    );
    
    // 直接更新
    await Card.updateOne({ _id: cardId }, { order: newOrder });
  } else {
    // 跨列移动
    
    // 1. 从源列移除
    await Card.updateMany(
      { board: boardId, columnId: sourceColumnId, order: { $gt: card.order } },
      { $inc: { order: -1 } }
    );
    
    // 2. 在目标列腾出位置
    await Card.updateMany(
      { board: boardId, columnId: targetColumnId, order: { $gte: newOrder } },
      { $inc: { order: 1 } }
    );
    
    // 3. 更新卡片
    card.columnId = targetColumnId;
    card.order = newOrder;
    await card.save();
  }
  
  // 广播移动事件
  wsService.broadcastToBoard(boardId, WebSocketEvent.CardMoved, {
    cardId,
    sourceColumnId,
    targetColumnId,
    newOrder,
  }, userId);
  
  return res.json({
    success: true,
    message: '卡片移动成功',
  });
});

    throw new ApiError('卡片不存在', 404);
  }
  
  const boardId = card.board.toString();
  const columnId = card.columnId;
  
  // 从 Board 中移除
  await Board.findByIdAndUpdate(boardId, {
    $pull: { 'columns.$[col].cardIds': cardId }
  }, {
    arrayFilters: [{ 'col._id': columnId }]
  });
  
  // 删除卡片
  await Card.deleteOne({ _id: cardId });
  
  // 重新排序
  await Card.updateMany(
    { board: boardId, columnId },
    { $inc: { order: -1 } }
  );
  
  // 广播删除事件
  wsService.broadcastToBoard(boardId, WebSocketEvent.CardDeleted, { cardId }, userId);
  
  return res.json({
    success: true,
    message: '卡片删除成功',
  });
});

/**
 * 看板状态管理上下文
 * 使用 Context + Reducer 管理全局状态
 */

import React, { createContext, useContext, useReducer, useEffect, useCallback } from 'react';
import { Board, Card, BoardState, BoardAction } from '../types';
import { useWebSocket } from '../hooks/useWebSocket';

// 初始状态
const initialState: BoardState = {
  board: null,
  cards: {},
  isLoading: true,
  error: null,
};

// Reducer 函数
function boardReducer(state: BoardState, action: BoardAction): BoardState {
  switch (action.type) {
    case 'SET_BOARD':
      return { ...state, board: action.payload };
    
    case 'SET_CARDS': {
      const cards: Record<string, Card> = {};
      action.payload.forEach(card => {
        cards[card.id] = card;
      });
      return { ...state, cards };
    }
    
    case 'ADD_CARD': {
      const card = action.payload;
      return {
        ...state,
        cards: { ...state.cards, [card.id]: card },
        board: state.board ? {
          ...state.board,
          columns: state.board.columns.map(col => 
            col.id === card.columnId
              ? { ...col, cardIds: [...col.cardIds, card.id] }
              : col
          ),
        } : null,
      };
    }
    
    case 'UPDATE_CARD': {
      const card = action.payload;
      return {
        ...state,
        cards: { ...state.cards, [card.id]: card },
      };
    }
    
    case 'DELETE_CARD': {
      const cardId = action.payload;
      const { [cardId]: _, ...remainingCards } = state.cards;
      return {
        ...state,
        cards: remainingCards,
        board: state.board ? {
          ...state.board,
          columns: state.board.columns.map(col => ({
            ...col,
            cardIds: col.cardIds.filter(id => id !== cardId),
          })),
        } : null,
      };
    }
    
    case 'MOVE_CARD': {
      const { cardId, sourceColumnId, targetColumnId, newOrder } = action.payload;
      const card = state.cards[cardId];
      if (!card) return state;
      
      // 更新卡片所在列
      const updatedCard = { ...card, columnId: targetColumnId, order: newOrder };
      
      return {
        ...state,
        cards: { ...state.cards, [cardId]: updatedCard },
        board: state.board ? {
          ...state.board,
          columns: state.board.columns.map(col => {
            if (col.id === sourceColumnId && sourceColumnId === targetColumnId) {
              // 同列移动
              const newCardIds = col.cardIds.filter(id => id !== cardId);
              newCardIds.splice(newOrder, 0, cardId);
              return { ...col, cardIds: newCardIds };
            } else if (col.id === sourceColumnId) {
              // 从源列移除
              return { ...col, cardIds: col.cardIds.filter(id => id !== cardId) };
            } else if (col.id === targetColumnId) {
              // 添加到目标列
              const newCardIds = [...col.cardIds];
              newCardIds.splice(newOrder, 0, cardId);
              return { ...col, cardIds: newCardIds };
            }
            return col;
          }),
        } : null,
      };
    }
    
    case 'ADD_COMMENT': {
      const { cardId, comment } = action.payload;
      const card = state.cards[cardId];
      if (!card) return state;
      
      return {
        ...state,
        cards: {
          ...state.cards,
          [cardId]: { ...card, comments: [...card.comments, comment] },
        },
      };
    }
    
    case 'SET_LOADING':
      return { ...state, isLoading: action.payload };
    
    case 'SET_ERROR':
      return { ...state, error: action.payload };
    
    default:
      return state;
  }
}

// Context 类型
interface BoardContextType {
  state: BoardState;
  dispatch: React.Dispatch<BoardAction>;
  loadBoard: (boardId: string) => Promise<void>;
  createCard: (columnId: string, title: string) => Promise<void>;
  updateCard: (cardId: string, updates: Partial<Card>) => Promise<void>;
  moveCard: (cardId: string, sourceColumnId: string, targetColumnId: string, newOrder: number) => void;
  deleteCard: (cardId: string) => Promise<void>;
}

const BoardContext = createContext<BoardContextType | undefined>(undefined);

/**
 * 看板 Provider

export const BoardProvider: React.FC<{ children: React.ReactNode; boardId: string }> = ({ 
  children, 
  boardId 
}) => {
  const [state, dispatch] = useReducer(boardReducer, initialState);
  
  // WebSocket hook
  const { sendMessage } = useWebSocket({
    boardId,
    onCardCreated: (card) => dispatch({ type: 'ADD_CARD', payload: card }),
    onCardUpdated: (card) => dispatch({ type: 'UPDATE_CARD', payload: card }),
    onCardDeleted: ({ cardId }) => dispatch({ type: 'DELETE_CARD', payload: cardId }),
    onCardMoved: (data) => dispatch({ type: 'MOVE_CARD', payload: data }),
    onCommentAdded: (data) => dispatch({ type: 'ADD_COMMENT', payload: data }),
  });
  
  /**
   * 加载看板数据
   */
  const loadBoard = useCallback(async (id: string) => {
    try {
      dispatch({ type: 'SET_LOADING', payload: true });
      
      // 获取看板信息
      const boardRes = await fetch(`/api/v1/boards/${id}`);
      const boardData = await boardRes.json();
      if (!boardData.success) throw new Error(boardData.message);
      
      dispatch({ type: 'SET_BOARD', payload: boardData.data });
      
      // 获取所有卡片
      const cardsRes = await fetch(`/api/v1/boards/${id}/cards`);
      const cardsData = await cardsRes.json();
      if (!cardsData.success) throw new Error(cardsData.message);
      
      dispatch({ type: 'SET_CARDS', payload: cardsData.data });
      
    } catch (error) {
      dispatch({ type: 'SET_ERROR', payload: error instanceof Error ? error.message : '加载失败' });
    } finally {
      dispatch({ type: 'SET_LOADING', payload: false });
    }
  }, []);
  
  // 初始加载
  useEffect(() => {
    if (boardId) {
      loadBoard(boardId);
    }
  }, [boardId, loadBoard]);
  
  /**
   * 创建卡片
   */
  const createCard = useCallback(async (columnId: string, title: string) => {
    const res = await fetch(`/api/v1/boards/${boardId}/cards`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ title, columnId }),
    });
    
    const data = await res.json();
    if (data.success) {
      dispatch({ type: 'ADD_CARD', payload: data.data });
    }
  }, [boardId]);
  
  /**
   * 更新卡片
   */
  const updateCard = useCallback(async (cardId: string, updates: Partial<Card>) => {
    const res = await fetch(`/api/v1/cards/${cardId}`, {
      method: 'PUT',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(updates),
    });
    
    const data = await res.json();
    if (data.success) {
      dispatch({ type: 'UPDATE_CARD', payload: data.data });
    }
  }, []);
  
  /**
   * 移动卡片（乐观更新 + 服务器同步）
   */
  const moveCard = useCallback((cardId: string, sourceColumnId: string, targetColumnId: string, newOrder: number) => {
    // 乐观更新（立即更新 UI）
    dispatch({ type: 'MOVE_CARD', payload: { cardId, sourceColumnId, targetColumnId, newOrder } });
    
    // 发送 WebSocket 消息
    sendMessage('card:move', { cardId, sourceColumnId, targetColumnId, newOrder });
    
    // 同步到服务器
    fetch(`/api/v1/cards/${cardId}/move`, {
      method: 'PUT',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ sourceColumnId, targetColumnId, newOrder }),
    }).catch(console.error);
  }, [sendMessage]);
  
  /**
   * 删除卡片
   */
  const deleteCard = useCallback(async (cardId: string) => {
    const res = await fetch(`/api/v1/cards/${cardId}`, {
      method: 'DELETE',
    });
    
    const data = await res.json();
    if (data.success) {
      dispatch({ type: 'DELETE_CARD', payload: cardId });
    }
  }, []);
  
  return (
    <BoardContext.Provider value={{
      state,
      dispatch,
      loadBoard,
      createCard,
      updateCard,
      moveCard,
      deleteCard,
    }}>
      {children}
    </BoardContext.Provider>
  );
};

  if (!context) {
    throw new Error('useBoard 必须在 BoardProvider 内部使用');
  }
  return context;
};

/**
 * WebSocket Hook
 * 管理 WebSocket 连接和消息处理
 */

import { useEffect, useRef, useCallback, useState } from 'react';
import { Card, Comment } from '../types';

interface WebSocketHandlers {
  onCardCreated?: (card: Card) => void;
  onCardUpdated?: (card: Card) => void;
  onCardDeleted?: (data: { cardId: string }) => void;
  onCardMoved?: (data: { cardId: string; sourceColumnId: string; targetColumnId: string; newOrder: number }) => void;
  onCommentAdded?: (data: { cardId: string; comment: Comment }) => void;
}

interface UseWebSocketReturn {
  isConnected: boolean;
  sendMessage: (event: string, payload: any) => void;
}

export const useWebSocket = (boardId: string, handlers: WebSocketHandlers): UseWebSocketReturn => {
  const [isConnected, setIsConnected] = useState(false);
  const socketRef = useRef<WebSocket | null>(null);
  const handlersRef = useRef(handlers);
  
  // 更新 handlers ref
  useEffect(() => {
    handlersRef.current = handlers;
  }, [handlers]);
  
  useEffect(() => {
    const token = localStorage.getItem('token');
    if (!token) return;
    
    const wsUrl = process.env.REACT_APP_WS_URL || 'ws://localhost:3001';
    const socket = new WebSocket(`${wsUrl}?token=${token}`);
    
    socketRef.current = socket;
    
    socket.onopen = () => {
      console.log('🔌 WebSocket 已连接');
      setIsConnected(true);
      
      // 加入看板房间
      socket.send(JSON.stringify({
        type: 'join',
        room: `board:${boardId}`,
      }));
    };
    
    socket.onmessage = (event) => {
      try {
        const message = JSON.parse(event.data);
        const { event: eventType, payload } = message;
        
        // 根据事件类型调用对应处理函数
        switch (eventType) {
          case 'card:created':
            handlersRef.current.onCardCreated?.(payload);
            break;
          case 'card:updated':
            handlersRef.current.onCardUpdated?.(payload);
            break;
          case 'card:deleted':
            handlersRef.current.onCardDeleted?.(payload);
            break;
          case 'card:moved':
            handlersRef.current.onCardMoved?.(payload);
            break;
          case 'comment:added':
            handlersRef.current.onCommentAdded?.(payload);
            break;
        }
      } catch (error) {
        console.error('解析 WebSocket 消息失败:', error);
      }
    };
    
    socket.onclose = () => {
      console.log('🔌 WebSocket 已断开');
      setIsConnected(false);
    };
    
    socket.onerror = (error) => {
      console.error('WebSocket 错误:', error);
    };
    
    return () => {
      // 离开房间
      socket.send(JSON.stringify({
        type: 'leave',
        room: `board:${boardId}`,
      }));
      socket.close();
    };
  }, [boardId]);
  
  /**
   * 发送消息
   */
  const sendMessage = useCallback((event: string, payload: any) => {
    if (socketRef.current?.readyState === WebSocket.OPEN) {
      socketRef.current.send(JSON.stringify({
        type: 'event',
        room: `board:${boardId}`,
        event,
        payload,
      }));
    }
  }, [boardId]);
  
  return {
    isConnected,
    sendMessage,
  };
};

export default useWebSocket;

  const priorityColors = {
    low: '#61bd4f',
    medium: '#f2d600',
    high: '#ff9f1a',
    urgent: '#eb5a46',
  };
  
  // 拖拽配置
  const [{ isDragging }, drag, preview] = useDrag({
    type: ItemTypes.CARD,
    item: () => ({
      id: card.id,
      columnId: card.columnId,
      index,
    }),
    collect: (monitor) => ({
      isDragging: monitor.isDragging(),
    }),
  });
  
  // 放置配置
  const [{ isOver }, drop] = useDrop({
    accept: ItemTypes.CARD,
    hover: (item: { id: string; columnId: string; index: number }, monitor) => {
      if (!monitor.isOver({ shallow: true })) return;
      
      // 如果是同一列且位置变化，则交换
      if (item.columnId === card.columnId && item.index !== index) {
        // 简单处理：同列移动不做什么，等 drop 时处理
      }
    },
    drop: (item: { id: string; columnId: string; index: number }) => {
      if (item.id !== card.id) {
        onDrop(item.id, item.columnId, card.columnId, index);
      }
    },
    collect: (monitor) => ({
      isOver: monitor.isOver(),
    }),
  });
  
  // 格式化日期
  const formatDueDate = (dateStr?: string): string => {
    if (!dateStr) return '';
    const date = new Date(dateStr);
    const today = new Date();
    const tomorrow = new Date(today);
    tomorrow.setDate(tomorrow.getDate() + 1);
    
    if (date.toDateString() === today.toDateString()) {
      return '今天';
    }
    if (date.toDateString() === tomorrow.toDateString()) {
      return '明天';
    }
    return date.toLocaleDateString('zh-CN', { month: 'short', day: 'numeric' });
  };
  
  // 检查是否逾期
  const isOverdue = (): boolean => {
    if (!card.dueDate) return false;
    return new Date() > new Date(card.dueDate);
  };
  
  return (
    <div
      ref={(node) => preview(drop(node))}
      className={`card ${isDragging ? 'card--dragging' : ''} ${isOver ? 'card--over' : ''}`}
      onClick={() => setIsExpanded(true)}
    >
      {/* 优先级指示条 */}
      <div 
        className="card__priority"
        style={{ backgroundColor: priorityColors[card.priority] }}
      />
      
      {/* 标签 */}
      {card.labels && card.labels.length > 0 && (
        <div className="card__labels">
          {card.labels.slice(0, 3).map((label) => (
            <span
              key={label.id}
              className="card__label"
              style={{ backgroundColor: label.color }}
            >
              {label.name}
            </span>
          ))}
        </div>
      )}
      
      {/* 标题 */}
      <h4 className="card__title">{card.title}</h4>
      
      {/* 元信息 */}
      <div className="card__meta">
        {/* 截止日期 */}
        {card.dueDate && (
          <span className={`card__due ${isOverdue() ? 'card__due--overdue' : ''}`}>
            📅 {formatDueDate(card.dueDate)}
          </span>
        )}
        
        {/* 负责人 */}
        {card.assignee && (
          <img
            className="card__assignee"
            src={card.assignee.avatar || '/default-avatar.png'}
            alt={card.assignee.username}
            title={card.assignee.username}
          />
        )}
        
        {/* 评论数 */}
        {card.comments && card.comments.length > 0 && (
          <span className="card__comments">
            💬 {card.comments.length}
          </span>
        )}
      </div>
      
      {/* 拖拽手柄 */}
      <div ref={drag} className="card__drag-handle">
        ⋮⋮
      </div>
    </div>
  );
};

export default Card;
```

  | { type: 'UPDATE'; payload: Item };

// Reducer 函数
function reducer(state: State, action: Action): State {
  switch (action.type) {
    case 'ADD':
      return [...state, action.payload];
    // ...
  }
}

function Provider({ children }) {
  const [state, dispatch] = useReducer(reducer, initialState);

## 面试要点

### 问题 1: React DnD 是如何工作的？

**参考答案**:
```
React DnD 使用 HTML5 Drag and Drop API 和 Touch Backend

核心概念：
1. DndProvider: 全局 Provider，提供 DnD 功能
2. useDrag: 定义拖拽源
   - type: 拖拽项目类型
   - item: 拖拽时传递的数据
   - collect: 收集状态
3. useDrop: 定义放置目标
   - accept: 接受的项目类型
   - drop: 放置处理
   - hover: 悬停处理
   - collect: 收集状态

拖拽流程：
1. 开始拖拽 → 收集 isDragging
2. 悬停目标 → 收集 isOver
3. 放置 → 调用 drop handler
4. 结束 → 重置状态
```

### 问题 2: WebSocket 和 HTTP 轮询的区别？

**参考答案**:
```
HTTP 轮询：
- 客户端定时发送请求
- 浪费带宽，延迟高
- 服务器压力大

WebSocket：
- 双向通信，连接持久
- 实时性高，带宽低
- 需要处理断线重连

适用场景：
- 实时聊天 → WebSocket
- 新闻推送 → SSE
- 数据刷新 → HTTP/2 Server Push
- 低频更新 → 轮询
```

乐观更新：先更新 UI，再同步服务器

注意事项：

3. 唯一标识：确保操作的幂等性
4. 错误提示：友好的错误反馈

实现方式：
- 使用 Reducer 记录操作
- 保存操作前的快照
- 实现撤销/重做功能
```
