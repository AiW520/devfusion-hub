# 方案六：T3 Stack 社交笔记应用 ⭐重点推荐

## 项目介绍

### 项目是什么
这是一个现代化的社交笔记应用，结合了 Notion 的编辑体验和 Twitter 的社交功能。支持 Markdown 编辑、实时协作、关注系统、点赞评论等功能。

### T3 Stack 是什么？
**T3 Stack** 是由 Theo (t3dotgg) 创建的现代全栈开发技术栈：

```
T3 Stack = Next.js + tRPC + TypeScript + Tailwind CSS + Prisma + NextAuth

- Next.js: React 框架（App Router）
- tRPC: 端到端类型安全的 API（无需 REST）
- TypeScript: 类型系统
- Tailwind CSS: 样式
- Prisma: 数据库 ORM
- NextAuth: 身份认证
```

### 为什么选 T3 Stack？

1. **端到端类型安全**：前后端共享类型，修改一处，全局生效
2. **开发体验极佳**：无需编写 API 文档，IDE 自动补全
3. **面试加分项**：展示对现代全栈开发的理解
4. **社区活跃**：有官方教程和大量示例

### 3. tRPC 核心配置

**src/server/api/trpc.ts**
```typescript
/**
 * tRPC 服务端配置
 * 
 * tRPC 的核心概念：
 * 1. Router: 定义 API 端点
 * 2. Procedure: 可调用远程函数
 * 3. Context: 请求上下文（session、数据库等）
 * 4. Middleware: 拦截器
 */

import { initTRPC, TRPCError } from 'trpc';
import { type CreateNextContextOptions } from '@trpc/server/adapters/next';
import { getServerSession } from 'next-auth';
import superjson from 'superjson';
import { ZodError } from 'zod';
import { authOptions } from '../auth';
import { prisma } from '../db';

/**
 * 创建请求上下文
 * 
 * 每个请求都会创建新的上下文
 * 包含 session、数据库等共享资源
 */
export const createTRPCContext = async (opts: CreateNextContextOptions) => {
  const { req, res } = opts;
  
  // 获取 session
  const session = await getServerSession(req, res, authOptions);
  
  return {
    prisma,
    session,
    req,
    res,
  };
};

/**
 * tRPC 初始化
 * 
 * tRPC 使用泛型来确保类型安全
 */
const t = initTRPC.context<typeof createTRPCContext>().create({
  /**
   * transformer: 序列化和反序列化数据
   * superjson 支持 Date、Map、Set 等特殊类型
   */
  transformer: superjson,
  
  /**
   * 错误格式化
   */
  errorFormatter({ shape, error }) {
    return {
      ...shape,
      data: {
        ...shape.data,
        zodError:
          error.cause instanceof ZodError ? error.cause.flatten() : null,
      },
    };
  },
});

/**
 * 创建可复用的 tRPC 实例
 * 包含中间件和工具
 */
export const createTRPCRouter = t.router;
export const publicProcedure = t.procedure;

/**
 * 认证中间件
 * 
 * 中间件是 tRPC 的核心概念
 * 用于在请求前进行权限检查
 */
const enforceUserIsAuthed = t.middleware(({ ctx, next }) => {
  if (!ctx.session?.user) {
    throw new TRPCError({ 
      code: 'UNAUTHORIZED',
      message: '请先登录',
    });
  }
  
  return next({
    ctx: {
      // 扩展上下文，添加用户信息
      session: { ...ctx.session, user: ctx.session.user },
    },
  });
});

/**
 * 需要认证的 Procedure
 */
export const protectedProcedure = t.procedure.use(enforceUserIsAuthed);
```

### 4. tRPC 路由实现

**src/server/api/routers/note.ts**
```typescript
/**
 * 笔记路由
 * 
 * 展示 tRPC 的核心用法
 */

import { z } from 'zod';
import {
  createTRPCRouter,
  publicProcedure,
  protectedProcedure,
} from '../trpc';
import { TRPCError } from '@trpc/server';

export const noteRouter = createTRPCRouter({
  /**
   * 获取笔记列表
   * 
   * publicProcedure: 公开访问
   * 可以链式调用 .query() 来定义查询
   */
  getAll: publicProcedure
    .input(
      z.object({
        limit: z.number().min(1).max(100).default(10),
        cursor: z.string().nullish(),  // 用于分页的游标
        tag: z.string().optional(),
      })
    )
    .query(async ({ ctx, input }) => {
      const { limit, cursor, tag } = input;
      
      const where: any = {
        isPublished: true,
      };
      
      if (tag) {
        where.tags = { has: tag };
      }
      
      const notes = await ctx.prisma.note.findMany({
        take: limit + 1,  // 多取一条判断是否有更多
        cursor: cursor ? { id: cursor } : undefined,
        where,
        orderBy: { createdAt: 'desc' },
        include: {
          author: {
            select: { id: true, name: true, image: true },
          },
          _count: {
            select: { likes: true, comments: true },
          },
        },
      });
      
      // 判断是否有下一页
      let nextCursor: typeof cursor | undefined = undefined;
      if (notes.length > limit) {
        const nextItem = notes.pop();
        nextCursor = nextItem!.id;
      }
      
      return {
        notes,
        nextCursor,
      };
    }),

  /**
   * 获取单个笔记
   */
  getById: publicProcedure
    .input(z.object({ id: z.string() }))
    .query(async ({ ctx, input }) => {
      const note = await ctx.prisma.note.findUnique({
        where: { id: input.id },
        include: {
          author: {
            select: { 
              id: true, 
              name: true, 
              image: true,
              bio: true,
              followers: { take: 5, select: { followerId: true } },
            },
          },
          comments: {
            include: {
              author: {
                select: { id: true, name: true, image: true },
              },
            },
            orderBy: { createdAt: 'desc' },
          },
          _count: {
            select: { likes: true },
          },
        },
      });
      
      if (!note) {
        throw new TRPCError({ 
          code: 'NOT_FOUND',
          message: '笔记不存在',
        });
      }
      
      // 检查访问权限
      const isOwner = ctx.session?.user?.email === note.authorId;
      
      if (!note.isPublished && !isOwner) {
        throw new TRPCError({ 
          code: 'FORBIDDEN',
          message: '无权访问此笔记',
        });
      }
      
      return note;
    }),

  /**
   * 获取用户的笔记
   */
  getByUsername: publicProcedure
    .input(z.object({ 
      username: z.string(),
      includeUnpublished: z.boolean().default(false),
    }))
    .query(async ({ ctx, input }) => {
      const user = await ctx.prisma.user.findUnique({
        where: { name: input.username },
      });
      
      if (!user) {
        throw new TRPCError({ 
          code: 'NOT_FOUND',
          message: '用户不存在',
        });
      }
      
      const where: any = {
        authorId: user.id,
      };
      
      // 如果不是作者本人，只显示已发布的
      const isOwner = ctx.session?.user?.id === user.id;
      if (!isOwner || !input.includeUnpublished) {
        where.isPublished = true;
      }
      
      const notes = await ctx.prisma.note.findMany({
        where,
        orderBy: { createdAt: 'desc' },
        include: {
          author: {
            select: { id: true, name: true, image: true },
          },
          _count: {
            select: { likes: true, comments: true },
          },
        },
      });
      
      return notes;
    }),

  /**
   * 创建笔记
   * 
   * protectedProcedure: 需要登录
   */
  create: protectedProcedure
    .input(
      z.object({
        title: z.string().min(1).max(200),
        content: z.string(),
        tags: z.array(z.string()).optional(),
        isPublished: z.boolean().default(false),
      })
    )
    .mutation(async ({ ctx, input }) => {
      const userId = ctx.session.user.id;
      
      // 生成摘要
      const excerpt = input.content
        .replace(/[#*`]/g, '')
        .substring(0, 200);
      
      const note = await ctx.prisma.note.create({
        data: {
          title: input.title,
          content: input.content,
          excerpt,
          tags: input.tags || [],
          isPublished: input.isPublished,
          publishedAt: input.isPublished ? new Date() : null,
          authorId: userId,
        },
      });
      
      return note;
    }),

  /**
   * 更新笔记
   */
  update: protectedProcedure
    .input(
      z.object({
        id: z.string(),
        title: z.string().min(1).max(200).optional(),
        content: z.string().optional(),
        tags: z.array(z.string()).optional(),
        isPublished: z.boolean().optional(),
      })
    )
    .mutation(async ({ ctx, input }) => {
      const { id, ...data } = input;
      
      // 检查所有权
      const note = await ctx.prisma.note.findUnique({ where: { id } });
      
      if (!note) {
        throw new TRPCError({ 
          code: 'NOT_FOUND',
          message: '笔记不存在',
        });
      }
      
      if (note.authorId !== ctx.session.user.id) {
        throw new TRPCError({ 
          code: 'FORBIDDEN',
          message: '无权修改此笔记',
        });
      }
      
      // 如果从草稿改为发布，设置发布时间
      if (data.isPublished === true && !note.isPublished) {
        data.isPublished = true;
      }
      
      const excerpt = data.content
        ? data.content.replace(/[#*`]/g, '').substring(0, 200)
        : note.excerpt || undefined;
      
      const updated = await ctx.prisma.note.update({
        where: { id },
        data: { ...data, excerpt },
      });
      
      return updated;
    }),

  /**
   * 删除笔记
   */
  delete: protectedProcedure
    .input(z.object({ id: z.string() }))
    .mutation(async ({ ctx, input }) => {
      const note = await ctx.prisma.note.findUnique({
        where: { id: input.id },
      });
      
      if (!note) {
        throw new TRPCError({ 
          code: 'NOT_FOUND',
          message: '笔记不存在',
        });
      }
      
      if (note.authorId !== ctx.session.user.id) {
        throw new TRPCError({ 
          code: 'FORBIDDEN',
          message: '无权删除此笔记',
        });
      }
      
      await ctx.prisma.note.delete({ where: { id: input.id } });
      
      return { success: true };
    }),

  /**
   * 点赞/取消点赞
   */
  toggleLike: protectedProcedure
    .input(z.object({ noteId: z.string() }))
    .mutation(async ({ ctx, input }) => {
      const userId = ctx.session.user.id;
      
      // 检查是否已点赞
      const existingLike = await ctx.prisma.like.findUnique({
        where: {
          noteId_userId: {
            noteId: input.noteId,
            userId,
          },
        },
      });
      
      if (existingLike) {
        // 取消点赞
        await ctx.prisma.$transaction([
          ctx.prisma.like.delete({
            where: { id: existingLike.id },
          }),
          ctx.prisma.note.update({
            where: { id: input.noteId },
            data: { likeCount: { decrement: 1 } },
          }),
        ]);
        
        return { liked: false };
      } else {
        // 添加点赞
        await ctx.prisma.$transaction([
          ctx.prisma.like.create({
            data: { noteId: input.noteId, userId },
          }),
          ctx.prisma.note.update({
            where: { id: input.noteId },
            data: { likeCount: { increment: 1 } },
          }),
        ]);
        
        return { liked: true };
      }
    }),

  /**
   * 添加评论
   */
  addComment: protectedProcedure
    .input(
      z.object({
        noteId: z.string(),
        content: z.string().min(1).max(2000),
      })
    )
    .mutation(async ({ ctx, input }) => {
      const userId = ctx.session.user.id;
      
      // 检查笔记存在
      const note = await ctx.prisma.note.findUnique({
        where: { id: input.noteId },
      });
      
      if (!note) {
        throw new TRPCError({ 
          code: 'NOT_FOUND',
          message: '笔记不存在',
        });
      }
      
      const comment = await ctx.prisma.comment.create({
        data: {
          noteId: input.noteId,
          authorId: userId,
          content: input.content,
        },
        include: {
          author: {
            select: { id: true, name: true, image: true },
          },
        },
      });
      
      // 更新评论数
      await ctx.prisma.note.update({
        where: { id: input.noteId },
        data: { commentCount: { increment: 1 } },
      });
      
      return comment;
    }),
});
```

```typescript
/**
 * 用户路由
 */

import { z } from 'zod';
import {
  createTRPCRouter,
  publicProcedure,
  protectedProcedure,
} from '../trpc';
import { TRPCError } from '@trpc/server';

export const userRouter = createTRPCRouter({
  /**
   * 获取用户信息
   */
  getByUsername: publicProcedure
    .input(z.object({ username: z.string() }))
    .query(async ({ ctx, input }) => {
      const user = await ctx.prisma.user.findUnique({
        where: { name: input.username },
        select: {
          id: true,
          name: true,
          image: true,
          bio: true,
          _count: {
            select: {
              followers: true,
              following: true,
              notes: { where: { isPublished: true } },
            },
          },
        },
      });
      
      if (!user) {
        throw new TRPCError({ 
          code: 'NOT_FOUND',
          message: '用户不存在',
        });
      }
      
      // 检查是否关注
      const isFollowing = ctx.session?.user?.id
        ? await ctx.prisma.follow.findUnique({
            where: {
              followerId_followingId: {
                followerId: ctx.session.user.id,
                followingId: user.id,
              },
            },
          })
        : false;
      
      return { ...user, isFollowing: !!isFollowing };
    }),

  /**
   * 关注/取消关注
   */
  toggleFollow: protectedProcedure
    .input(z.object({ userId: z.string() }))
    .mutation(async ({ ctx, input }) => {
      const myId = ctx.session.user.id;
      
      if (myId === input.userId) {
        throw new TRPCError({
          code: 'BAD_REQUEST',
          message: '不能关注自己',
        });
      }
      
      const existing = await ctx.prisma.follow.findUnique({
        where: {
          followerId_followingId: {
            followerId: myId,
            followingId: input.userId,
          },
        },
      });
      
      if (existing) {
        await ctx.prisma.follow.delete({ where: { id: existing.id } });
        return { following: false };
      } else {
        await ctx.prisma.follow.create({
          data: {
            followerId: myId,
            followingId: input.userId,
          },
        });
        return { following: true };
      }
    }),

  /**
   * 获取时间线（关注的人的笔记）
   */
  getTimeline: protectedProcedure
    .input(
      z.object({
        limit: z.number().min(1).max(50).default(10),
        cursor: z.string().nullish(),
      })
    )
    .query(async ({ ctx, input }) => {
      const myId = ctx.session.user.id;
      
      // 获取关注的人的笔记
      const notes = await ctx.prisma.note.findMany({
        take: input.limit + 1,
        cursor: input.cursor ? { id: input.cursor } : undefined,
        where: {
          isPublished: true,
          OR: [
            // 我关注的
            { authorId: { in: await ctx.prisma.follow.findMany({
              where: { followerId: myId },
              select: { followingId: true },
            }).then(f => f.map(x => x.followingId)) } },
            // 我的
            { authorId: myId },
          ],
        },
        orderBy: { createdAt: 'desc' },
        include: {
          author: {
            select: { id: true, name: true, image: true },
          },
          _count: {
            select: { likes: true, comments: true },
          },
        },
      });
      
      let nextCursor: typeof input.cursor | undefined = undefined;
      if (notes.length > input.limit) {
        const nextItem = notes.pop();
        nextCursor = nextItem!.id;
      }
      
      return { notes, nextCursor };
    }),
});
```

```typescript
'use client';

import { useState } from 'react';
import { useRouter } from 'next/router';
import { trpc } from '@/utils/trpc';
import { useSession } from 'next-auth/react';

export function NoteEditor({ initialData }: { initialData?: any }) {
  const router = useRouter();
  const { data: session } = useSession();
  
  const [title, setTitle] = useState(initialData?.title || '');
  const [content, setContent] = useState(initialData?.content || '');
  const [tags, setTags] = useState<string[]>(initialData?.tags || []);
  const [tagInput, setTagInput] = useState('');
  const [isPublished, setIsPublished] = useState(initialData?.isPublished || false);
  
  const createNote = trpc.note.create.useMutation({
    onSuccess: (data) => {
      router.push(`/note/${data.id}`);
    },
  });
  
  const updateNote = trpc.note.update.useMutation({
    onSuccess: () => {
      alert('保存成功！');
    },
  });

  const handleSubmit = (e: React.FormEvent) => {
    e.preventDefault();
    
    if (!session) {
      alert('请先登录');
      return;
    }
    
    if (initialData?.id) {
      updateNote.mutate({
        id: initialData.id,
        title,
        content,
        tags,
        isPublished,
      });
    } else {
      createNote.mutate({
        title,
        content,
        tags,
        isPublished,
      });
    }
  };

  const addTag = () => {
    const tag = tagInput.trim();
    if (tag && !tags.includes(tag) && tags.length < 5) {
      setTags([...tags, tag]);
      setTagInput('');
    }
  };

  return (
    <form onSubmit={handleSubmit} className="note-editor">
      <input
        type="text"
        placeholder="标题"
        value={title}
        onChange={(e) => setTitle(e.target.value)}
        className="note-editor__title"
        required
      />
      
      <textarea
        placeholder="用 Markdown 写下你的想法..."
        value={content}
        onChange={(e) => setContent(e.target.value)}
        className="note-editor__content"
        rows={20}
        required
      />
      
      <div className="note-editor__tags">
        {tags.map((tag) => (
          <span key={tag} className="tag">
            #{tag}
            <button
              type="button"
              onClick={() => setTags(tags.filter((t) => t !== tag))}
            >
              ×
            </button>
          </span>
        ))}
        {tags.length < 5 && (
          <input
            type="text"
            placeholder="添加标签"
            value={tagInput}
            onChange={(e) => setTagInput(e.target.value)}
            onKeyDown={(e) => e.key === 'Enter' && (e.preventDefault(), addTag())}
          />
        )}
      </div>
      
      <div className="note-editor__actions">
        <label className="note-editor__publish">
          <input
            type="checkbox"
            checked={isPublished}
            onChange={(e) => setIsPublished(e.target.checked)}
          />
          立即发布
        </label>
        
        <button type="submit" disabled={createNote.isLoading}>
          {createNote.isLoading ? '保存中...' : initialData ? '保存' : '创建'}
        </button>
      </div>
    </form>
  );
}
```

### 1. tRPC 端到端类型安全

```typescript
// 传统 REST API
// 后端定义
app.post('/api/notes', async (req, res) => {
  const { title, content } = req.body;  // 手动定义

// 前端调用
const response = await fetch('/api/notes', {
  method: 'POST',
  body: JSON.stringify({ title, content }),  // 手动写
});

// tRPC（类型自动共享）
// 后端定义
const noteRouter = createTRPCRouter({
  create: protectedProcedure
    .input(z.object({ title: z.string(), content: z.string() }))
    .mutation(async ({ ctx, input }) => {
      // input 的类型自动推断
      return ctx.prisma.note.create({ data: input });
    }),
});

// 前端调用 - 完全类型安全！
const note = await trpc.note.create.useMutation({
  // 直接使用 inferred types
  onSuccess: (data) => {
    console.log(data.title);  // IDE 自动补全！
  },
});
```

### 2. tRPC vs REST vs GraphQL

```
tRPC:
- 优点：类型安全、无需代码生成、开发体验好
- 缺点：只能前后端同构、不适合微服务
- 适用：全栈应用、Monorepo

REST:
- 优点：通用、简单、缓存友好
- 缺点：需要维护文档、类型不共享
- 适用：公开 API、微服务

GraphQL:
- 优点：灵活、一次请求多资源
- 缺点：复杂度高、过度查询
- 适用：移动端、多客户端
```

### 3. Procedure 类型详解

```typescript
// 公开查询 - 任何人都可以访问
publicProcedure.query()

// 受保护查询 - 需要登录
protectedProcedure.query()

// 公开变更 - 任何人可以调用
publicProcedure.mutation()

// 受保护变更 - 需要登录
protectedProcedure.mutation()
```
