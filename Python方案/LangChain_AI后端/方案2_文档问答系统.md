## 📋 项目介绍

### 1.1 项目概述
本项目是一个**文档问答系统**，用户可以上传 PDF/Word/Markdown 文档，系统自动解析并建立索引，然后用户可以针对文档内容提问，系统给出精准回答并标注引用来源。

### 1.2 核心技术亮点
- **多格式文档解析**：支持 PDF、Word、Markdown 等多种格式
- **ChromaDB 向量数据库**：轻量级、易部署的向量检索
- **流式输出**：实时显示 AI 思考过程
- **引用追踪**：精准标注答案来源，支持溯源
- **多文档对话**：支持跨文档联合问答

### 1.3 适用场景
- 企业内部知识库
- 法律/财务文档分析
- 论文阅读助手
- 客服知识库构建

### 2.4 文档解析服务 document_parser.py
```python
"""
文档解析服务 - 支持多种格式的文档解析

核心功能：
1. PDF 解析（支持目录、页码）
2. Word 解析
3. Markdown 解析
4. 文本提取和清洗
"""
from typing import List, Dict, Any, Optional
from pathlib import Path
import re
from abc import ABC, abstractmethod


class DocumentParser(ABC):
    """
    文档解析器基类
    
    使用策略模式：定义统一的解析接口，不同格式实现不同的解析逻辑
    """
    
    @abstractmethod
    def parse(self, file_path: str) -> Dict[str, Any]:
        """
        解析文档
        
        Args:
            file_path: 文件路径
            
        Returns:
            {
                "text": str,           # 完整文本
                "chunks": List[str],   # 分块后的文本
                "metadata": dict       # 元数据（页数、目录等）
            }
        """
        pass


class PDFParser(DocumentParser):
    """
    PDF 解析器
    
    使用 PyMuPDF (fitz) 库，支持：
    - 文本提取
    - 页码追踪
    - 目录解析
    """
    
    def __init__(self, chunk_size: int = 500, chunk_overlap: int = 50):
        self.chunk_size = chunk_size
        self.chunk_overlap = chunk_overlap
    
    def parse(self, file_path: str) -> Dict[str, Any]:
        """解析 PDF 文件"""
        import fitz  # PyMuPDF
        
        full_text = ""
        pages_info = []
        
        # 打开 PDF
        doc = fitz.open(file_path)
        
        for page_num, page in enumerate(doc, 1):
            # 提取文本
            text = page.get_text("text")
            
            if text.strip():
                full_text += f"\n\n[第{page_num}页]\n\n" + text
                pages_info.append({
                    "page": page_num,
                    "text_length": len(text),
                    "preview": text[:100]
                })
        
        doc.close()
        
        # 清理文本
        cleaned_text = self._clean_text(full_text)
        
        # 分块
        chunks = self._chunk_text(cleaned_text)
        
        return {
            "text": cleaned_text,
            "chunks": chunks,
            "metadata": {
                "total_pages": len(pages_info),
                "total_characters": len(cleaned_text),
                "pages_info": pages_info
            }
        }
    
    def _clean_text(self, text: str) -> str:
        """
        清理文本
        
        为什么要清理？
        - 移除多余空白（节省 token）
        - 修复常见格式问题
        - 提高后续处理质量
        """
        # 替换多个空白为单个
        text = re.sub(r'\s+', ' ', text)
        
        # 移除特殊控制字符
        text = re.sub(r'[\x00-\x08\x0b-\x0c\x0e-\x1f\x7f]', '', text)
        
        return text.strip()
    
    def _chunk_text(self, text: str) -> List[str]:
        """
        文本分块
        
        策略：按段落分块，保留语义完整性
        """
        # 先按段落分割
        paragraphs = text.split("\n\n")
        
        chunks = []
        current_chunk = ""
        
        for para in paragraphs:
            para = para.strip()
            if not para:
                continue
            
            # 如果当前块加上这个段落超过限制
            if len(current_chunk) + len(para) > self.chunk_size:
                # 保存当前块
                if current_chunk:
                    chunks.append(current_chunk.strip())
                
                # 如果单个段落就超过限制，按句子分割
                if len(para) > self.chunk_size:
                    sentences = self._split_sentences(para)
                    for sent in sentences:
                        if len(sent) > self.chunk_size:
                            # 继续按字数分割
                            for i in range(0, len(sent), self.chunk_size - self.chunk_overlap):
                                chunk = sent[i:i + self.chunk_size]
                                if chunk.strip():
                                    chunks.append(chunk)
                        else:
                            chunks.append(sent)
                    current_chunk = ""
                else:
                    current_chunk = para
            else:
                current_chunk += "\n\n" + para if current_chunk else para
        
        # 保存最后一个块
        if current_chunk.strip():
            chunks.append(current_chunk.strip())
        
        return chunks
    
    def _split_sentences(self, text: str) -> List[str]:
        """按句子分割"""
        # 中英文句子分隔符
        sentences = re.split(r'[。！？.!?]+', text)
        return [s.strip() for s in sentences if s.strip()]


class DocxParser(DocumentParser):
    """
    Word 文档解析器
    
    使用 python-docx 库
    """
    
    def __init__(self, chunk_size: int = 500, chunk_overlap: int = 50):
        self.chunk_size = chunk_size
        self.chunk_overlap = chunk_overlap
        self.pdf_parser = PDFParser(chunk_size, chunk_overlap)
    
    def parse(self, file_path: str) -> Dict[str, Any]:
        """解析 Word 文档"""
        from docx import Document
        
        doc = Document(file_path)
        paragraphs = []
        
        # 提取所有段落
        for para in doc.paragraphs:
            text = para.text.strip()
            if text:
                paragraphs.append(text)
        
        # 提取表格内容
        for table in doc.tables:
            for row in table.rows:
                for cell in row.cells:
                    text = cell.text.strip()
                    if text:
                        paragraphs.append(f"[表格] {text}")
        
        # 合并文本
        full_text = "\n\n".join(paragraphs)
        
        # 清理和分块
        cleaned_text = self.pdf_parser._clean_text(full_text)
        chunks = self.pdf_parser._chunk_text(cleaned_text)
        
        return {
            "text": cleaned_text,
            "chunks": chunks,
            "metadata": {
                "paragraph_count": len(paragraphs),
                "table_count": len(doc.tables),
                "total_characters": len(cleaned_text)
            }
        }


class MarkdownParser(DocumentParser):
    """
    Markdown 解析器
    
    保留 Markdown 格式，便于后续处理
    """
    
    def __init__(self, chunk_size: int = 500, chunk_overlap: int = 50):
        self.chunk_size = chunk_size
        self.chunk_overlap = chunk_overlap
    
    def parse(self, file_path: str) -> Dict[str, Any]:
        """解析 Markdown 文件"""
        with open(file_path, "r", encoding="utf-8") as f:
            content = f.read()
        
        # 清理但保留基本格式
        # 移除多余的空白行
        content = re.sub(r'\n{3,}', '\n\n', content)
        
        # 按标题分块（保留语义结构）
        chunks = self._chunk_by_headings(content)
        
        return {
            "text": content,
            "chunks": chunks,
            "metadata": {
                "format": "markdown",
                "total_characters": len(content)
            }
        }
    
    def _chunk_by_headings(self, content: str) -> List[str]:
        """
        按标题分块
        
        策略：如果两个标题之间内容小于限制，就合并
        否则单独成块
        """
        # 匹配 Markdown 标题
        heading_pattern = r'^#{1,6}\s+.+$'
        lines = content.split('\n')
        
        chunks = []
        current_chunk = ""
        current_heading = ""
        
        for line in lines:
            is_heading = bool(re.match(heading_pattern, line))
            
            if is_heading:
                # 遇到新标题
                # 先保存当前块（如果有内容）
                if current_chunk.strip():
                    chunks.append(current_chunk.strip())
                
                current_heading = line
                current_chunk = line
            else:
                current_chunk += "\n" + line
        
        # 保存最后一块
        if current_chunk.strip():
            chunks.append(current_chunk.strip())
        
        # 如果分块太多或太少，按普通方式处理
        if len(chunks) < 2:
            # 太少，使用默认分块
            return self._simple_chunk(content)
        elif len(chunks) > 50:
            # 太多，合并相邻的块
            return self._merge_chunks(chunks)
        
        return chunks
    
    def _simple_chunk(self, text: str) -> List[str]:
        """简单分块"""
        chunks = []
        for i in range(0, len(text), self.chunk_size - self.chunk_overlap):
            chunk = text[i:i + self.chunk_size]
            if chunk.strip():
                chunks.append(chunk.strip())
        return chunks
    
    def _merge_chunks(self, chunks: List[str]) -> List[str]:
        """合并相邻块"""
        merged = []
        current = ""
        
        for chunk in chunks:
            if len(current) + len(chunk) < self.chunk_size * 1.5:
                current += "\n\n" + chunk
            else:
                if current:
                    merged.append(current)
                current = chunk
        
        if current:
            merged.append(current)
        
        return merged


class TextParser(DocumentParser):
    """纯文本解析器"""
    
    def __init__(self, chunk_size: int = 500, chunk_overlap: int = 50):
        self.chunk_size = chunk_size
        self.chunk_overlap = chunk_overlap
    
    def parse(self, file_path: str) -> Dict[str, Any]:
        """解析纯文本文件"""
        with open(file_path, "r", encoding="utf-8") as f:
            content = f.read()
        
        # 清理文本
        content = re.sub(r'\s+', ' ', content)
        
        # 分块
        chunks = []
        for i in range(0, len(content), self.chunk_size - self.chunk_overlap):
            chunk = content[i:i + self.chunk_size]
            if chunk.strip():
                chunks.append(chunk.strip())
        
        return {
            "text": content,
            "chunks": chunks,
            "metadata": {"total_characters": len(content)}
        }


# 工厂函数：根据文件类型选择解析器
def get_parser(file_path: str, chunk_size: int = 500, chunk_overlap: int = 50) -> DocumentParser:
    """
    获取合适的解析器
    
    工厂模式：根据文件扩展名返回对应的解析器
    """
    suffix = Path(file_path).suffix.lower()
    
    parsers = {
        ".pdf": PDFParser,
        ".docx": DocxParser,
        ".doc": DocxParser,
        ".md": MarkdownParser,
        ".txt": TextParser,
    }
    
    parser_class = parsers.get(suffix)
    if not parser_class:
        raise ValueError(f"不支持的文件格式: {suffix}")
    
    return parser_class(chunk_size, chunk_overlap)


def parse_document(
    file_path: str,
    document_id: str,
    chunk_size: int = 500,
    chunk_overlap: int = 50
) -> Dict[str, Any]:
    """
    解析文档的统一入口
    
    Args:
        file_path: 文件路径
        document_id: 文档ID
        chunk_size: 块大小
        chunk_overlap: 块重叠
        
    Returns:
        {
            "document_id": str,
            "file_name": str,
            "chunks": [{"id": str, "content": str, "metadata": dict}],
            "metadata": dict
        }
    """
    # 选择解析器
    parser = get_parser(file_path, chunk_size, chunk_overlap)
    
    # 解析
    result = parser.parse(file_path)
    
    # 构建块列表
    chunks = []
    for i, chunk_text in enumerate(result["chunks"]):
        chunks.append({
            "id": f"{document_id}_chunk_{i}",
            "content": chunk_text,
            "metadata": {
                "chunk_index": i,
                "document_id": document_id,
                "char_count": len(chunk_text)
            }
        })
    
    return {
        "document_id": document_id,
        "file_name": Path(file_path).name,
        "file_path": file_path,
        "chunks": chunks,
        "metadata": {
            **result["metadata"],
            "chunk_count": len(chunks)
        }
    }
```

### 2.5 Embedding 服务 embedding_service.py
```python
"""
Embedding 服务 - 文本向量化

Embedding = 将文本转为数字向量
相似语义的文本会有相似的向量
"""
from typing import List, Optional
import numpy as np
from sentence_transformers import SentenceTransformer
import torch


class EmbeddingService:
    """
    向量化服务
    
    使用 sentence-transformers 的本地模型
    优点：免费、不需要 API、隐私性好
    """
    
    def __init__(
        self,
        model_name: str = "sentence-transformers/all-MiniLM-L6-v2"
    ):
        """
        初始化向量化服务
        
        Args:
            model_name: HuggingFace 模型名称
                - all-MiniLM-L6-v2: 速度快，效果好 (推荐)
                - all-mpnet-base-v2: 效果更好，但更慢
                - paraphrase-multilingual-MiniLM-L12-v2: 多语言支持
        """
        # 加载预训练模型
        # 这会自动下载模型到本地
        self.model = SentenceTransformer(model_name)
        
        # 模型维度（用于验证）
        self.embedding_dim = self.model.get_sentence_embedding_dimension()
        
        # 设备选择：优先使用 GPU
        self.device = "cuda" if torch.cuda.is_available() else "cpu"
        self.model.to(self.device)
    
    def encode(
        self,
        texts: List[str],
        batch_size: int = 32,
        normalize: bool = True,
        show_progress: bool = False
    ) -> np.ndarray:
        """
        将文本列表转为向量
        
        Args:
            texts: 文本列表
            batch_size: 批处理大小
            normalize: 是否归一化（归一化后余弦相似度=点积）
            show_progress: 是否显示进度
            
        Returns:
            numpy 数组，shape = (len(texts), embedding_dim)
        """
        # 批量编码
        embeddings = self.model.encode(
            texts,
            batch_size=batch_size,
            normalize_embeddings=normalize,
            show_progress_bar=show_progress,
            convert_to_numpy=True
        )
        
        return embeddings
    
    def encode_single(self, text: str) -> np.ndarray:
        """
        编码单个文本
        
        Args:
            text: 单个文本
            
        Returns:
            向量数组
        """
        embedding = self.model.encode(
            text,
            normalize_embeddings=True,
            convert_to_numpy=True
        )
        return embedding
    
    def compute_similarity(
        self,
        text1: str,
        text2: str
    ) -> float:
        """
        计算两个文本的相似度
        
        Args:
            text1: 文本1
            text2: 文本2
            
        Returns:
            相似度分数 (0-1)
        """
        emb1 = self.encode_single(text1)
        emb2 = self.encode_single(text2)
        
        # 归一化后，点积就是余弦相似度
        similarity = np.dot(emb1, emb2)
        
        return float(similarity)


# 全局实例
embedding_service: Optional[EmbeddingService] = None


def get_embedding_service() -> EmbeddingService:
    """获取向量化服务实例（单例）"""
    global embedding_service
    if embedding_service is None:
        from config import settings
        embedding_service = EmbeddingService(
            model_name=settings.embedding_model_name
        )
    return embedding_service
```

```python
"""
向量数据库服务 - ChromaDB 操作封装

ChromaDB 特点：

- Python 原生，支持向量 CRUD

import chromadb
from chromadb.config import Settings as ChromaSettings
import hashlib

class VectorDB:
    """
    向量数据库管理器
    
    功能：
    1. 集合管理（创建、删除、获取）
    2. 文档增删改查
    3. 相似度搜索
    """
    
    def __init__(self, persist_directory: str = "./data/chroma_db"):
        """
        初始化向量数据库
        
        Args:
            persist_directory: 持久化目录
        """

            settings=ChromaSettings(
                anonymized_telemetry=False,  # 关闭遥测
                allow_reset=True  # 允许重置数据库
            )
        )
        
        # Embedding 服务
        self.embedding_service = get_embedding_service()
    
    def create_collection(
        self,
        name: str,
        metadata: Optional[Dict] = None
    ) -> chromadb.Collection:
        """
        创建集合
        
        Collection = 文档集合，类似表
        
        Args:
            name: 集合名称
            metadata: 元数据
            
        Returns:
            Collection 对象
        """
        # 如果已存在，返回现有的
        try:
            collection = self.client.get_or_create_collection(
                name=name,
                metadata=metadata or {"description": "文档集合"}
            )
            return collection
        except Exception as e:
            # 如果创建失败，删除后重新创建
            self.client.delete_collection(name)
            return self.client.create_collection(
                name=name,
                metadata=metadata or {"description": "文档集合"}
            )
    
    def get_collection(self, name: str) -> chromadb.Collection:
        """
        获取集合
        """
        return self.client.get_collection(name)
    
    def add_documents(
        self,
        collection_name: str,
        documents: List[str],
        metadatas: List[Dict[str, Any]],
        ids: Optional[List[str]] = None
    ):
        """
        添加文档到集合
        
        Args:
            collection_name: 集合名称
            documents: 文档内容列表
            metadatas: 元数据列表
            ids: ID 列表（可选，自动生成）
        """
        collection = self.get_collection(collection_name)
        
        # 生成 ID
        if ids is None:
            ids = [self._generate_id(doc) for doc in documents]
        
        # 生成向量
        # ChromaDB 可以自动调用 embedding 函数
        # 但我们手动做可以更好地控制
        embeddings = self.embedding_service.encode(documents)
        
        # 添加到集合
        collection.add(
            documents=documents,
            embeddings=embeddings.tolist(),  # 转为 list
            metadatas=metadatas,
            ids=ids
        )
    
    def similarity_search(
        self,
        collection_name: str,
        query: str,
        top_k: int = 4,
        filter_metadata: Optional[Dict] = None
    ) -> List[Dict[str, Any]]:
        """
        相似度搜索
        
        Args:
            collection_name: 集合名称
            query: 查询文本
            top_k: 返回前 K 个结果
            filter_metadata: 元数据过滤条件
            
        Returns:
            相似文档列表
        """
        collection = self.get_collection(collection_name)
        
        # 构建查询参数
        query_params = {
            "query_texts": [query],
            "n_results": top_k,
            "include": ["documents", "metadatas", "distances"]  # 返回什么
        }
        
        # 添加过滤条件
        if filter_metadata:
            # ChromaDB 的 where 语法
            query_params["where"] = filter_metadata
        
        # 执行查询
        results = collection.query(**query_params)
        
        # 解析结果
        documents = []
        if results.get("documents") and results["documents"][0]:
            for i, doc in enumerate(results["documents"][0]):
                documents.append({
                    "content": doc,
                    "metadata": results["metadatas"][0][i] if results.get("metadatas") else {},
                    "distance": results["distances"][0][i] if results.get("distances") else 0,
                    "id": results["ids"][0][i] if results.get("ids") else None
                })
        
        return documents
    
    def get_document(self, collection_name: str, doc_id: str) -> Optional[Dict]:
        """
        获取单个文档
        """
        collection = self.get_collection(collection_name)
        result = collection.get(ids=[doc_id])
        
        if result.get("documents"):
            return {
                "id": doc_id,
                "content": result["documents"][0],
                "metadata": result["metadatas"][0] if result.get("metadatas") else {}
            }
        return None
    
    def delete_documents(
        self,
        collection_name: str,
        ids: List[str]
    ):
        """
        删除文档
        """
        collection = self.get_collection(collection_name)
        collection.delete(ids=ids)
    
    def delete_by_document_id(
        self,
        collection_name: str,
        document_id: str
    ):
        """
        删除某个文档的所有块
        
        Args:
            collection_name: 集合名称
            document_id: 原始文档ID
        """
        collection = self.get_collection(collection_name)
        
        # 获取该文档的所有块
        result = collection.get(
            where={"document_id": document_id}
        )
        
        if result.get("ids"):
            collection.delete(ids=result["ids"])
    
    def get_collection_stats(self, collection_name: str) -> Dict[str, Any]:
        """获取集合统计信息"""
        collection = self.get_collection(collection_name)
        return {
            "name": collection_name,
            "count": collection.count(),
            "metadata": collection.metadata
        }
    
    def reset(self):

        self.client.reset()
    
    @staticmethod
    def _generate_id(content: str) -> str:
```

### 2.7 问答引擎 qa_engine.py
```python
"""
问答引擎 - 核心 RAG 逻辑

RAG = Retrieval + Generation

流程：
1. 理解问题
2. 检索相关文档
3. 构建 Prompt
4. 调用 LLM 生成答案
5. 返回答案和引用
"""
from typing import List, Dict, Any, Optional, AsyncGenerator
import time
from langchain.chat_models import ChatOpenAI
from langchain.prompts import PromptTemplate
from langchain.schema import StrOutputParser
from langchain_core.output_parsers import JsonOutputParser
import os


class QAEngine:
    """
    问答引擎
    
    核心功能：
    1. 检索相关文档
    2. 构建答案
    3. 流式输出
    """
    
    # 默认集合名称
    DEFAULT_COLLECTION = "documents"
    
    def __init__(self):
        """初始化问答引擎"""
        # LLM 配置
        self.llm = ChatOpenAI(
            model=os.getenv("OPENAI_MODEL", "gpt-3.5-turbo"),
            temperature=0.3,  # 问答用低温度
            streaming=True,   # 支持流式
            api_key=os.getenv("OPENAI_API_KEY"),
            base_url=os.getenv("OPENAI_API_BASE", "https://api.openai.com/v1")
        )
        
        # 向量数据库
        from .vector_db import get_vector_db
        self.vector_db = get_vector_db()
        
        # 系统提示词
        self.system_prompt = """你是一个专业的文档阅读助手，负责根据提供的内容回答用户问题。

【工作原则】
1. 严格基于提供的文档内容回答，不要编造信息
2. 如果文档中没有相关信息，明确告知用户
3. 回答要准确、简洁、有条理
4. 对于复杂问题，先分析再总结
5. 引用原文时要标注来源

【回答格式】
1. 直接回答问题
2. 如有必要，列出关键点
3. 标注引用来源（如 [来源1]）
"""
    
    async def answer(
        self,
        question: str,
        document_ids: Optional[List[str]] = None,
        top_k: int = 4,
        collection: str = None
    ) -> Dict[str, Any]:
        """
        回答问题（非流式）
        
        Args:
            question: 用户问题
            document_ids: 指定文档ID列表（空则搜索全部）
            top_k: 检索数量
            collection: 集合名称
            
        Returns:
            {"answer": str, "sources": list, "latency_ms": int}
        """
        start_time = time.time()
        
        collection = collection or self.DEFAULT_COLLECTION
        
        # 1. 检索相关文档
        filter_meta = {"document_id": {"$in": document_ids}} if document_ids else None
        docs = self.vector_db.similarity_search(
            collection_name=collection,
            query=question,
            top_k=top_k,
            filter_metadata=filter_meta
        )
        
        # 2. 构建上下文
        context = self._build_context(docs)
        
        # 3. 生成答案
        answer = await self._generate_answer(question, context)
        
        # 4. 构建来源
        sources = self._build_sources(docs)
        
        latency_ms = int((time.time() - start_time) * 1000)
        
        return {
            "answer": answer,
            "sources": sources,
            "latency_ms": latency_ms,
            "docs_found": len(docs)
        }
    
    async def stream_answer(
        self,
        question: str,
        document_ids: Optional[List[str]] = None,
        top_k: int = 4,
        collection: str = None
    ) -> AsyncGenerator[Dict[str, Any], None]:
        """
        流式回答问题
        
        逐步返回：思考内容 -> 答案 -> 来源
        
        Args:
            question: 用户问题
            document_ids: 指定文档ID
            top_k: 检索数量
            
        Yields:
            {"type": str, "content": str}
        """
        collection = collection or self.DEFAULT_COLLECTION
        
        # 1. 检索
        yield {"type": "status", "content": "正在检索相关文档..."}
        
        filter_meta = {"document_id": {"$in": document_ids}} if document_ids else None
        docs = self.vector_db.similarity_search(
            collection_name=collection,
            query=question,
            top_k=top_k,
            filter_metadata=filter_meta
        )
        
        yield {
            "type": "status",
            "content": f"找到 {len(docs)} 篇相关文档"
        }
        
        # 2. 构建上下文
        context = self._build_context(docs)
        
        # 3. 流式生成
        yield {"type": "status", "content": "正在分析文档内容..."}
        
        # 构建 Prompt
        prompt = self._build_prompt(question, context)
        
        # 流式调用
        answer_content = ""
        async for chunk in self.llm.astream(prompt):
            if hasattr(chunk, 'content') and chunk.content:
                answer_content += chunk.content
                yield {"type": "content", "content": chunk.content}
        
        # 4. 返回来源
        yield {
            "type": "sources",
            "sources": self._build_sources(docs)
        }
        
        yield {"type": "done", "content": answer_content}
    
    def _build_context(self, docs: List[Dict[str, Any]]) -> str:
        """
        构建上下文字符串
        
        将检索到的文档合并成上下文
        """
        if not docs:
            return "【没有找到相关文档】"
        
        context_parts = []
        for i, doc in enumerate(docs, 1):
            metadata = doc.get("metadata", {})
            doc_id = metadata.get("document_id", "未知")
            chunk_idx = metadata.get("chunk_index", 0)
            
            context_parts.append(
                f"【文档{i} - {doc_id} 第{chunk_idx + 1}部分】\n"
                f"{doc['content']}"
            )
        
        return "\n\n".join(context_parts)
    
    def _build_prompt(self, question: str, context: str) -> List:
        """
        构建 Prompt
        
        返回消息列表格式
        """
        from langchain.schema import HumanMessage, SystemMessage
        
        return [
            SystemMessage(content=self.system_prompt),
            HumanMessage(content=f"""【相关文档内容】
{context}

【用户问题】
{question}

请根据以上文档内容回答用户问题。""")
        ]
    
    def _generate_answer(self, question: str, context: str) -> str:
        """
        生成答案（非流式）
        """
        prompt = self._build_prompt(question, context)
        
        # 同步调用
        response = self.llm.invoke(prompt)
        
        return response.content if hasattr(response, 'content') else str(response)
    
    def _build_sources(self, docs: List[Dict[str, Any]]) -> List[Dict]:
        """
        构建来源列表
        """
        sources = []
        for doc in docs:
            metadata = doc.get("metadata", {})
            sources.append({
                "document_id": metadata.get("document_id", ""),
                "chunk_id": doc.get("id", ""),
                "content": doc.get("content", "")[:200] + "...",
                "relevance_score": 1 - doc.get("distance", 0),  # 距离转相似度
                "char_count": len(doc.get("content", ""))
            })
        
        return sources


# 全局实例
qa_engine: Optional[QAEngine] = None


def get_qa_engine() -> QAEngine:
    """获取问答引擎实例"""
    global qa_engine
    if qa_engine is None:
        qa_engine = QAEngine()
    return qa_engine
```

```python
                question=request.question,
                document_ids=request.document_ids,

                if chunk_type == "content":

                    yield f"data: {{'type': 'sources', 'sources': {chunk['sources']}}}\n\n"
                elif chunk_type == "done":
                    yield f"data: {{'type': 'done'}}\n\n"
                else:
```

```typescript
const API_BASE = '/api/chat'

export interface Source {
  document_id: string
  chunk_id: string
  content: string
  relevance_score: number
}

export interface QARequest {
  question: string
  document_ids?: string[]
  top_k?: number
}

export interface QAResponse {
  answer: string
  sources: Source[]
  latency_ms: number
}

/**
 * 普通问答
 */
export async function askQuestion(request: QARequest): Promise<QAResponse> {
  const response = await fetch(`${API_BASE}/ask`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(request)
  })
  return response.json()
}

/**
 * 流式问答
 */
export async function* askQuestionStream(request: QARequest) {
  const response = await fetch(`${API_BASE}/ask/stream`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(request)
  })
  
  const reader = response.body.getReader()
  const decoder = new TextDecoder()
  
  while (true) {
    const { done, value } = await reader.read()
    if (done) break
    
    const chunk = decoder.decode(value)
    const lines = chunk.split('\n').filter(line => line.startsWith('data: '))
    
    for (const line of lines) {
      const data = JSON.parse(line.replace('data: ', ''))
      yield data
    }
  }
}
```

```tsx
import { askQuestionStream, Source } from '../api/qaApi'

export const QAInterface: React.FC = () => {
  const [question, setQuestion] = useState('')
  const [answer, setAnswer] = useState('')
  const [sources, setSources] = useState<Source[]>([])
  const [loading, setLoading] = useState(false)
  const [status, setStatus] = useState('')

  /**
   * 提交问题
   */
  const handleSubmit = useCallback(async (e: React.FormEvent) => {
    e.preventDefault()
    if (!question.trim() || loading) return

    setLoading(true)
    setAnswer('')
    setSources([])
    setStatus('准备中...')

    try {
      // 流式获取结果
      for await (const chunk of askQuestionStream({ question })) {
        switch (chunk.type) {
          case 'status':
            setStatus(chunk.content)
            break
          case 'content':
            setAnswer(prev => prev + chunk.content)
            break
          case 'sources':
            setSources(chunk.sources)
            break
          case 'done':
            setStatus('')
            break
          case 'error':
            setStatus(`错误: ${chunk.content}`)
            break
        }
      }
    } catch (error) {
      console.error('问答失败:', error)
      setStatus('问答失败，请重试')
    } finally {
      setLoading(false)
    }
  }, [question, loading])

  return (
    <div className="qa-interface">
      {/* 问题输入 */}
      <form onSubmit={handleSubmit} className="question-form">
        <textarea
          value={question}
          onChange={e => setQuestion(e.target.value)}
          placeholder="请输入你的问题..."
          disabled={loading}
        />
        <button type="submit" disabled={loading || !question.trim()}>
          {loading ? '分析中...' : '提问'}
        </button>
      </form>

      {/* 状态显示 */}
      {status && <div className="status">{status}</div>}

      {/* 答案显示 */}
      {answer && (
        <div className="answer-section">
          <h3>📝 答案</h3>
          <div className="answer-content">
            {answer}
          </div>
        </div>
      )}

      {/* 来源显示 */}
      {sources.length > 0 && (
        <div className="sources-section">
          <h3>📚 参考来源</h3>
          {sources.map((source, idx) => (
            <div key={idx} className="source-item">
              <div className="source-score">
                相关度: {(source.relevance_score * 100).toFixed(1)}%
              </div>
              <div className="source-content">
                {source.content}
              </div>
            </div>
          ))}
        </div>
      )}
    </div>
  )
}
```

## 🔍 代码关键点说明

### 3.1 ChromaDB vs Weaviate

| 特性 | ChromaDB | Weaviate |
|------|----------|----------|
| 部署方式 | 内嵌/轻量 | 独立服务 |
| 适合场景 | 开发/小规模 | 生产/大规模 |
| API | Python优先 | 多语言 |
| 性能 | 中等 | 更高 |
| 配置复杂度 | 简单 | 较复杂 |

**选择建议**：
- 学习/演示：ChromaDB 更简单
- 生产环境：Weaviate 性能更好

### 3.2 Embedding 模型选择

```
英文场景:
- all-MiniLM-L6-v2 ⭐ (推荐：速度快，效果好)
- all-mpnet-base-v2 (效果更好，稍慢)

多语言/中文:
- paraphrase-multilingual-MiniLM-L12-v2 ⭐
- moka-ai/m3e-base (国产中文模型)
```

### 3.3 文档分块策略

**常用策略**：
1. **固定大小分块**：简单，但可能切断语义
2. **按段落分块**：保留语义完整性
3. **按标题分块**（Markdown）：层次清晰
4. **递归分块**：先大块再小块，智能切割

**参数选择**：
- chunk_size: 500-1000 字符
- chunk_overlap: 50-100 字符

## 💼 面试要点

### 问题 1: RAG 的完整流程是什么？

**参考答案：**
```
用户问题 → 问题向量化 → 向量数据库检索 → 获取 Top-K 相关文档 
→ 构建 Prompt（问题 + 文档） → LLM 生成答案 → 返回答案 + 引用
```

### 问题 2: 如何提升 RAG 的效果？

**参考答案：**
1. **文档处理**：更好的分块策略（语义分块）
2. **检索优化**：混合搜索（向量+关键词）
3. **重排序**：用更精确的模型对检索结果重排
4. **Prompt 工程**：优化系统提示词
5. **查询改写**：将口语问题转为检索友好的查询

### 问题 3: ChromaDB 的使用场景？

**参考答案：**
- 原型开发、快速验证
- 小规模应用（<100万向量）
- 需要快速迭代的项目
- 需要内嵌部署的场景

**不适用**：大规模生产环境、高并发场景
