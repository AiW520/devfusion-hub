**关键代码 - WebSocket 协作处理器**：
```python
# backend/routers/collaboration.py
from fastapi import WebSocket, WebSocketDisconnect
from typing import Dict, List
import json
import asyncio

class CollaborationManager:
    """协作会话管理器"""
    
    def __init__(self):
        # document_id -> [WebSocket connections]
        self.active_sessions: Dict[str, List[WebSocket]] = {}
        # connection_id -> user_info
        self.user_info: Dict[WebSocket, dict] = {}
    
    async def connect(self, websocket: WebSocket, document_id: str, user: dict):
        """连接协作会话"""
        await websocket.accept()
        
        if document_id not in self.active_sessions:
            self.active_sessions[document_id] = []
        
        self.active_sessions[document_id].append(websocket)
        self.user_info[websocket] = user
        
        # 广播用户加入
        await self.broadcast(document_id, {
            'type': 'user_joined',
            'user': user
        })
    
    async def disconnect(self, websocket: WebSocket, document_id: str):
        """断开连接"""
        if document_id in self.active_sessions:
            self.active_sessions[document_id].remove(websocket)
        
        user = self.user_info.get(websocket)
        if user:
            await self.broadcast(document_id, {
                'type': 'user_left',
                'user': user
            })
            del self.user_info[websocket]
    
    async def broadcast(self, document_id: str, message: dict):
        """广播消息给所有参与者"""
        if document_id not in self.active_sessions:
            return
        
        disconnected = []
        
        for connection in self.active_sessions[document_id]:
            try:
                await connection.send_json(message)
            except:
                disconnected.append(connection)
        
        # 清理断开的连接
        for conn in disconnected:
            await self.disconnect(conn, document_id)
```

    def __init__(self):

    def create_customer(self, tenant: Tenant) -> str:

        customer = self.stripe.Customer.create(
            name=tenant.name,
            metadata={'tenant_id': str(tenant.id)}
        )
        return customer.id
    
    def create_subscription(self, customer_id: str, plan: Plan, interval: str = 'monthly'):
        """创建订阅"""

            payment_behavior='default_incomplete',

## 💼 面试要点

### 问题 1: WebSocket 和 HTTP 的区别？

**参考答案**：
- HTTP：请求-响应，无状态
- WebSocket：全双工通信，长连接
- 适用场景：实时聊天、协作编辑、游戏

### 问题 2: SaaS 多租户架构？

**参考答案**：
- 共享数据库 + tenant_id 隔离
- 独立数据库（高隔离要求）
- Schema 隔离（PostgreSQL）

### 问题 3: 如何保证消息可靠性？

**参考答案**：
1. 消息持久化到数据库
2. 消息确认机制
3. 重试机制
4. 幂等性处理
