## 📋 项目介绍

### 1.1 项目概述
本项目是一个**用户行为数据管道**，用于采集、清洗、转换和存储用户行为数据（如点击、浏览、购买等），为用户画像、推荐系统、AB测试提供数据支持。采用 Lambda 架构（批处理 + 流处理），保证数据的实时性和准确性。

### 1.2 核心技术亮点
- **Lambda 架构**：批处理层 + 速度层 + 服务层
- **Spark 批处理**：每日全量数据处理
- **Spark Streaming**：实时数据处理
- **Airflow DAG**：任务调度和工作流管理
- **数据仓库分层**：ODS → DWD → DWS → ADS

### 1.3 适用场景
- 用户行为分析
- 用户画像构建
- 个性化推荐
- A/B 测试分析
- 留存和转化分析

```sql
-- 转化漏斗表（ADS 层）
CREATE TABLE ads.conversion_funnel (
    stat_date DATE NOT NULL,
```

### 2.3 批处理代码
```python
# batch_processing/src/batch_job.py
"""
每日批处理主程序

处理流程：
1. 数据质量检查
2. ODS 层数据清洗
3. DWD 层维度构建
4. DWS 层指标聚合
5. ADS 层报表生成
"""
import os
import sys
import logging
from datetime import datetime, timedelta
from pyspark.sql import SparkSession
from pyspark.sql.functions import (
    col, when, lit, sum as spark_sum, avg, count, max as spark_max, min as spark_min,
    countDistinct, expr, to_date, from_unixtime, unix_timestamp,
    concat_ws, split, substring, lower, trim, regexp_replace
)
from pyspark.sql.types import StructType, StructField, StringType, IntegerType, DoubleType, TimestampType

# 配置日志
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(levelname)s - %(message)s'
)
logger = logging.getLogger(__name__)


class BatchProcessor:
    """
    批处理处理器
    
    处理每日增量数据和全量刷新
    """
    
    def __init__(self, config: dict):
        """
        初始化处理器
        
        Args:
            config: 配置字典
        """
        self.config = config
        self.spark = self._create_spark_session()
        self.stat_date = config.get("stat_date", datetime.now().strftime("%Y-%m-%d"))
        
    def _create_spark_session(self) -> SparkSession:
        """创建 Spark Session"""
        config = self.config
        
        builder = SparkSession.builder \
            .appName(f"BatchProcessor_{self.__class__.__name__}") \
            .config("spark.sql.adaptive.enabled", "true") \
            .config("spark.sql.adaptive.coalescePartitions.enabled", "true")
        
        # PostgreSQL JDBC 配置
        if config.get("jdbc_url"):
            builder = builder \
                .config("spark.sql.jdbc.url", config["jdbc_url"]) \
                .config("spark.sql.jdbc.driver", "org.postgresql.Driver")
        
        # 添加 PostgreSQL 驱动
        builder = builder.config(
            "spark.jars.packages",
            "org.postgresql:postgresql:42.7.1"
        )
        
        return builder.getOrCreate()
    
    def process(self):
        """执行完整批处理流程"""
        logger.info(f"🚀 开始批处理，日期: {self.stat_date}")
        
        try:
            # 1. 数据质量检查
            self._check_data_quality()
            
            # 2. ODS 层处理
            ods_df = self._process_ods()
            
            # 3. DWD 层处理
            self._process_dwd(ods_df)
            
            # 4. DWS 层处理
            self._process_dws(ods_df)
            
            # 5. ADS 层处理
            self._process_ads()
            
            # 6. 更新用户画像
            self._update_user_profiles()
            
            logger.info(f"✅ 批处理完成，日期: {self.stat_date}")
            
        except Exception as e:
            logger.error(f"❌ 批处理失败: {e}")
            raise
        
        finally:
            self.spark.stop()
    
    def _check_data_quality(self):
        """数据质量检查"""
        logger.info("🔍 进行数据质量检查...")
        
        # 检查 ODS 层数据量
        ods_count = self._get_ods_count()
        logger.info(f"📊 ODS 层数据量: {ods_count}")
        
        # 检查数据完整性
        if ods_count == 0:
            raise ValueError("ODS 层数据为空！")
        
        # 检查重复数据
        duplicates = self._check_duplicates()
        if duplicates > 0:
            logger.warning(f"⚠️ 发现 {duplicates} 条重复数据")
        
        # 检查数据分布
        self._check_data_distribution()
        
        logger.info("✅ 数据质量检查通过")
    
    def _get_ods_count(self) -> int:
        """获取 ODS 层数据量"""
        query = f"""
            SELECT COUNT(*) as cnt 
            FROM ods.user_events 
            WHERE DATE(ts) = '{self.stat_date}'
        """
        df = self.spark.read \
            .format("jdbc") \
            .option("url", self.config["jdbc_url"]) \
            .option("query", query) \
            .option("driver", "org.postgresql.Driver") \
            .load()
        return df.collect()[0]["cnt"]
    
    def _check_duplicates(self) -> int:
        """检查重复数据"""
        query = f"""
            SELECT COUNT(*) - COUNT(DISTINCT event_id) as duplicates
            FROM ods.user_events
            WHERE DATE(ts) = '{self.stat_date}'
        """
        df = self.spark.read \
            .format("jdbc") \
            .option("url", self.config["jdbc_url"]) \
            .option("query", query) \
            .option("driver", "org.postgresql.Driver") \
            .load()
        return df.collect()[0]["duplicates"]
    
    def _check_data_distribution(self):
        """检查数据分布"""
        query = f"""
            SELECT 
                event_type,
                COUNT(*) as cnt,
                COUNT(DISTINCT user_id) as user_cnt
            FROM ods.user_events
            WHERE DATE(ts) = '{self.stat_date}'
            GROUP BY event_type
        """
        df = self.spark.read \
            .format("jdbc") \
            .option("url", self.config["jdbc_url"]) \
            .option("query", query) \
            .option("driver", "org.postgresql.Driver") \
            .load()
        
        logger.info("📈 数据分布:")
        for row in df.collect():
            logger.info(f"  {row['event_type']}: {row['cnt']} 事件, {row['user_cnt']} 用户")
    
    def _process_ods(self):
        """
        ODS 层处理
        
        数据清洗和标准化
        """
        logger.info("📦 处理 ODS 层数据...")
        
        # 从 Kafka 导出的数据读取（实际从 Kafka 或文件系统读取）
        # 这里演示从 PostgreSQL 读取
        query = f"""
            SELECT * FROM ods.user_events
            WHERE DATE(ts) = '{self.stat_date}'
        """
        
        df = self.spark.read \
            .format("jdbc") \
            .option("url", self.config["jdbc_url"]) \
            .option("query", query) \
            .option("driver", "org.postgresql.Driver") \
            .load()
        
        # 数据清洗
        cleaned_df = df \
            .filter(col("event_id").isNotNull()) \
            .filter(col("ts").isNotNull()) \
            .filter(col("event_type").isNotNull()) \
            .withColumn("user_id", 
                when(col("user_id").isNull(), lit("anonymous")).otherwise(col("user_id"))) \
            .withColumn("device_type", lower(trim(col("device_type")))) \
            .withColumn("os_type", lower(trim(col("os_type")))) \
            .withColumn("event_type", lower(trim(col("event_type")))) \
            .withColumn("duration_ms", 
                when(col("duration_ms") < 0, lit(0)).otherwise(col("duration_ms"))) \
            .withColumn("stat_date", to_date(col("ts")))
        
        # 缓存
        cleaned_df = cleaned_df.cache()
        
        # 去重
        # 使用事件ID去重，保留第一条
        dedup_df = cleaned_df.dropDuplicates(["event_id"])
        
        logger.info(f"✅ ODS 处理完成: {dedup_df.count()} 条记录")
        
        return dedup_df
    
    def _process_dwd(self, ods_df):
        """
        DWD 层处理
        
        构建维度表
        """
        logger.info("🏗️ 处理 DWD 层数据...")
        
        # 1. 处理用户维度表
        self._process_dim_user(ods_df)
        
        # 2. 处理商品维度表
        self._process_dim_item(ods_df)
        
        logger.info("✅ DWD 处理完成")
    
    def _process_dim_user(self, ods_df):
        """
        构建用户维度表
        
        使用 SCD Type 2 缓慢变化维度
        """
        # 聚合用户基础信息
        user_df = ods_df.groupBy("user_id").agg(
            # 用户类型
            count("*").alias("total_events"),
            countDistinct("session_id").alias("total_sessions"),
            min("ts").alias("first_seen_at"),
            max("ts").alias("last_seen_at"),
            # 设备偏好
            expr("mode(device_type)").alias("device_preference"),
            expr("mode(os_type)").alias("os_preference"),
            # 地理信息
            expr("mode(geo_city)").alias("geo_region"),
            # 渠道信息
            expr("mode(channel)").alias("first_channel"),
            # 支付标识
            spark_sum(when(col("event_type") == "purchase", 1).otherwise(0)).alias("purchase_count"),
            spark_sum(when(col("event_type") == "purchase", col("duration_ms")).otherwise(0)).alias("lifetime_value")
        ) \
        .withColumn("user_type",
            when(col("total_events") >= 100, "active")
            .when(col("total_events") >= 10, "normal")
            .otherwise("new")) \
        .withColumn("is_paying", when(col("purchase_count") > 0, True).otherwise(False)) \
        .withColumn("lifetime_value", col("lifetime_value").cast(DoubleType()) / 100)  # 转为金额
        
        # 去重后写入
        self._write_to_postgres(
            user_df, 
            "dwd.dim_user",
            mode="upsert",
            key_col="user_id"
        )
    
    def _process_dim_item(self, ods_df):
        """
        构建商品维度表
        """
        # 过滤有商品ID的事件
        item_events = ods_df.filter(col("item_id").isNotNull())
        
        item_df = item_events.groupBy("item_id").agg(
            min("ts").alias("first_seen_at"),
            count(when(col("event_type") == "click")).alias("click_count"),
            count(when(col("event_type") == "view")).alias("view_count"),
            count(when(col("event_type") == "cart")).alias("cart_count"),
            count(when(col("event_type") == "purchase")).alias("purchase_count")
        )
        
        self._write_to_postgres(
            item_df,
            "dwd.dim_item",
            mode="upsert",
            key_col="item_id"
        )
    
    def _process_dws(self, ods_df):
        """
        DWS 层处理
        
        汇总事实表
        """
        logger.info("📊 处理 DWS 层数据...")
        
        # 1. 每日用户行为汇总
        self._aggregate_daily_user_behavior(ods_df)
        
        # 2. 每日商品统计
        self._aggregate_daily_item_stats(ods_df)
        
        logger.info("✅ DWS 处理完成")
    
    def _aggregate_daily_user_behavior(self, ods_df):
        """
        聚合每日用户行为
        """
        daily_df = ods_df.groupBy("stat_date", "user_id").agg(
            # 事件统计
            count(when(col("event_type") == "click")).alias("click_count"),
            count(when(col("event_type") == "view")).alias("view_count"),
            count(when(col("event_type") == "cart")).alias("cart_count"),
            count(when(col("event_type") == "purchase")).alias("purchase_count"),
            spark_sum(when(col("event_type") == "purchase", 1).otherwise(0) * 100)
                .alias("purchase_amount"),  # 简化，实际从商品表获取价格
            # 会话统计
            countDistinct("session_id").alias("session_count"),
            avg("duration_ms").alias("avg_page_duration"),
            spark_max("duration_ms").alias("max_depth")
        ) \
        .withColumn("purchase_amount", (col("purchase_amount") / 100).cast(DoubleType())) \
        .withColumn("avg_page_duration", col("avg_page_duration").cast(IntegerType()))
        
        self._write_to_postgres(
            daily_df,
            "dws.daily_user_behavior",
            mode="append"
        )
    
    def _aggregate_daily_item_stats(self, ods_df):
        """
        聚合每日商品统计
        """
        item_df = ods_df.filter(col("item_id").isNotNull())
        
        stats_df = item_df.groupBy("stat_date", "item_id", "category_id").agg(
            count(when(col("event_type") == "click")).alias("click_count"),
            count(when(col("event_type") == "view")).alias("view_count"),
            countDistinct(when(col("event_type") == "view", "user_id")).alias("view_user_count"),
            count(when(col("event_type") == "cart")).alias("cart_count"),
            countDistinct(when(col("event_type") == "cart", "user_id")).alias("cart_user_count"),
            count(when(col("event_type") == "purchase")).alias("purchase_count"),
            countDistinct(when(col("event_type") == "purchase", "user_id")).alias("purchase_user_count")
        )
        
        self._write_to_postgres(
            stats_df,
            "dws.daily_item_stats",
            mode="append"
        )
    
    def _process_ads(self):
        """
        ADS 层处理
        
        生成分析报表
        """
        logger.info("📈 生成 ADS 层报表...")
        
        # 1. 生成转化漏斗
        self._generate_conversion_funnel()
        
        # 2. 生成留存报表
        self._generate_retention_report()
        
        logger.info("✅ ADS 处理完成")
    
    def _generate_conversion_funnel(self):
        """
        生成转化漏斗
        """
        query = f"""
            WITH funnel AS (
                SELECT 
                    user_id,
                    MAX(CASE WHEN event_type = 'view' THEN 1 ELSE 0 END) as viewed,
                    MAX(CASE WHEN event_type = 'cart' THEN 1 ELSE 0 END) as carted,
                    MAX(CASE WHEN event_type = 'purchase' THEN 1 ELSE 0 END) as purchased
                FROM ods.user_events
                WHERE DATE(ts) = '{self.stat_date}'
                GROUP BY user_id
            )
            SELECT 
                'visit' as funnel_step,
                COUNT(*) as user_count
                FROM (SELECT DISTINCT user_id FROM ods.user_events WHERE DATE(ts) = '{self.stat_date}')
            UNION ALL
            SELECT 'view' as funnel_step, COUNT(*) as user_count FROM funnel WHERE viewed = 1
            UNION ALL
            SELECT 'cart' as funnel_step, COUNT(*) as user_count FROM funnel WHERE carted = 1
            UNION ALL
            SELECT 'purchase' as funnel_step, COUNT(*) as user_count FROM funnel WHERE purchased = 1
        """
        
        funnel_df = self.spark.read \
            .format("jdbc") \
            .option("url", self.config["jdbc_url"]) \
            .option("query", query) \
            .option("driver", "org.postgresql.Driver") \
            .load()
        
        # 计算转化率
        rows = funnel_df.collect()
        total_users = rows[0]["user_count"]
        
        result = []
        prev_count = total_users
        for row in rows:
            step = row["funnel_step"]
            count = row["user_count"]
            result.append({
                "stat_date": self.stat_date,
                "funnel_step": step,
                "user_count": count,
                "conversion_rate": count / prev_count if prev_count > 0 else 0,
                "overall_rate": count / total_users if total_users > 0 else 0
            })
            prev_count = count
        
        result_df = self.spark.createDataFrame(result)
        self._write_to_postgres(result_df, "ads.conversion_funnel", mode="append")
    
    def _generate_retention_report(self):
        """
        生成留存报表
        """
        # 计算每日留存
        query = f"""
            WITH first_seen AS (
                SELECT user_id, DATE(MIN(ts)) as cohort_date
                FROM ods.user_events
                WHERE ts >= DATE_SUB('{self.stat_date}', 30)
                GROUP BY user_id
            ),
            daily_active AS (
                SELECT user_id, DATE(ts) as active_date
                FROM ods.user_events
                WHERE DATE(ts) >= DATE_SUB('{self.stat_date}', 30)
            )
            SELECT 
                f.cohort_date,
                DATEDIFF(d.active_date, f.cohort_date) as days_since_signup,
                COUNT(DISTINCT d.user_id) as retained_users
            FROM first_seen f
            JOIN daily_active d ON f.user_id = d.user_id
            WHERE DATEDIFF(d.active_date, f.cohort_date) IN (1, 3, 7, 14, 30)
            GROUP BY f.cohort_date, DATEDIFF(d.active_date, f.cohort_date)
        """
        
        retention_df = self.spark.read \
            .format("jdbc") \
            .option("url", self.config["jdbc_url"]) \
            .option("query", query) \
            .option("driver", "org.postgresql.Driver") \
            .load()
        
        # 计算留存率
        cohort_size_df = retention_df.groupBy("cohort_date").agg(
            spark_sum("retained_users").alias("cohort_size")
        )
        
        retention_df = retention_df.join(
            cohort_size_df,
            "cohort_date"
        ).withColumn(
            "retention_rate",
            col("retained_users").cast(DoubleType()) / col("cohort_size")
        ).withColumn(
            "stat_date", lit(self.stat_date)
        )
        
        self._write_to_postgres(retention_df, "ads.user_retention", mode="append")
    
    def _update_user_profiles(self):
        """
        更新用户画像
        
        基于历史数据计算用户特征
        """
        logger.info("👤 更新用户画像...")
        
        # 计算用户历史统计
        query = """
            WITH user_stats AS (
                SELECT 
                    user_id,
                    COUNT(*) as total_events,
                    COUNT(DISTINCT DATE(ts)) as active_days,
                    MAX(COUNT(*)) OVER() as max_events
                FROM ods.user_events
                GROUP BY user_id
            )
            SELECT 
                user_id,
                CASE 
                    WHEN total_events > max_events * 0.8 THEN 'highly_active'
                    WHEN total_events > max_events * 0.5 THEN 'active'
                    WHEN total_events > max_events * 0.2 THEN 'normal'
                    ELSE 'low_active'
                END as activity_level,
                CASE
                    WHEN active_days >= 20 THEN 'daily_user'
                    WHEN active_days >= 10 THEN 'weekly_user'
                    WHEN active_days >= 3 THEN 'occasional_user'
                    ELSE 'casual_user'
                END as engagement_level
            FROM user_stats
        """
        
        profile_df = self.spark.read \
            .format("jdbc") \
            .option("url", self.config["jdbc_url"]) \
            .option("query", query) \
            .option("driver", "org.postgresql.Driver") \
            .load()
        
        logger.info("✅ 用户画像更新完成")
    
    def _write_to_postgres(
        self, 
        df, 
        table: str, 
        mode: str = "append",
        key_col: str = None
    ):
        """
        写入 PostgreSQL
        
        Args:
            df: DataFrame
            table: 表名
            mode: 写入模式 (append, overwrite, upsert)
            key_col: 主键列（upsert 模式需要）
        """
        properties = {
            "url": self.config["jdbc_url"],
            "driver": "org.postgresql.Driver",
            "batchsize": "10000"  # 批量写入大小
        }
        
        if mode == "upsert" and key_col:
            # 实现 upsert：先删除再插入
            # 注意：这里简化处理，实际应该用 MERGE
            logger.info(f"📤 写入 {table} (upsert)...")
            df.write \
                .format("jdbc") \
                .option("url", properties["url"]) \
                .option("dbtable", table) \
                .option("driver", properties["driver"]) \
                .mode("overwrite") \
                .save()
        else:
            logger.info(f"📤 写入 {table} ({mode})...")
            df.write \
                .jdbc(
                    url=properties["url"],
                    table=table,
                    mode=mode,
                    properties=properties
                )
        
        logger.info(f"✅ {table} 写入完成: {df.count()} 条记录")
```

```python
# streaming/src/streaming_job.py
"""
实时流处理

功能：
1. 消费 Kafka 行为数据
2. 实时计算指标
3. 实时更新用户画像
4. 实时告警
"""
from pyspark.sql import SparkSession
from pyspark.sql.functions import (
    col, from_json, window, current_timestamp, expr,
    when, sum as spark_sum, count, avg, max as spark_max
)
from pyspark.sql.types import StructType, StructField, StringType, IntegerType, LongType, TimestampType
import logging

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)


# 事件 Schema
EVENT_SCHEMA = StructType([
    StructField("event_id", StringType(), False),
    StructField("event_type", StringType(), False),
    StructField("user_id", StringType(), True),
    StructField("item_id", StringType(), True),
    StructField("session_id", StringType(), True),
    StructField("device_type", StringType(), True),
    StructField("os_type", StringType(), True),
    StructField("channel", StringType(), True),
    StructField("duration_ms", IntegerType(), True),
    StructField("ts", StringType(), False),
    StructField("price", IntegerType(), True),  # 分为单位
])


class StreamingJob:
    """
    流处理任务
    
    处理流程：
    1. 从 Kafka 读取
    2. 解析 JSON
    3. 实时聚合
    4. 输出到各 Sink
    """
    
    def __init__(self, config: dict):
        self.config = config
        self.spark = self._create_spark_session()
        self.parser = EventParser()
        
    def _create_spark_session(self) -> SparkSession:
        """创建 Spark Session"""
        return SparkSession.builder \
            .appName("StreamingBehaviorProcessor") \
            .config("spark.sql.streaming.checkpointLocation", "/tmp/streaming_checkpoint") \
            .config("spark.jars.packages", 
                    "org.apache.spark:spark-sql-kafka-0-10_2.12:3.5.0,"
                    "org.postgresql:postgresql:42.7.1") \
            .getOrCreate()
    
    def process(self):
        """启动流处理"""
        logger.info("🚀 启动流处理...")
        
        # 1. 从 Kafka 读取
        raw_df = self._read_from_kafka()
        
        # 2. 解析事件
        events_df = self._parse_events(raw_df)
        
        # 3. 实时聚合
        self._realtime_aggregation(events_df)
        
        # 4. 实时告警
        self._realtime_alerts(events_df)
        
        # 5. 输出到 PostgreSQL
        self._write_to_postgres(events_df)
        
        # 6. 输出到 Kafka（下游消费）
        self._write_to_kafka(events_df)
        
        # 等待终止
        self.spark.streams.awaitAnyTermination()
    
    def _read_from_kafka(self):
        """从 Kafka 读取"""
        return self.spark \
            .readStream \
            .format("kafka") \
            .option("kafka.bootstrap.servers", self.config["kafka_servers"]) \
            .option("subscribe", self.config["kafka_topic"]) \
            .option("startingOffsets", "latest") \
            .load()
    
    def _parse_events(self, df):
        """解析事件"""
        return df \
            .select(from_json(col("value").cast("string"), EVENT_SCHEMA).alias("data")) \
            .select("data.*") \
            .withColumn("event_time", col("ts").cast("timestamp")) \
            .withColumn("event_date", expr("DATE(ts)"))
    
    def _realtime_aggregation(self, df):
        """
        实时聚合
        
        计算实时指标
        """
        # 每分钟聚合
        minute_agg = df \
            .withWatermark("event_time", "10 minutes") \
            .groupBy(
                window(col("event_time"), "1 minute"),
                col("event_type")
            ) \
            .agg(
                count("*").alias("event_count"),
                countDistinct("user_id").alias("user_count"),
                countDistinct("session_id").alias("session_count"),
                avg("duration_ms").alias("avg_duration")
            )
        
        # 输出到控制台（调试）
        minute_agg.writeStream \
            .format("console") \
            .outputMode("update") \
            .option("truncate", "false") \
            .start()
        
        # 输出到 Kafka
        minute_agg.selectExpr("to_json(struct(*)) as value") \
            .writeStream \
            .format("kafka") \
            .option("kafka.bootstrap.servers", self.config["kafka_servers"]) \
            .option("topic", "realtime-metrics") \
            .option("checkpointLocation", "/tmp/kafka_checkpoint") \
            .start()
    
    def _realtime_alerts(self, df):
        """
        实时告警
        
        检测异常并告警
        """
        # 错误率告警（每分钟）
        error_stats = df \
            .withWatermark("event_time", "5 minutes") \
            .groupBy(window(col("event_time"), "1 minute")) \
            .agg(
                (spark_sum(when(col("event_type") == "error", 1).otherwise(0)) / 
                 count("*") * 100).alias("error_rate")
            )
        
        # 过滤高错误率
        high_error = error_stats.filter(col("error_rate") > 5)
        
        # 输出告警（实际应该发送到告警系统）
        high_error.writeStream \
            .foreachBatch(self._send_alert) \
            .outputMode("update") \
            .start()
    
    def _send_alert(self, df, batch_id):
        """发送告警"""
        for row in df.collect():
            logger.warning(
                f"🚨 告警: 错误率 {row['error_rate']:.2f}% "
                f"超过阈值，时间窗口 {row['window']}"
            )
            # TODO: 发送到告警系统（钉钉/飞书/Slack）
    
    def _write_to_postgres(self, df):
        """
        输出到 PostgreSQL
        
        使用 foreachBatch 实现批量写入
        """
        def write_batch(batch_df, batch_id):
            batch_df.write \
                .format("jdbc") \
                .option("url", self.config["jdbc_url"]) \
                .option("dbtable", "ods.realtime_events") \
                .option("driver", "org.postgresql.Driver") \
                .mode("append") \
                .save()
        
        df.writeStream \
            .foreachBatch(write_batch) \
            .outputMode("append") \
            .start()
    
    def _write_to_kafka(self, df):
        """输出到 Kafka"""
        # 选择需要的字段
        output_df = df.selectExpr(
            "event_id as key",
            "to_json(struct(event_type, user_id, item_id, session_id)) as value"
        )
        
        output_df.writeStream \
            .format("kafka") \
            .option("kafka.bootstrap.servers", self.config["kafka_servers"]) \
            .option("topic", "enriched-events") \
            .option("checkpointLocation", "/tmp/enrich_checkpoint") \
            .start()


class EventParser:
    """
    事件解析器
    
    事件类型：
    - click: 点击
    - view: 浏览
    - cart: 加购
    - purchase: 购买
    - refund: 退款
    """
    
    VALID_EVENTS = ["click", "view", "cart", "purchase", "refund"]
    
    def validate(self, df):
        """验证事件"""
        return df \
            .filter(col("event_id").isNotNull()) \
            .filter(col("event_type").isin(self.VALID_EVENTS)) \
            .filter(col("ts").isNotNull())
    
    def enrich(self, df):
        """丰富事件数据"""
        return df \
            .withColumn("is_conversion", 
                        when(col("event_type").isin(["cart", "purchase"]), True)
                        .otherwise(False)) \
            .withColumn("hour", expr("HOUR(ts)")) \
            .withColumn("weekday", expr("DAYOFWEEK(ts)"))


def main():
    config = {
        "kafka_servers": "localhost:9092",
        "kafka_topic": "user-events",
        "jdbc_url": "jdbc:postgresql://localhost:5432/analytics"
    }
    
    job = StreamingJob(config)
    job.process()


if __name__ == "__main__":
    main()
```

## 💼 面试要点

### 问题 1: 数据仓库分层的目的是什么？

**参考答案：**

1. **清晰的数据结构**：每层职责明确
2. **解耦**：上层变化不影响下层
3. **数据复用**：中间层可以被多处使用
4. **性能优化**：针对不同层使用不同的存储和处理
5. **数据质量**：逐层清洗，提高数据质量

**各层职责**：
- ODS：原始数据，保留历史
- DWD：清洗标准化，一致性
- DWS：按主题汇总
- ADS：面向应用，报表数据

### 问题 2: Lambda 架构的优缺点？

**参考答案：**

**优点**：
1. 同时保证实时性和准确性
2. 批处理层可以修正流处理层的错误
3. 容错性好

**缺点**：
1. 维护两套代码
2. 复杂度高
3. 资源消耗大

**替代方案**：Kappa 架构（只用流处理）

### 问题 3: 如何保证数据质量？

**参考答案：**

1. **完整性**：NOT NULL、记录数检查
```python
def check_completeness(df, required_cols):
    for col in required_cols:
        null_count = df.filter(col(col).isNull()).count()
        assert null_count == 0, f"{col} has null values"
```

2. **唯一性**：主键唯一
```sql
SELECT user_id, COUNT(*) 
FROM dwd.dim_user 
GROUP BY user_id 
HAVING COUNT(*) > 1
```

3. **一致性**：跨表数据一致
```sql
-- ODS 和 DWD 记录数对比
SELECT 
    (SELECT COUNT(*) FROM ods.user_events) as ods_count,
    (SELECT COUNT(*) FROM dwd.dim_user) as dwd_count
```

4. **及时性**：数据延迟监控
```python
# 检查最新数据时间
latest_ts = df.select(max("ts")).collect()[0][0]
assert (datetime.now() - latest_ts).seconds < 3600, "Data delayed!"
```
