## 📋 项目介绍

### 1.1 项目概述
本项目是一个功能完整的**电商后台管理系统**，包含商品管理、订单管理、用户管理、库存管理、数据统计等核心模块。采用前后端分离架构，Django REST Framework 提供 API，React 提供前端界面。

### 1.2 核心技术亮点
- **Django REST Framework**：完善的 RESTful API
- **JWT 认证**：安全的身份验证
- **Celery 异步任务**：订单处理、邮件发送
- **Redis 缓存**：热点数据缓存
- **Django Admin**：增强的后台管理

### 1.3 适用场景
- 电商后台管理
- ERP 系统
- CRM 系统
- 企业内部管理系统

```python
"""
商品模块模型

包含：商品分类、品牌、商品、SKU、库存
"""
from django.db import models
from django.contrib.auth import get_user_model
from decimal import Decimal

User = get_user_model()


class Category(models.Model):
    """
    商品分类
    
    支持多级分类（父子关系）
    """
    name = models.CharField("分类名称", max_length=100)
    slug = models.SlugField("URL别名", unique=True)
    parent = models.ForeignKey(
        'self',
        verbose_name="父分类",
        on_delete=models.CASCADE,
        null=True,
        blank=True,
        related_name='children'
    )
    icon = models.ImageField("图标", upload_to='categories/', null=True, blank=True)
    sort_order = models.IntegerField("排序", default=0)
    is_active = models.BooleanField("是否启用", default=True)
    created_at = models.DateTimeField("创建时间", auto_now_add=True)
    updated_at = models.DateTimeField("更新时间", auto_now=True)

    class Meta:
        verbose_name = "商品分类"
        verbose_name_plural = verbose_name
        ordering = ['sort_order', 'name']

    def __str__(self):
        return self.name

    def get_full_path(self):
        """获取完整路径"""
        if self.parent:
            return f"{self.parent.get_full_path()} > {self.name}"
        return self.name


class Brand(models.Model):
    """
    品牌
    """
    name = models.CharField("品牌名称", max_length=100)
    slug = models.SlugField("URL别名", unique=True)
    logo = models.ImageField("Logo", upload_to='brands/')
    description = models.TextField("描述", blank=True)
    website = models.URLField("官网", blank=True)
    is_active = models.BooleanField("是否启用", default=True)
    created_at = models.DateTimeField("创建时间", auto_now_add=True)

    class Meta:
        verbose_name = "品牌"
        verbose_name_plural = verbose_name

    def __str__(self):
        return self.name


class Product(models.Model):
    """
    商品（SPU）
    
    SPU = Standard Product Unit，标准产品单元
    如：iPhone 15 是一个 SPU
    """
    STATUS_CHOICES = [
        ('draft', '草稿'),
        ('pending', '待审核'),
        ('approved', '已上架'),
        ('rejected', '已拒绝'),
        ('offline', '已下架'),
    ]

    name = models.CharField("商品名称", max_length=200)
    slug = models.SlugField("URL别名")
    category = models.ForeignKey(
        Category,
        verbose_name="分类",
        on_delete=models.PROTECT,  # 保护删除
        related_name='products'
    )
    brand = models.ForeignKey(
        Brand,
        verbose_name="品牌",
        on_delete=models.SET_NULL,
        null=True,
        blank=True,
        related_name='products'
    )
    
    # 基本信息
    description = models.TextField("商品描述")
    images = models.JSONField("图片列表", default=list)  # ["url1", "url2"]
    video_url = models.URLField("视频链接", blank=True)
    
    # 属性
    tags = models.JSONField("标签", default=list)  # ["新品", "热卖"]
    keywords = models.CharField("搜索关键词", max_length=500, blank=True)
    
    # 统计
    view_count = models.IntegerField("浏览次数", default=0)
    favorite_count = models.IntegerField("收藏次数", default=0)
    sale_count = models.IntegerField("销量", default=0)
    
    # 审核
    status = models.CharField("状态", max_length=20, choices=STATUS_CHOICES, default='draft')
    reject_reason = models.TextField("拒绝原因", blank=True)
    
    # 管理员
    created_by = models.ForeignKey(
        User,
        verbose_name="创建人",
        on_delete=models.SET_NULL,
        null=True,
        related_name='created_products'
    )
    
    created_at = models.DateTimeField("创建时间", auto_now_add=True)
    updated_at = models.DateTimeField("更新时间", auto_now=True)

    class Meta:
        verbose_name = "商品"
        verbose_name_plural = verbose_name
        ordering = ['-created_at']
        indexes = [
            models.Index(fields=['category', 'status']),
            models.Index(fields=['status', '-sale_count']),
        ]

    def __str__(self):
        return self.name


class SKU(models.Model):
    """
    商品规格（SKU）
    
    SKU = Stock Keeping Unit，库存量单位
    如：iPhone 15 128GB 黑色 是一个 SKU
    """
    product = models.ForeignKey(
        Product,
        verbose_name="商品",
        on_delete=models.CASCADE,
        related_name='skus'
    )
    
    # SKU 编码
    code = models.CharField("SKU编码", max_length=50, unique=True)
    
    # 规格属性（JSON 存储）
    # {"颜色": "黑色", "内存": "128GB"}
    specs = models.JSONField("规格属性", default=dict)
    
    # 价格
    price = models.DecimalField("售价", max_digits=10, decimal_places=2)
    cost = models.DecimalField("成本", max_digits=10, decimal_places=2)
    original_price = models.DecimalField("原价", max_digits=10, decimal_places=2)
    
    # 库存
    stock = models.IntegerField("库存", default=0)
    reserved_stock = models.IntegerField("预留库存", default=0)  # 已下单未发货
    
    # 重量（克）
    weight = models.IntegerField("重量", default=0)
    
    # 状态
    is_active = models.BooleanField("是否上架", default=True)
    
    created_at = models.DateTimeField("创建时间", auto_now_add=True)
    updated_at = models.DateTimeField("更新时间", auto_now=True)

    class Meta:
        verbose_name = "SKU"
        verbose_name_plural = verbose_name
        indexes = [
            models.Index(fields=['code']),
            models.Index(fields=['product', 'is_active']),
        ]

    def __str__(self):
        specs_str = ' '.join([f"{k}:{v}" for k, v in self.specs.items()])
        return f"{self.product.name} {specs_str}"

    @property
    def available_stock(self):
        """可用库存"""
        return self.stock - self.reserved_stock


class InventoryLog(models.Model):
    """
    库存变动日志
    """
    OPERATION_CHOICES = [
        ('in', '入库'),
        ('out', '出库'),
        ('adjust', '调整'),
        ('lock', '锁定'),
        ('unlock', '解锁'),
    ]

    sku = models.ForeignKey(
        SKU,
        verbose_name="SKU",
        on_delete=models.CASCADE,
        related_name='inventory_logs'
    )
    operation = models.CharField("操作类型", max_length=20, choices=OPERATION_CHOICES)
    quantity = models.IntegerField("数量", help_text="正数入库，负数出库")
    before_stock = models.IntegerField("变动前库存")
    after_stock = models.IntegerField("变动后库存")
    
    reason = models.CharField("原因", max_length=200, blank=True)
    order = models.ForeignKey(
        'orders.Order',
        verbose_name="关联订单",
        on_delete=models.SET_NULL,
        null=True,
        blank=True
    )
    
    operator = models.ForeignKey(
        User,
        verbose_name="操作人",
        on_delete=models.SET_NULL,
        null=True
    )
    created_at = models.DateTimeField("创建时间", auto_now_add=True)

    class Meta:
        verbose_name = "库存日志"
        verbose_name_plural = verbose_name
        ordering = ['-created_at']
```

```python
class SKUSerializer(serializers.ModelSerializer):
    """
    SKU 序列化器
    
    只读字段用于展示，可写字段用于创建/更新
    """
    available_stock = serializers.IntegerField(
        source='available_stock',
        read_only=True
    )
    product_name = serializers.CharField(
        source='product.name',
        read_only=True
    )

    class Meta:
        model = SKU
        fields = [
            'id', 'product', 'product_name', 'code',
            'specs', 'price', 'cost', 'original_price',
            'stock', 'reserved_stock', 'available_stock',
            'weight', 'is_active',
            'created_at', 'updated_at'
        ]
        read_only_fields = ['reserved_stock']

    def validate_code(self, value):
        """验证 SKU 编码"""
        # 编码格式检查：大写字母+数字
        import re
        if not re.match(r'^[A-Z0-9]+$', value):
            raise serializers.ValidationError("SKU编码必须是大写字母和数字的组合")
        
        # 唯一性检查（更新时排除自己）
        instance = self.instance
        if SKU.objects.filter(code=value).exclude(
            pk=instance.pk if instance else None
        ).exists():
            raise serializers.ValidationError("SKU编码已存在")
        
        return value

    def validate_price(self, value):
        """验证价格"""
        if value <= 0:
            raise serializers.ValidationError("价格必须大于0")
        return value


class ProductListSerializer(serializers.ModelSerializer):
    """
    商品列表序列化器

        source='category.name',

    brand_name = serializers.CharField(
        source='brand.name',
        read_only=True
    )
    min_price = serializers.SerializerMethodField()
    image = serializers.SerializerMethodField()
    sku_count = serializers.IntegerField(
        source='skus.count',
        read_only=True
    )

    class Meta:
        model = Product
        fields = [
            'id', 'name', 'slug', 'category', 'category_name',
            'brand', 'brand_name', 'image', 'tags',
            'price', 'min_price', 'status',
            'view_count', 'sale_count', 'sku_count',
            'created_at'
        ]

    def get_min_price(self, obj):
        """获取最低价"""
        skus = obj.skus.filter(is_active=True)
        if skus.exists():
            return float(min(skus.values_list('price', flat=True)))
        return None

    def get_image(self, obj):
        """获取首图"""
        if obj.images and len(obj.images) > 0:
            request = self.context.get('request')
            if request:
                return request.build_absolute_uri(obj.images[0])
            return obj.images[0]
        return None
```

```python
    ordering_fields = ['sort_order', 'name', 'created_at']
    
    def get_queryset(self):
        """优化查询"""
        queryset = super().get_queryset()
        
        # 支持只获取顶级分类
        if self.action == 'list' and not self.request.query_params.get('all'):
            # 过滤掉子分类
            queryset = queryset.filter(parent__isnull=True)
        
        return queryset.select_related('parent')
    
    @action(detail=False, methods=['get'])
    def tree(self, request):
        """获取分类树"""
        cache_key = 'category_tree'
        cached_data = cache.get(cache_key)
        
        if cached_data:
            return Response(cached_data)
        
        # 获取所有顶级分类及其子分类
        top_categories = self.queryset.filter(parent__isnull=True)
        data = CategorySerializer(top_categories, many=True).data
        
        cache.set(cache_key, data, timeout=3600)  # 缓存1小时
        return Response(data)

class SKUViewSet(viewsets.ModelViewSet):
    """SKU管理"""
    queryset = SKU.objects.all()
    serializer_class = SKUSerializer
    lookup_field = 'code'
    filter_backends = [DjangoFilterBackend, filters.OrderingFilter]
    filterset_fields = ['product', 'is_active']
    ordering_fields = ['price', 'stock', 'created_at']
    
    def get_permissions(self):
        """权限控制"""
        if self.action in ['create', 'update', 'partial_update', 'destroy']:
            return [IsAdminUser()]
        return [IsAuthenticated()]
    
    def get_queryset(self):
        """优化查询"""
        return super().get_queryset().select_related('product')
    
    @action(detail=True, methods=['post'], permission_classes=[IsAdminUser])
    def adjust_stock(self, request, code=None):
        """
        调整库存
        
        管理员手动调整库存
        """
        sku = self.get_object()
        quantity = int(request.data.get('quantity', 0))
        reason = request.data.get('reason', '')
        
        if quantity == 0:
            return Response(
                {'error': '数量不能为0'},
                status=status.HTTP_400_BAD_REQUEST
            )
        
        # 记录变动
        InventoryLog.objects.create(
            sku=sku,
            operation='adjust',
            quantity=quantity,
            before_stock=sku.stock,
            after_stock=sku.stock + quantity,
            reason=reason,
            operator=request.user
        )
        
        # 更新库存
        sku.stock = F('stock') + quantity
        sku.save(update_fields=['stock', 'updated_at'])
        sku.refresh_from_db()
        
        return Response({
            'sku_code': sku.code,
            'new_stock': sku.stock
        })
    
    @action(detail=False, methods=['post'])
    def check_stock(self, request):
        """
        批量检查库存
        
        接收 SKU 编码列表，返回库存信息
        """
        codes = request.data.get('codes', [])
        
        if not codes:
            return Response(
                {'error': 'codes 不能为空'},
                status=status.HTTP_400_BAD_REQUEST
            )
        
        skus = SKU.objects.filter(code__in=codes, is_active=True)
        
        result = {
            sku.code: {
                'stock': sku.stock,
                'available': sku.available_stock,
                'is_active': sku.is_active
            }
            for sku in skus
        }
        
        return Response(result)
```

```python
订单异步任务

使用 Celery 实现异步任务处理
"""
from celery import shared_task
from django.conf import settings
from django.core.mail import send_mail
import logging

logger = logging.getLogger(__name__)


@shared_task
def send_order_confirmation(order_id: int):
    """
    发送订单确认邮件
    
    Args:
        order_id: 订单ID
    """
    from apps.orders.models import Order
    
    try:
        order = Order.objects.select_related('user').get(id=order_id)
        
        subject = f"订单确认 - #{order.order_no}"
        message = f"""
        您好 {order.user.username}，
        
        您的订单已确认，订单号：{order.order_no}

        支付方式：{order.get_payment_method_display()}
        
        我们将尽快为您发货。
        
        感谢您的购买！
        """
        
        send_mail(
            subject=subject,
            message=message,
            from_email=settings.DEFAULT_FROM_EMAIL,
            recipient_list=[order.user.email],
            fail_silently=False
        )
        
        logger.info(f"订单确认邮件已发送: {order_id}")
        
    except Exception as e:
        logger.error(f"发送订单确认邮件失败: {e}")


@shared_task
def process_order_payment(order_id: int):
    """
    处理订单支付
    
    1. 验证支付状态
    2. 更新订单状态
    3. 扣减库存
    4. 发送通知
    """
    from apps.orders.models import Order, OrderItem
    from apps.products.models import SKU
    
    try:
        order = Order.objects.get(id=order_id)
        
        # 更新订单状态
        order.status = 'paid'
        order.paid_at = timezone.now()
        order.save(update_fields=['status', 'paid_at', 'updated_at'])
        
        # 扣减库存
        for item in order.items.all():
            SKU.objects.filter(id=item.sku_id).update(
                stock=F('stock') - item.quantity,
                reserved_stock=F('reserved_stock') - item.quantity
            )
        
        # 记录库存日志
        for item in order.items.all():
            InventoryLog.objects.create(
                sku_id=item.sku_id,
                operation='out',
                quantity=-item.quantity,
                before_stock=item.sku.stock,
                after_stock=item.sku.stock - item.quantity,
                order=order,
                reason='订单支付'
            )
        
        logger.info(f"订单支付处理完成: {order_id}")
        
    except Exception as e:
        logger.error(f"处理订单支付失败: {e}")
        raise


@shared_task
def expire_orders():
    """
    过期未支付订单处理
    
    每天执行一次，释放预留库存
    """
    from apps.orders.models import Order, OrderItem
    from apps.products.models import SKU, InventoryLog
    from django.utils import timezone
    
    # 查找超过30分钟未支付的订单
    expired_orders = Order.objects.filter(
        status='pending_payment',
        created_at__lt=timezone.now() - timezone.timedelta(minutes=30)
    )
    
    for order in expired_orders:
        # 释放预留库存
        for item in order.items.all():
            SKU.objects.filter(id=item.sku_id).update(
                reserved_stock=F('reserved_stock') - item.quantity
            )
            
            InventoryLog.objects.create(
                sku_id=item.sku_id,
                operation='unlock',
                quantity=item.quantity,
                before_stock=item.sku.stock,
                after_stock=item.sku.stock,  # 库存不变，只是释放预留
                order=order,
                reason='订单过期释放'
            )
        
        # 更新订单状态
        order.status = 'expired'
        order.save(update_fields=['status', 'updated_at'])
        
        logger.info(f"订单已过期: {order.order_no}")
    
    return f"处理了 {expired_orders.count()} 个过期订单"
```

### 3.1 Django REST Framework 认证机制

```python
# JWT vs Session 认证

# JWT (无状态，适合前后端分离)
# - Token 存储在客户端
# - 每次请求携带 Authorization: Bearer <token>
# - 无服务端会话

# - Session 存储在服务端
# - 通过 Cookie 传递 Session ID
# - 可存储更多数据
```

## 💼 面试要点

### 问题 1: Django REST Framework 的核心组件？

**参考答案：**

1. **Serializer**：数据序列化和验证
2. **ViewSet**：封装了 CRUD 操作的视图
3. **Router**：自动生成 URL
4. **Authentication**：认证机制
5. **Permission**：权限控制
6. **Throttling**：限流
7. **Filtering**：过滤
8. **Pagination**：分页

### 问题 2: Django ORM 如何优化查询？

**参考答案：**

1. **select_related**：一对一/外键预加载（SQL JOIN）
```python
Product.objects.select_related('category', 'brand')
```

2. **prefetch_related**：多对多/反向外键预加载（多次查询）
```python
Product.objects.prefetch_related('skus', 'tags')
```

3. **only/defer**：只选择需要的字段
```python
Product.objects.only('id', 'name', 'price')
```

4. **values/values_list**：只获取字典/元组
```python
Product.objects.values_list('name', 'price')
```

5. **分页**：避免一次性加载全部数据

### 问题 3: 如何保证 Django 应用的安全？

**参考答案：**

1. **SQL 注入**：使用 ORM，避免原始 SQL
2. **XSS**：开启 CSRF 防护，模板自动转义
3. **CSRF**：使用 {% csrf_token %}
4. **密码**：使用 Django 内置哈希算法
5. **权限**：使用 DRF Permission 组件
6. **敏感数据**：环境变量，settings 分离
